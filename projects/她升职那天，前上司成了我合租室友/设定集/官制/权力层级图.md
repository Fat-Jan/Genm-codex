# 决策链图

## 项目决策链

```text
澜曜互动内部：
苏蔓（商务总监）
  -> 姜栀（代理客户组长）
    -> 执行组 / 达人投放 / 复盘支持

栖山咖啡内部：
裴述（品牌市场负责人）
  -> 程竞（电商负责人）
    -> 采购 / 财务 / 运营执行
```

## 权力边界

- 姜栀负责拿方案和执行结果，不能越过苏蔓直接调全公司资源。
- 裴述负责给业务判断和窗口，但不能单独替供应商走完正式续约。
- 程竞只认试运行后的数据回收，不认旧人情。

## 戏剧约束

- 任何关键推进都不能写成“裴述一句话就全放行”。
- 任何工作收益都必须经过至少一个组织节点确认。
- 关系线一旦影响到项目判断，必然触发名誉或流程代价。
