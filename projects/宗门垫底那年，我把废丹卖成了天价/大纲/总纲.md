# 《宗门垫底那年，我把废丹卖成了天价》总纲

## 基础信息

```yaml
书名: 宗门垫底那年，我把废丹卖成了天价
题材: 玄幻脑洞
目标平台: 番茄
预计章节数: 10
预计字数: 40000
```

## 一句话简介

在青岚宗外门垫底、连住处都快保不住的沈砚秋，意外唤醒一杆能给废丹重新定价的碎纹丹秤。从此别人倒掉的废丹，在他手里不再是垃圾，而是一门能换灵石、换人脉、也换命的生意。

## 核心卖点

- 第一章直接落“宗门垫底 + 资源见底 + 今夜可能被赶下山”的生存困局
- 脑洞不是单纯鉴宝，而是“废丹按适配人群重新定价”，立刻作用于现实交易
- 前三章完成“差点被逐下山 -> 废丹第一次见效 -> 当众卖出天价反打压迫者”的小爽点闭环

## 主角人设

- 姓名：沈砚秋
- 身份：青岚宗外门丹房最底层药童
- 当前困局：月考垫底，贡献点不够，今晚交不上保名钱就得滚下山
- 核心性格：穷怕了，所以比谁都算得清；挨压时不服软，翻身时也不飘

## 高概念机制

- 金手指：碎纹丹秤
- 核心脑洞：废丹不是一律无用，而是被粗暴地按“正常修士标准”定义成废品。碎纹丹秤能看见废丹残效、适配人群和可拆卖价格。
- 基础规则：
  - 只能看见已有残效，不能凭空造丹
  - 同一批废丹卖错了人一文不值，卖对了人价格翻数十倍
  - 每次赚到大价，都会动到既有利益链

## 主线剧情

1. 第1-3章：沈砚秋在被逐下山边缘唤醒碎纹丹秤，用废火丹完成第一次翻身，保住外门资格。
2. 第4-7章：他开始系统化收废丹、拆残效、做高价交易，逐渐碰上丹院和黑市的压价链。
3. 第8-10章：碎纹丹秤的来源被掀开一角，沈砚秋意识到“废丹被当废”的背后，本就是一门有人故意压住的生意。

## 第一卷规划

### 第一卷：从废丹窖翻到外门桌面

```yaml
卷目标: 让主角从宗门垫底、今夜可能被赶下山，切到手里有第一桶灵石和第一批人脉
起始章节: 1
结束章节: 10
核心冲突: 废丹在宗门眼里是垃圾，在主角手里却成了值钱货；谁先掌握定价权，谁就能踩着别人往上爬
情绪走向: 压制 -> 点亮 -> 见效 -> 小翻身 -> 被盯上 -> 更大生意和更大风险
```

## 黄金三章锁定

- 第001章：月考垫底、今夜逐出、资源差距直接砸脸；碎纹丹秤点亮，告诉他废火丹并非真废。
- 第002章：碎纹丹秤第一次真正见效，沈砚秋靠废火丹救急并拿到灵石和留宗资格。
- 第003章：沈砚秋在外门小集把废丹卖成天价，当众反打许观，完成一轮小爽点闭环，但惹来内门视线。

## 关键关系

- 沈砚秋 vs 许观：外门底层和管库执事的直接压迫关系
- 沈砚秋 vs 乔双双：互相利用的合作，她看中的是赚钱路子，不是白送人情
- 沈砚秋 vs 陆长河：一场救急换来的人脉，后续能变成往内门伸手的桥
- 沈砚秋 vs 秦执炉：别人不要的废丹，可能正是他不想让外人知道的东西
