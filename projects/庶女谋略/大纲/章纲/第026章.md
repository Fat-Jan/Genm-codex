# 第026章 残字无名

## 章节信息

- 章节编号：026
- 章节标题：残字无名
- 目标字数：4000-5000字
- 章节类型：旧案抓手章节
- 卷内位置：完成第二卷前段第一轮“婚配换账 + 旧闻入场”的收束

## 本章核心问题

- 当那个旧闻背后终于露出一点像“名字”的痕迹时，苏照棠怎样判断这条线值得她冒多大的险？

## 章节目标

1. 给旧闻线一个比流言更硬的抓手
2. 让主角明确把这条线当成未来筹码来源之一
3. 收束 `024-026` 这组“第一次换到账 + 旧闻入场”
4. 为下阶段更深的苏府内斗和高门扩场做桥

## 场景拆解

### 场景1：被抹掉的名字尾痕

- 场景：她在旧礼例或夹页边角，终于看见一处被擦改过的残字，足够说明那位寄居表姑娘并非完全无名。
- 主动方：旧册痕迹
- 防守方：苏照棠
- 误判点：名字被抹，比死亡本身更像有意为之
- 场景结果：旧闻从“病故流言”升级为“疑似被处理过的痕迹”

### 场景2：她判断这不只是巧合

- 场景：把婚配线里对自己的“压住她”、旧闻里的“稳静好命”和被抹名痕迹放在一起后，她第一次觉得这不是普通高门悲剧。
- 主动方：苏照棠
- 防守方：谨慎
- 误判点：异常不是说明书，而是一种反复出现的结构
- 场景结果：第二卷深层问题得到第一次明确命名，但仍不讲透

### 场景3：她决定把旧闻也写进账

- 场景：苏照棠不再把旧闻当成闲线，而是正式记进自己的大账，视作未来能换位置的一条暗筹。
- 主动方：苏照棠
- 防守方：眼前事务太多的现实
- 误判点：能救命的东西，起初常常不像主线
- 场景结果：婚配线、旧闻线第一次被主角主动并账

### 场景4：王氏推进下一轮安排

- 场景：正院正式放出下一轮婚配或外家安排，迫使主角不能沉迷旧闻，必须继续借势应对眼前。
- 主动方：王氏
- 防守方：苏照棠
- 误判点：第二卷不是侦查旧案卷，而是旧案如何改变主角当下破局方式
- 场景结果：下一个十章段落的现实压力启动

### 场景5：她守住两条线

- 场景：章末主角确认，接下来她必须一手守眼前婚配和苏府换账，一手慢慢摸成国公府旧闻。
- 主动方：苏照棠
- 防守方：被单线牵走的风险
- 误判点：真正的第二卷能力，不是只盯一笔账，而是学会并账
- 场景结果：`024-026` 的第二组链条收住

## 本章账本

- 收益：
  - 旧闻线拿到第一份硬痕抓手
  - 主角形成“并账”意识，开始同时处理眼前与暗线
- 代价：
  - 她知道的东西又多了一层，退路却并未因此变多
  - 继续查下去，会更接近不该碰的高门旧事
- 外部观感：
  - “二姑娘病后是越发安静，可那安静不像认命”

## 章末钩子

- 下一轮苏府安排里，第一次出现一个把婚配、中馈和外家脸面绑在一起的新局。
