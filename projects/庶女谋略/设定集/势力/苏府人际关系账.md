# 苏府人际关系账

## 用途

- 记录第一卷主要人物之间的真实利益、恐惧和当前风向
- 让后续正文里每个人都像在按自己的账活，不像围着主角转

## 关系总原则

- 苏府不是“主角 vs 反派”单线结构
- 每个人都在先保自己的位置
- 主角的每次变化，都会通过不同人的利益过滤成不同观感

## 核心关系账

### 苏照棠 -> 王氏

- 表层关系：继女对嫡母
- 当前判断：王氏是最会把秩序变成刀的人
- 当前策略：不正面撕，不轻易送把柄，优先拆她包装好的“体面”

### 王氏 -> 苏照棠

- 当前判断：病后一场，二姑娘不像从前那样顺了
- 当前担忧：
  - 老爷多看她几眼
  - 下人重新估量她
  - 她学会拉低位人心
- 当前策略：
  - 先用规矩和名声压
  - 再把她放进婚配筹码体系

### 苏照棠 -> 苏婉柔

- 表层关系：庶妹对嫡姐
- 当前判断：苏婉柔最会利用“理所当然”来压人
- 当前策略：不争嫡女主位，但也不再白白认低

### 苏婉柔 -> 苏照棠

- 当前判断：这个庶妹病后变得会说、会看、会藏锋
- 当前担忧：
  - 父亲把她纳入比较
  - 她破坏嫡庶差的自然秩序
- 当前策略：
  - 用“夸”“劝”“替她着想”的方式定她名声

### 苏照棠 -> 苏文渊

- 表层关系：女儿对父亲
- 当前判断：父亲只认体面、秩序和可用性，不会天然救她
- 当前策略：借他的视线，不依赖他的偏爱

### 苏文渊 -> 苏照棠

- 当前判断：二姑娘近来像是长进了，并非全无心思
- 当前需求：
  - 她别把家里闹难看
  - 她若可用，能为家中添一分体面
- 当前态度：可再看，但必须守边界

### 苏照棠 -> 林姨娘

- 当前判断：母亲的怕是真的，且这种怕会拖住自己
- 当前策略：不能逼她变强，只能尽量少让她替自己挨刀

### 林姨娘 -> 苏照棠

- 当前判断：女儿病后像是醒透了，也更敢了
- 当前担忧：她越招眼，正院越会拿姨娘房开刀
- 当前期待：只盼她别走到太危险的地方

### 苏照棠 -> 林若雪

- 当前判断：她聪明，但更怕赌输
- 当前策略：不急着收编，以互换信息和彼此试探为主

### 林若雪 -> 苏照棠

- 当前判断：二姑娘变了，像是能护人，也像是会招祸
- 当前担忧：离她太近会被正院一起记恨
- 当前收益：能从她这里看到另一种活法

## 风向节点

### 最容易放风的人

- 东院大丫鬟
- 正院嬷嬷
- 管事媳妇

### 最容易听见真话的人

- 林若雪
- 茶水房和针线房来回跑的小丫鬟
- 在回廊角门干活的婆子

### 最容易改风向的节点

- 苏文渊的一句点评
- 王氏的一句“夸”
- 家宴、见客、花会这种公开场面

## 第一卷关系变化轨迹

- `001-005`：小桃先看出主角变了，苏婉柔开始认真记恨，王氏开始亲自评估
- `006-010`：林若雪进入脆弱同盟，下人开始怕她，父亲开始留意
- `011-015`：林姨娘先替她承压，父亲把她纳入“可再看”序列，嫡姐把她升级为长期提防对象
- `016-020`：所有人都开始从婚配角度重新定义她

## 写作提醒

- 每章冲突都尽量对应这张关系账里的某条利益和恐惧
- 若某人只剩“支持主角/反对主角”一种功能，说明写扁了
