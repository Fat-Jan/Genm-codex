# 小型家谱

> 用途：把苏府当前会用到的嫡庶、齿序、称谓压成一张一眼能看的图。

---

## 文字树

```text
苏文渊（兵部侍郎，正三品）
├── 原配：沈氏（已故）
│   └── （无出）
├── 继室：王氏（掌中馈）
│   └── 长女 / 大姑娘：苏婉柔（嫡出，状态：在府）【前期对手】
├── 妾室：林姨娘
│   └── 次女 / 二姑娘：苏照棠（庶出，状态：在府）【主角】
└── 妾室：赵姨娘
    └── 三女 / 三姑娘：苏婉如（庶出，状态：在府）【预留】
```

---

## 当前故事只看这几条

| 线路 | 需要确认的事实 |
|------|----------------|
| 主角线 | 苏照棠是林姨娘所出的庶女，被嫡姐和继母压制 |
| 对手线 | 苏婉柔是王氏所出的嫡长女，得体面与资源 |
| 继母线 | 王氏是继室，法统上是正室，掌中馈 |
| 称谓线 | “庶女 / 嫡姐 / 二姑娘”都能在树上闭合 |

---

## 包装前检查

- [x] 主角位置合法：苏照棠是庶出二姑娘
- [x] 对手位置合法：苏婉柔是嫡出大姑娘
- [x] 称谓闭合：不存在“只有二姑娘没有大姑娘”的问题
