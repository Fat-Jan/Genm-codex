# 创作简报

## 核心概念

**一句话概括**：

商户嫡女沈见秋重生回成婚前三日，识破定安侯府借婚事吃绝户、夺子和抹平旧案的布局，婚后以掌账为刀，一笔笔清算整座侯府。

## 基础设定

| 项目 | 内容 |
|------|------|
| 题材类型 | 宫斗宅斗 / 古代家族权力 |
| 子流派 | 主母重生清算 |
| 目标平台 | 番茄 |
| 目标字数 | 120万-150万 |

## 核心要素

- 主桶：`宫斗宅斗`
- 强标签：`主母重生`、`吃绝户`、`夺子`、`嫁妆账本`、`名节压杀`、`旧案外溢`
- 叙事模式：`账本清算`
- 风味护栏：`狠但不白拿`、`每次翻盘都掉代价`、`感情线后置`

## 读者动机

- 看一个本来要被侯府吃干抹净的女人，怎么先保自己的钱、名分和活路。
- 看“规矩”和“账本”怎样一起压死人，又怎样被女主反拿来清算。
- 看一桩被写顺的井案，怎样从后宅旧账里重新浮出来。

## 故事骨架

**开局（第1-3章）**：

- 主角现状：成婚前三日，被逼重列嫁妆单、换陪房
- 触发事件：重生回婚前，确认侯府仍准备借婚事吃绝户
- 初始目标：保住原嫁妆单底本、陪房名单和私账底册
- 第1章先抢到什么：原嫁妆单底本和春绫旧案第一条线索
- 第2章必须在哪卡一下：婚事不能轻退，外家已被聘财和商路绑定
- 第3章不能赢满什么：可以保住局部账权入口，但不能立刻全面掌家或直接翻案

**中期方向（第10-50章）**：

- 核心冲突：侯府吃绝户、夺子、做假账与井案旧账并线
- 主要对手：秦氏、顾行舟、周成、许妈妈背后的更高层口径
- 升级路径：保嫁妆 -> 保陪房 -> 摸账房 -> 翻旧账 -> 抬出井案

**终极目标**：

- 女主最终要实现的目标：拿回自己的财权、名分解释权和婚姻选择权，并把侯府这些年吃人吃账吃规矩的旧债一起结清
