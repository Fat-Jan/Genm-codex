# 侯府宅院结构与动线

## 核心地点

### 正院

- 使用者：秦氏
- 功能：内宅权力中心
- 主要戏份：
  - 立规矩
  - 压新妇
  - 交钥匙/不交钥匙
  - 对外统一体面口径

### 世子院

- 使用者：顾行舟、沈见秋
- 功能：婚后第一层夫妻战场
- 主要戏份：
  - 嫁妆单冲突
  - 陪房去留
  - 私账与婚后日常账

### 账房

- 使用者：周成及账房人手
- 功能：侯府账本入口
- 主要戏份：
  - 假账
  - 对账
  - 把私账并入公账

### 库房

- 使用者：秦氏一系、库房婆子、陪房
- 功能：实物与账本对应的地方
- 主要戏份：
  - 嫁妆实物被挪
  - 钥匙控制
  - 旧物失踪
  - 春绫生前最后接触过的物件

### 后园旧井

- 使用者：后院杂役偶尔经过
- 功能：井案地点
- 对外口径：春绫失足投井
- 当前规则：
  - 不能太早变成公开调查场
  - 更适合通过旧物、旧账、旧人慢慢回到这里

### 祠堂 / 小佛堂

- 使用者：老夫人、秦氏、族中长辈
- 功能：名节和规矩压迫场
- 主要戏份：
  - 跪规矩
  - 扣失德帽子
  - 逼认错

### 花厅

- 使用者：外家来人、侯府男主人、账外事务
- 功能：内外两套话术切换点
- 主要戏份：
  - 婚事体面口径
  - 外家与侯府周旋
  - 商路和聘财压力

## 关键动线

- `正院 -> 世子院`
  - 日常受压主线
- `世子院 -> 库房`
  - 女主第一次把“账”落到“物”
- `账房 -> 库房`
  - 假账和实物是否对应的验证链
- `库房/旧物 -> 后园旧井`
  - 井案由物件和旧账慢慢连回死地
- `正院/祠堂 -> 花厅`
  - 体现侯府内里吃人、外头装体面的双口径
