# 创作简报

## 核心概念

**一句话概括**：

程既明是清河县县衙最底层的刑房书吏，被推出去给一桩做脏过的旧命案顶错判之责；他靠一个只奖励“拒绝无效劳动”的系统，从一页残缺供词开始翻案洗罪，再顺着案卷和粮账一路摸到县衙替罪链的根。

## 基础设定

| 项目 | 内容 |
|------|------|
| 题材类型 | 历史 |
| 子流派 | 历史脑洞 / 系统流 / 低位逆袭 |
| 目标平台 | 番茄 |
| 目标字数 | 100万-150万 |

## 古代官场权力补充

| 项目 | 内容 |
|------|------|
| 制度口径 | 高度架空混融，以明清县衙逻辑为骨 |
| 核心官场线 | 地方官场 |
| 主角/核心对手的官名或身份 | 主角：刑房书吏；对手：杜主簿、罗县尉 |
| 这是实职、品秩、爵位还是命妇 | 主角为胥吏，不入流；对手为县衙实务权力位 |
| 他真正能命谁 | 主角前期谁也命不了，只能借流程缝隙反制 |
| 准备用进标题/简介的官名 | 县衙小吏、主簿、县尉、错判 |
| 这些官名是否已被真值表支持 | yes |

## 核心要素

**关键爽点**：

- 系统
- 低位逆袭
- 规则反杀
- 身份反转
- 小人物翻案

**主角人设关键词**：

- 性格：冷静、算账快、嘴上躺平
- 背景：寒门土著
- 金手指：省力生存系统
- 目标：先活下来，再不替任何人白死

**预期风格**：

- 历史脑洞
- 轻松外壳 + 真实压迫
- 快节奏智斗

## 开局（第1-3章）

```markdown
主角现状：县衙刑房底层书吏，卷宗最熟，身份最轻，最适合背锅。
触发事件：州府要求复核旧案，县衙准备推出一个最便宜的替罪羊。
初始目标：洗掉自己头上的错判锅，先活下来。
第1章先抢到什么：一页被压进旧档底层的残缺供词。
第2章必须在哪卡一下：证据在，但流程不认，上官也不认。
第3章不能赢满什么：只能证明案子不干净、自己不该背锅，还不能抓到真正幕后。
```

## 中期方向（第10-50章）

```markdown
核心冲突：旧命案背后牵出灾年征粮和仓账黑洞，县衙与豪强之间存在长期替罪链。
主要对手：杜主簿、罗县尉、地方豪强盐行。
升级路径：洗罪 -> 立足 -> 护人 -> 看懂规则 -> 争改规则资格。
```

## 终极目标

```markdown
从最容易被推出去填坑的小吏，变成能决定“案子怎么判、锅该谁背、底层能不能活”的人。
```

## 推进阻力与代价

| 项目 | 内容 |
|------|------|
| 当前主阻力源 | 规矩 + 人 |
| 主角最常用的推进手段 | 查旧档、卡流程、借时间差、放信息差 |
| 每次推进最可能付出的代价 | 被盯上、被孤立、被扣薪、被报复、被反向栽赃 |
| 最容易写顺的环节 | 证据 |
| 第2章要出现的卡点 | 证据存在，但人证、签押和流程都能被卡 |
| 第3章要保留的残留风险 | 程既明洗掉替罪锅，但主簿和县尉都意识到他不能再被随便处理 |
