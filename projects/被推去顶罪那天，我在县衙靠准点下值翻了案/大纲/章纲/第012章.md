# 第012章章纲

## 章节目标

完成第四批三章的小闭环：让程既明逼出“这条粮账线确实碰到州府口径”的阶段性事实，同时把故事正式从“清河县内部查账”推进到“县外压力入局”。

## 章节冲突

- 硬证在手，但一旦亮出来，就可能让裴知衡自己先掉进州府的刀口。
- 杜主簿和罗县尉一定会尝试重做一个更大、更体面的替罪方案。
- 程既明若逼得太猛，裴知衡本人也可能转成新的阻力。

## 章节主线

- 主线：程既明用新硬证逼裴知衡做出一轮比“封仓停平码”更危险的选择，让州府无法继续只隔空施压。
- 次级线：县衙内部旧三角关系开始裂变，裴知衡不再只是摇摆秩序方，而是正式站到“要不要把更高层拖下水”的岔口上。

## 关键场景

1. 杜主簿抛出一套更大的止损方案：
   - 推一个已经死了或失踪的中间人吃整条线
   - 或借州府口风反压“知县越权查仓”
2. 罗县尉从执行层配合，准备把现有证人和基层经手人都压成同一套口供。
3. 程既明把新硬证和前面几层证据并到一起，证明：
   - 这不是清河县自己能编出来的错账
   - 州府至少已经碰过、看过、默许过这条口径
4. 裴知衡被逼得必须选边：
   - 继续往上递
   - 或暂时压下，保住自己
5. 章末只赢一半：
   - 这条线已坐实碰到州府层
   - 但真正拿大头的人仍未露面
   - 程既明也从此不再只是一个县衙小吏在查案，而是被更上层注意到的人

## 主动方 / 防守方

- 主动方：程既明，逼裴知衡做出更危险的动作。
- 防守方：杜主簿、罗县尉和州府口风一起收口。

## 独立议程

- 裴知衡这一章不是赢主角，也不是输主角，而是在赌自己把证据递上去后，头顶那把刀会不会先落自己身上。
- 程既明开始必须考虑：若知县退，他还有没有第二条路。

## 信息差 / 误判点

- 前九章大家都以为这是县衙吃账、县衙灭口。
- 到第012章，要让读者知道：县衙只是这一条粮缺链里最容易露出的那一层皮。

## 第十二章兑现点

- 完成“县内查账 -> 州府入局”的阶段升级。
- 给下一批留下明确的新战场，而不是重复查仓查簿。
- 保持半胜：局抬高了，但人还没揪净，报复也会更狠。

## 下一章残账

- 州府谁会先下场。
- 裴知衡会不会先被调离、压制或切责。
- 程既明若失去知县这层壳，还能不能继续查下去。
