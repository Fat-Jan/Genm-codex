# 地方县衙办案流程口径

## 本书口径

- 县衙办案以“受理 -> 拿人/传人 -> 初审口供 -> 验状/物证 -> 成卷 -> 签押 -> 归档”为主。
- 本书重点不是神探推理，而是“旧秩序怎么借流程压人，主角又怎么借流程反咬”。

## 最小流程

| 步骤 | 经手人 | 作用 | 最容易出问题的地方 |
|------|--------|------|--------------------|
| 报案/呈状 | 原告/差役 | 把案子送进县衙 | 被压不立案、先改口径 |
| 受理 | 书吏线 | 是否正式入档 | 可被拖延、被抽掉关键描述 |
| 拿人/传人 | 县尉线 | 控制相关人到案 | 可逼供、可串供 |
| 初审口供 | 刑房协作 | 形成第一版说法 | 口供可被删改 |
| 验状/物证 | 仵作 | 补身体和现场证据 | 结果可被含糊、延迟 |
| 成卷 | 刑房书吏 | 把口供、验状、批示整理成卷 | 最容易藏残页、换顺序 |
| 签押/判词 | 知县拍板 | 形成明面结论 | 为求稳而草率定案 |
| 归档 | 主簿线 | 旧案沉底 | 后续最难翻，但也最容易留漏洞 |

## 主角可操作的四个节点

1. 旧供词残页
2. 调档顺序
3. 签押时间差
4. 验状、物证、口供不一致

## 第一卷流程规则

- 程既明第一次翻盘，只能做到“案子有问题 + 自己不该背锅”。
- 第一卷前期不能直接一步查到最终幕后。
- 每次推进都要留下代价：
  - 被盯上
  - 被扣薪
  - 被报复
  - 被反向栽赃
