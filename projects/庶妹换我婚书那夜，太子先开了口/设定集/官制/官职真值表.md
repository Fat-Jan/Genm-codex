## 核心制度分层

| 体系 | 当前采用口径 | 备注 |
|------|--------------|------|
| 皇权 / 储君 | 太子与东宫可直接对涉东宫规矩的案件发压 | 当前外层真正可用的是“太子开口” |
| 外朝中枢 | 未细锁 | 不把未锁官名写进包装主方案 |
| 内廷 / 宫职 | 老内侍、内侍负责传令与护持 | 不等于外朝判案官 |
| 东宫 | 东宫主位 + 詹事府 + 内侍执行线 | 当前主线核心权力入口 |
| 地方行政 | 非当前主舞台 | 暂不外推 |
| 军职 / 边镇 | 非当前主舞台 | 暂不外推 |
| 宗室 / 爵位 | 安平伯府属勋贵爵门 | 伯爵身份不等于外朝实职 |
| 命妇 / 封号 | 未锁 | 不外推 |
