# 古代权力层级图

## 文字树

```text
皇权核心
└── 太子 / 东宫
    ├── 东宫詹事府
    │   └── 涉婚书局、旧账局的问话与封证
    ├── 内侍 / 老内侍
    │   └── 传令、引路、护持、递话
    └── 东宫旧档线
        └── 旧档房 / 副册 / 旧账并卷

勋贵府门
└── 安平伯府
    ├── 安平伯（家主 / 爵门权力）
    ├── 柳氏（后宅主母 / 中馈）
    ├── 谢云昭（庶二姑娘）
    └── 谢明珠（继室所出嫡三姑娘）

婚约外部线
└── 顾家
    └── 顾家二房 / 顾承州

旧账外朝线
└── 宋崇（具体官位未锁）
```

## 当前故事只看这几条

| 线路 | 需要确认的事实 |
|------|----------------|
| 东宫线 | 太子通过东宫与詹事府发压，不必写成亲自事事出面 |
| 伯府线 | 安平伯是勋贵家主，不是外朝部长 |
| 内侍线 | 负责传令执行，不代替外朝判案 |
| 外朝旧账线 | 宋崇能接旧账，但具体官位未锁，不外推 |

## 关键冲突校验

- [x] 主角怕的人，在图里真的比她高一层
- [x] 对手压人的方式，和她的家门/中馈位置匹配
- [x] `太子` 的压制力来自储君与东宫规则，不靠空喊官名
- [x] `安平伯府` 的压制力来自爵门和府内资源，不直接等于朝廷实权
- [x] 东宫、伯府、顾家、旧账外朝线当前边界清楚
