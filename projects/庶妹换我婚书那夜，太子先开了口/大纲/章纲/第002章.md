# 第002章章纲

## 基础信息

```yaml
章节序号: 2
章节标题: 东宫问婚，先低头的不是我
字数规划: 3200-3800
本章目标: 借东宫来人把婚书争夺从“内宅姐妹争风”改写成“必须当众开匣验真”的家门公案
预期爽点: 柳氏急压失手 / 原婚书回手 / 东宫正式点名
本章阻力: 柳氏压消息、伯爷护脸面、原婚书和聘银单都不在谢云昭手里
本章代价: 谢云昭必须当众撕开家门体面，把自己彻底推到不可退的明面位置
本章局部收益: 原婚书归属被坐实，谢云昭拿到亲入东宫回话的资格
本章残留风险: 柳氏今夜会先一步去灭旧人口，婚书局开始转成旧账局
```

## 场景分解

### 场景 1

```yaml
场景地点: 安平伯府正厅
出场角色: 谢云昭、柳氏、谢明珠、伯爷、族叔、管家
场景目标: 让东宫来人的压力先压乱伯府内部秩序
场景阻力: 柳氏想把局压回“误会”，伯爷想保家门体面，谢云昭仍是被动位
情节概要: 东宫口谕一到，柳氏急着把谢云昭按回房中“冷静”，伯爷转而想用私下解释把事情压小。谢云昭不抢着哭诉，而是先逼所有人承认：现在最怕的不是她，而是婚书真被当众打开。
关键对话: “冷静？母亲的意思，是把我带回去，再让我像昨夜那样病一场？” / “东宫点名，和后宅私换婚书，根本不是一回事。”
情绪变化: 慌乱 -> 试压 -> 谢云昭把节奏抢回来
场景代价/变化: 谢云昭当众顶撞柳氏，等于主动切断“回屋私了”的退路
```

### 场景 2

```yaml
场景地点: 正厅案前
出场角色: 谢云昭、柳氏、谢明珠、伯爷、族叔、媒人
场景目标: 把“婚书原件必须开匣验真”这件事逼成当场不能回避的动作
场景阻力: 原件和旧册在伯府手里，谢云昭只有旧抄和话柄；一旦压不住场，反咬会被重新打成庶女胡闹
情节概要: 谢云昭不满足于旧抄占理，而是借族叔在场与东宫点名的压力，逼伯爷和族叔一起开匣，把原婚书、旧册、聘银单据摊到台面。柳氏想用礼数和“姑娘讲钱丢脸”压回去，反而坐实她最怕人看见的不是婚事，而是账。
关键对话: “请您和父亲一起开匣。” / “母亲既觉得不该讲，为何又敢在背后拿这些东西定我的死活？”
情绪变化: 绷紧 -> 卡住 -> 原婚书被迫开匣
场景代价/变化: 家门体面被当众撕开，伯爷第一次正面意识到谢云昭不是不会争，而是在等能咬穿喉咙的时机
```

### 场景 3

```yaml
场景地点: 正厅门口与案前
出场角色: 东宫内侍、谢云昭、柳氏、谢明珠、伯爷
场景目标: 兑现本章硬收益，同时把下一章的代价和更大局一起挂出来
场景阻力: 东宫不会替谢云昭直接伸冤，只会按自己的规则办；柳氏和伯爷仍想把事情压成婚事误会
情节概要: 东宫内侍亲自进门，重申次日巳时前请“二姑娘”携婚书与信物入东宫回话，且连聘银旧册也一并带上。谢云昭因此第一次拿到真正落回自己手里的主动权，但她也立刻意识到：今夜之后，柳氏一定会先掐所有知道旧账的人。
关键对话: “既然姑娘会问这句，那就都带上。” / “既然点的是我，那就由我自己去。”
情绪变化: 短暂胜利 -> 更大压力落下
场景代价/变化: 谢云昭拿到了次日入东宫的资格，但也把崔妈妈等旧人彻底暴露在柳氏的灭口时钟下
```

## 节奏安排

| 段落 | 内容 | 字数 | 情绪 |
|------|------|------|------|
| 开头 | 东宫口谕压进正厅，柳氏急压、伯爷急保脸面 | 350-500 | 高压 |
| 中段1 | 谢云昭反逼开匣，婚书与旧册开始摊开 | 900-1100 | 绷紧 |
| 中段2 | 柳氏失态、伯爷被逼表态、婚书归属坐实 | 900-1100 | 反制 |
| 高潮 | 东宫内侍进门点名，旧账被正式一起带上桌 | 700-900 | 爽点 |
| 结尾 | 谢云昭意识到旧人会被先下手，婚书局转成旧账局 | 250-400 | 更险 |

## 推进校验

- 本章关键收益不是自动到账，而是借东宫点名、族叔在场和当众开匣硬抢回来的。
- 本章真正的阻力来自柳氏压消息、伯爷保体面，以及原件和账册都不在谢云昭手里。
- 主角为了这次收益付出的代价，是把自己和家门一起推到无法回撤的明面位置。
- 这次赢完以后，更危险的是：旧人和旧账都暴露了，柳氏当夜一定反扑。
- 这是第2章：卡点在“能不能当众开匣、让原件真的见光”。

## 关键元素

### 伏笔处理

- 生母旧账：从旧抄延伸到账册和聘银单据，开始露出“婚书和账不是两件事”
- 东宫介入：从传话升级成正式点名回话，说明盯的不是单纯婚事

### 新伏笔

- 崔妈妈与其他旧人口：本章末明确进入高危状态 -> 第003章/第004章回收
- 东宫不只看婚书：由聘银旧册一起带上桌埋下旧案入口 -> 第003章深化

### 设定展示

- 伯府家门规矩在东宫口谕前第一次失灵
- “二姑娘”这个称谓第一次被东宫正式锁定到谢云昭身上

### 角色状态变更

- 谢云昭：从被动挨压，升级为正式被东宫点名回话的人
- 柳氏：从内宅掌控局势，转为急于今夜补最后一道手尾
- 伯爷：从看热闹式护脸，转为被拖下水的被动人

## Continuity Notes

- 继续保持当前项目“礼法、账目、话柄先于情绪爆发”的反咬路线。
- 本章的收益要控制在“婚书归属坐实 + 入东宫资格”，不能把旧案一口气掀完。
- 章末必须让读者明确：真正危险的不是明日入东宫，而是今夜谁会先去灭旧人。
