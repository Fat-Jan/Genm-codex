# 第003章章纲

## 基础信息

```yaml
章节序号: 3
章节标题: 太子只问一句：谁敢再换她的婚书
字数规划: 3300-3900
本章目标: 兑现“太子开口”的书名承诺，但把收益控制在‘借东宫改写规则层’而非‘旧案全开’
预期爽点: 东宫改写叙事 / 口谕落地 / 柳氏与伯爷第一次真正压不住
本章阻力: 东宫不替她私人伸冤、伯府仍想压成内宅误会、旧人口随时会被灭口
本章代价: 谢云昭为了拿到口谕，必须主动把婚书局拉成规矩局，也因此把崔妈妈和沈氏旧账一起推上更危险的位置
本章局部收益: 拿到东宫口谕与令签，婚书与账册暂由她自持，并获得三日内带旧人回话的资格
本章残留风险: 柳氏会先一步去庄子灭口；东宫给的是门，不是包办到底；旧案真正入口才刚打开
```

## 场景分解

### 场景 1

```yaml
场景地点: 东宫宫门外 -> 偏殿
出场角色: 谢云昭、伯爷、柳氏、谢明珠、年长嬷嬷、内侍
场景目标: 先把伯府这边所有“长辈替答”的路堵死
场景阻力: 伯爷想跟进、柳氏想隔着门继续控局、东宫并不天然站谢云昭
情节概要: 宫门一开，东宫先用“只点二姑娘入内”这条规矩，把伯爷和柳氏都压在门外，让谢云昭必须独自承担成败。她意识到：今日若只会哭婚事，东宫不会替她做主。
关键对话: “殿下没点旁人。” / “你今日进东宫，是想争婚事，还是想借东宫压内宅？”
情绪变化: 紧绷 -> 被单独拎上桌 -> 迅速稳住
场景代价/变化: 谢云昭失去所有“长辈在场帮她兜话”的余地，只能自己把局写大
```

### 场景 2

```yaml
场景地点: 东宫偏殿案前
出场角色: 谢云昭、内侍、屏风后之人
场景目标: 把“被夺婚事的庶女求做主”改写成“谁敢坏规矩、吞旧账”
场景阻力: 东宫要的是可用的局，不是可怜的哭诉；她若只讲婚事，会被压回内宅争风
情节概要: 谢云昭不求东宫替她争男人，而是把婚书、聘银、旧账并成一件事，说清楚柳氏换婚书不是怕她争亲事，而是怕她站在原婚书位上继续翻出沈氏旧账。东宫因此第一次明确看见：这不是单一婚事，而是规矩和旧账一起被人吞了。
关键对话: “臣女不敢求东宫替臣女争男人。” / “臣女争的不是婚事本身，是谁也不能把规矩踩成一张废纸。”
情绪变化: 试探 -> 顶住 -> 说动上位规则
场景代价/变化: 她主动把婚书局抬成规矩局，也等于自断后路，往后再也回不到“家中误会”那条路
```

### 场景 3

```yaml
场景地点: 偏殿 -> 宫门外 -> 马车上
出场角色: 谢云昭、内侍、屏风后之人、伯爷、柳氏、谢明珠、青枝
场景目标: 给出本卷第一份真正能带回家的“硬保护”，同时立刻挂出更危险的新战场
场景阻力: 口谕并不等于东宫替她走完路；她还缺旧人和活证
情节概要: 东宫口谕落下，婚书与账册暂由谢云昭自持，三日内带崔妈妈回话。谢云昭因此第一次拿到真正可落回伯府的刀。但她刚出宫门，就听见柳氏的人已经先赶往庄子，说明这份收益不是结束，而是把庄子旧人线直接推进成生死赛跑。
关键对话: “婚书既写她名，谁再换，按欺东宫论。” / “下一次再进东宫，带实证，不带眼泪。”
情绪变化: 短促硬爽 -> 更冷的清醒 -> 立即转急
场景代价/变化: 谢云昭拿到口谕，但也等于把旧人口和沈氏账线彻底暴露在柳氏的灭口线里
```

## 推进校验

- 本章硬收益不是自动到账，而是谢云昭主动把自己从“哭婚事的人”改写成“替规矩发问的人”才抢来的。
- 本章真正的阻力来自：东宫不管私人伸冤、伯府想压回内宅、旧人线随时会被灭口。
- 主角的代价是：她把局抬大了，也把崔妈妈和沈氏旧账一起推到了台面上。
- 本章不是全赢：她拿到了门，但还没拿到人，更没拿到旧案答案。
- 这是第3章：硬兑现是“令签到手”，不是“旧案查清”。

## 关键元素

### 伏笔处理

- 东宫为何介入：本章从“婚书异动”升级为“规矩和旧账一起被吞”的问题
- 沈氏旧账：首次被东宫认可为继续往下查的价值线

### 新伏笔

- 崔妈妈的生死赛跑：本章章末立刻接入 -> 第004章回收
- 东宫给门不替走：后续旧档局与更大旧案入口 -> 第009章后深化

### 设定展示

- 东宫权力的边界：给规则、给门、给令签，但不替她私人伸冤到底
- 谢云昭的打法：不求怜悯，只求把别人最不敢上桌的东西一起拖上桌

### 角色状态变更

- 谢云昭：从伯府受压庶二姑娘，升级为可持东宫口谕回府的人
- 伯爷：第一次清楚意识到事情已无法按家法收口
- 柳氏：从压婚书，转为抢时间灭口旧人

## Continuity Notes

- 延续当前项目“礼法、规矩、账目先于哭诉”的女主路线。
- 本章的收益必须控制在“规则升级 + 入口打开”，不能直接把旧案线打穿。
- 章末要非常明确：真正的新战场不是东宫，而是庄子和旧人口。
