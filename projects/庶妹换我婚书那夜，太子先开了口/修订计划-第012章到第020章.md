## 修订目标

### 字数目标

修后目标不是机械统一，但应大致回到番茄女频可连载正文章密度：

- 单章目标：`2200-3200` 字符
- 阶段收束章可略高
- 低于 `1800` 字符默认视为未达标，除非该章明确是极短过桥章且已通过 review

### 章节形态目标

每章至少要有：

1. 一个明确场景
2. 一次有效对抗或试探
3. 一次信息推进或证据推进
4. 一个章末钩子

不允许：

- 正文中大段项目符号式总结
- 主要剧情只用回顾句带过
- 连续两章都只有“她知道了 / 她看见了 / 她想明白了”

### 风格目标

继续保持：

- 宫斗宅斗女频向读者驱动
- 恶女 overlay 的冷硬利己感
- 压制 -> 试探 -> 反咬 -> 挂更大局

避免：

- 纯旁白概述
- 一章只说结果不演过程
- 用“她终于明白 / 她知道 / 这才是 / 原来”反复替代场景演出

## 推荐执行配置

### 本次任务：救第012章到第020章

- 任务性质：问题章节返修，不是新章续写
- 推荐模型：强模型
- 推荐思考强度：`high`

原因：

- 这批章节的问题是“正文退化成摘要态”，需要补场景、补对话、补对抗

只在这些时刻升：

- 修第012章前，先重排第012-015章的小弧线

- 修完一批后，发现必须重排第009-020章中段结构

### 批次建议

- 第012-014章：强模型 + `high`
- 第015-017章：强模型 + `high`
- 第018-020章：强模型 + `high`
- 每批修完后 `review / 质量门`：强模型 + `high`
- 查询 / 状态 / 轻量检查：中高模型 + `medium`

### 硬提醒

- 一次最多修 `3` 章
- 不要把第012-020章一口气连修到底
- 每批修完必须先过质量门，再决定是否继续下一批

## 具体修法

每章按这个顺序修：

1. 明确本章唯一主场景
2. 明确本章核心对抗双方
3. 明确本章必须落地的信息点
4. 明确本章末尾挂出的下一钩子
5. 再补：
   - 对话
   - 动作
   - 神态
   - 局势变化

不允许先写“总结句”再往里填内容。
