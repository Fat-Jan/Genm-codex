# 《公司裁我那天，系统先赔了我一百万》总纲

## 基础信息

```yaml
书名: 公司裁我那天，系统先赔了我一百万
题材: 都市脑洞
目标平台: 番茄
预计章节数: 10
预计字数: 40000
```

## 一句话简介

被公司违法裁员的那天，周既明还在想怎么凑母亲的治疗押金，手机却先收到了系统打来的一百万元生存预赔金。从这一刻开始，他每一次被现实坑出来的损失，都可能变成反咬回去的筹码。

## 核心卖点

- 现实困局开场，系统不做背景铺垫，第一章就直接见效
- 脑洞机制不是万能许愿机，只赔付真实亏损，逼主角自己去打现实仗
- 前三章完成“被裁压制 -> 系统垫钱 -> 现场反杀赔偿”的小爽点闭环

## 主角人设

- 姓名：周既明
- 身份：本地互联网公司产品经理
- 核心性格：冷，快，能算，受不了别人拿规则当刀架在他脖子上
- 核心动机：先把眼前的生活保住，再把压到自己头上的亏损链一条条撕开

## 主线剧情

1. 第1-3章：公司违法裁员，亏损赔付系统激活，周既明拿回第一轮赔偿，完成小闭环。
2. 第4-7章：行业黑名单和新赔付对象浮出，周既明开始把系统从被动补血工具，用成主动反制工具。
3. 第8-10章：他逐渐逼近系统真正的结算逻辑，也意识到有人在故意制造更多人的亏损链来喂系统。

## 第一卷规划

### 第一卷：先赔一百万，再把账一笔笔讨回来

```yaml
卷目标: 让主角从被违法裁员、现金流断裂的谷底，切到能用系统和证据反制现实规则
起始章节: 1
结束章节: 10
核心冲突: 公司想零成本清走主角，系统给了他第一口气，但真正的钱和位置仍要靠现实博弈拿回来
情绪走向: 压制 -> 诡异反转 -> 稳住生活 -> 试错 -> 反杀 -> 新风险
```

## 前三章锁定

- 第001章：现实困局直接砸脸，系统当场点亮，一百万元先到账。
- 第002章：系统第一次改变现实处境，主角用钱和证据先稳住命门，再确认系统边界。
- 第003章：主角靠证据和现场反制逼公司转账，形成一次完整小爽点闭环，但不直接无敌通关。

## 关键关系

- 周既明 vs 韩振：成果挪用和职场压制的正面对撞
- 周既明 vs 林乔：成本控制话术和个人权益的对撞
- 周既明 vs 宋雯：带条件的互帮，她不是单纯助攻，也在保自己
- 周既明 vs 系统：系统给钱很快，但每次钱都逼他把更大的现实问题顶出来
