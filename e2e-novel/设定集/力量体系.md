# 力量体系

> 供 `outline` 与 `write` 统一引用。第一卷只使用前两境，不提前展开更高境细节。

## 体系概览

```yaml
体系名称: 灵脉修行体系
力量来源: 吸纳天地灵气，经灵脉运转，再由功法、血脉与器物放大
核心风格: 低起点逆袭 + 有代价金手指 + 快节奏爆点
第一卷主轴: 锻体境 -> 聚气境
```

## 境界划分

```yaml
锻体境:
  核心: 淬炼筋骨、打磨肉身
  细分: 一重至九重
  标志: 纯肉身发力，尚未形成稳定真气

聚气境:
  核心: 引灵入窍，凝成第一口真气
  细分: 一重至九重
  标志: 真气可短暂离体，招式威力明显提升

灵台境:
  核心: 神识凝台，术法初成
  标志: 能稳定驾驭复杂术法与灵器

化海境:
  核心: 丹田化海，灵力可持续外放
  标志: 正式具备镇压一方的中层战力

真丹境:
  核心: 真丹凝成，道基稳固
  标志: 可作为宗门长老级战力参考
```

## 第一卷突破规则

- 沈照开局停在 **锻体境九重**，核心障碍是残脉无法顺畅纳灵。
- 第一次破境依赖 `残星引脉诀 + 古镜提纯资源 + 生死压迫` 三者叠加。
- 第一卷只允许突破到 **聚气境二重**，不跨入灵台境。
- 对更高小境界取胜必须依赖破绽、地形、镜返或资源，不允许纯数值碾压。

## 金手指接口

```yaml
名称: 照骨古镜
接入方式: 第一章 800 字内，以鲜血激活
核心收益:
  - 映照功法破绽
  - 提纯废丹、药渣、残损灵气
  - 在高压下短暂反照见过的一式
限制条件:
  - 需要镜光或气血驱动
  - 无法无视大境界压制
  - 频繁使用会导致经脉灼伤与镜纹开裂
升级路径:
  - 觉醒
  - 第一枚镜纹点亮
  - 掌握镜返
  - 收集第二块镜片
```

## 配套功法与资源

- **裂石拳**：外门粗浅拳法，适合开篇打脸。
- **青云步**：外门基础身法，负责近身拉扯和节奏变化。
- **残星引脉诀**：后山石室所得，帮助残脉完成第一次纳灵。
- **玄火砂**：雾隐谷石室资源，可用于淬脉疗伤和赛前强化。
- **照骨残卷**：藏经阁得到的后续成长锚点，不在第一卷提前展开完整能力。

## 设计硬约束

- 大境界压制默认不可逾越，越级必须有特殊手段与代价。
- 突破必须有铺垫、资源或事件触发，不能凭空升级。
- 战力提升尽量量化，便于番茄平台快节奏表达。
- 新设定单章不超过 3 个，避免世界观堆砌。
- 古镜不能直接赐予完整功法答案，只能给“线索、破绽、放大器”。

## 爽点触发位

- 越级反杀
- 境界突破
- 机缘获得
- 打脸震惊
- 旧案真相推进
