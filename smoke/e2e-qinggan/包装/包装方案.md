# 《离婚后我不再忍了》包装方案

## 内容桶判断

- 当前激活内容桶来自 `.mighty/state.json` 的 `genre_profile.bucket`：`现实情感`
- 该桶要求标题优先突出社会痛点和关系冲突，简介先抛伤害，再给觉醒与反击
- 当前总纲核心矛盾是婚姻失衡、婆家控制、现实利益切割，和 `现实情感` 的高痛点、高共鸣、高反击模板高度匹配

## 推荐主方案

### 主方向

离婚切割 + 婆家控制 + 女主止损反击

### 推荐书名

《被公开羞辱后，我和前夫一家算总账》

### 一句话卖点

被前夫当众羞辱、被婆家联手施压的沈知意，终于不再退让，在离婚切割里一步步拿回尊严、住所和生活主导权。

### 推荐简介

公开场合的一次羞辱，让沈知意彻底看清自己这些年在婚姻里到底失去了什么。  
当她提出离婚，前夫和婆家却同时扑上来，想用房子、钱和体面把她逼回原位。  
这一次，她不再忍，不再替任何人兜底，搬离婚房、整理证据、把该算的账一笔笔算清。  
可当切割越往下做，她越发现，这段婚姻里最见不得光的，从来不只是感情。

### 推荐开篇包装

1. 第一屏直接给“公开羞辱”现场，不先铺背景，让读者立刻站到女主这一边。
2. 伤害发生后立刻接“她第一次明确说离婚”，不要把决断拖到很后面。
3. 第一章末尾就要抛出更大的现实钩子，例如婚房、财务记录或婆家真实诉求。
4. 第二章内给出第一次有效反击，让读者看到她不是只会受气。

## 备选方案

### 备选书名

2. 《婆家逼我净身出户，我反手把账算清》

4. 《那次当众羞辱后，我决定离婚反击》

### 备选简介方向

#### 平台稳妥版

沈知意隐忍多年，本以为退让能换来婚姻稳定，直到一次当众羞辱彻底击碎她的幻想。  
她提出离婚后，前夫和婆家却轮番施压，想让她在财务、住房和体面上全部退让。  
不再忍的她开始搬离婚房、咨询律师、整理证据，把失衡婚姻一点点切开。  
可真相翻出来后，她才知道，自己面对的从来不是简单的感情破裂。

#### 冲突强化版

公开羞辱那天，沈知意终于明白，自己越懂事，别人越敢踩她。  
前夫用情绪和钱拿捏她，婆家把她当成随时能丢掉的工具人。  
她带着证据走出婚房，第一次不讲情面，只讲边界和代价。  
而她越想脱身，那一家人就越慌，因为她手里攥着他们最怕被翻出来的东西。

#### 共鸣强化版

很多人都劝沈知意再忍一忍，可没人替她承担那一次次羞辱和消耗。  
当婚姻把她磨成一个只会收拾残局的人，她终于决定把自己从这段关系里剥离出来。  
离婚不是结束，而是她第一次认真站到自己这一边。  
只是她没想到，真正难打的不是离婚官司，而是那些打着“为你好”名义的控制。

## 暂不推荐方向

- 不建议包装成纯“抓出轨爽文”，当前总纲重点是失衡婚姻和利益切割，不只是单点狗血
- 不建议主打职场逆袭，现阶段最强钩子仍然是婚姻压迫与反切割
- 不建议标题过于温吞治愈，`现实情感` 桶要求先把痛点和关系冲突顶到前面

## 包装约束

- 标题要优先满足“人物关系 + 核心伤害 + 反击结果”或“社会痛点 + 高情绪词 + 逆转”
- 简介必须先抛痛点，再抛被逼处境，再亮出反击动作，最后留下更大后果
- 开头前 `100-200` 字就要出现伤害现场，不能先写日常铺垫
- 反击必须建立在现实手段上，例如证据、法律常识、谈判和财务切割
- 情绪要集中，仇恨值要拉高，但不能写成不符合现实逻辑的碾压
