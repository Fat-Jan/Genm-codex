# 第004章章纲

## 基础信息

```yaml
章节序号: 4
章节标题: 他把求和闹到我公司门口，我先把那张假签字拍给了法务
字数规划: 3000-3600
本章目标: 把离婚切割从银行与婚房线推进到职场和母亲那笔三十万转账线，兑现一次“现实反制不是吵赢，而是把战场改成自己更能掌控的地方”。
预期爽点: 前夫带着婆婆来公司门口装求和 / 沈知意不回情绪战，直接把伪签材料抛到法务和人事面前 / 母亲那笔三十万终于坐实来源
本章阻力: 周承礼想把经营贷与首付款异常重新压回“夫妻私事”，周母要借“体面”和“你妈知道吗”反压；公司一侧不会无条件站沈知意，HR 和直属领导更想先稳住舆情与客户项目
本章代价: 沈知意若想保住工作边界，就必须让公司法务、人事和直属领导知道自己婚姻切割已经影响到职场；她也得主动把母亲拉进这场账的核对里，等于撕掉最后一点“我先自己扛”的缓冲层
本章局部收益: 坐实三十万确实来自母亲账户，拿到母亲的转账凭证或语音确认；同时把周承礼“伪签授权”从夫妻口角升级成法务和公司都看得见的现实风险
本章残留风险: 沈知意虽然守住了公司门口和法务视角，但周承礼会更急着补首付款那条线；母亲也因此第一次真正知道她这场婚姻的账已经烂到什么程度
主动方: 沈知意
防守方: 周承礼 / 周母 / 公司的“先别闹大”惯性
关键误判: 周承礼以为冲到公司门口装低头就能逼沈知意顾体面；周母以为拿母亲和“离婚丢脸”能把她压回去；沈知意一开始也低估了公司对“私人纠纷影响项目”的敏感程度
独立议程: 直属领导愿意给她会议室和缓冲时间，不是站队，而是怕客户看到闹剧影响季度项目；HR/法务愿意介入，不是为了替她撑腰，而是要提前隔离职场风险；母亲愿意把三十万转账凭证翻出来，不是因为认同她所有做法，而是因为终于意识到这笔钱可能被周家吞平
关系残账: 沈知意和母亲会因为“当年为什么不早说这笔钱的用途”产生新的裂口；她和领导的关系也从“能扛事的主管”变成“需要公司评估风险的人”
```

## 场景分解

### 场景 1
```yaml
场景地点: 公司楼下 / 前台门口
出场角色: 沈知意、周承礼、周母、前台、小陈
场景目标: 把前夫和婆婆的“求和”定性成越界施压，而不是让他们拿到情绪主场
场景阻力: 公司是公共场所，沈知意若当场失控就会先输；周承礼会装低头、装沟通，周母会当众抹泪讲“家丑别外扬”
情节概要: 周承礼和周母堵到公司楼下，话术不再是纯威胁，而是“我们是来好好谈”的低姿态。沈知意不接“回家说”，而是先让前台和同事看见：这不是夫妻吵架，是对她工作场域的越界干扰。
关键对话: “你们今天不是来求和，是来堵我上班。” / “我不怕你们在楼下丢脸，我只怕自己再把你们的脸当回事。”
情绪变化: 突袭 -> 绷住 -> 反拿场域
场景代价/变化: 她为了不被堵回情绪战，必须把私人矛盾公开成职场干扰
```

### 场景 2
```yaml
场景地点: 公司小会议室
出场角色: 沈知意、直属领导、HR、法务同事、周承礼
场景目标: 把“伪签授权”和“经营贷代扣”从夫妻争吵升级成现实合规问题
场景阻力: 领导和 HR 更在意影响项目和办公秩序，不会主动站在沈知意这边；周承礼会拼命把事情说成“夫妻之间误会”
情节概要: 领导出于项目稳定让人把双方带进会议室。沈知意没有继续吵，而是直接把代扣通知、伪签影印页和律师函草稿摆上桌，让 HR 和法务自己看“这已经不是夫妻口角，是未经同意使用员工账户和疑似伪签”。周承礼原本想打圆场，反而被现实材料逼得更难收口。
关键对话: “你们要我冷静，我现在就很冷静。” / “这不是他哄不哄我的问题，是他有没有权利替我签字的问题。” / “公司如果担心影响办公秩序，那就请把风险记录清楚。”
情绪变化: 被围观 -> 借规则反打 -> 周承礼第一次在‘外部规则’前失控
场景代价/变化: 她保住了场面主导权，但也让公司正式知道她的婚姻切割已经影响到工作边界
```

### 场景 3
```yaml
场景地点: 公司天台 / 母亲电话
出场角色: 沈知意、母亲、许雯
场景目标: 坐实三十万的来源，并把这笔钱从“她隐约知道”推进成“可被继续追的硬线”
场景阻力: 母亲最怕事情闹大，过去也默认“成家了就别老算那么清”；许雯提醒她要拿确认证据，不要只靠回忆
情节概要: 会议室散后，沈知意在天台给母亲打电话，第一次直接问出“当年那三十万是不是你转的”。母亲起初想让她先别闹，后来在沈知意把经营贷和伪签一并讲清后，才承认那笔钱确实是她悄悄塞进去给女儿压底的，并答应翻旧转账凭证或语音记录。
关键对话: “妈，我现在不是要面子，我是要把账拿回来。” / “那钱是我给你的，不是给他们家的。” / “你要是真查，我把当年的转账记录翻出来给你。”
情绪变化: 强撑 -> 终于问出口 -> 现实坐实
场景代价/变化: 沈知意把母亲正式拖进这场账，也因此逼自己承认，这不是她一个人悄悄切割就能解决的事
```

### 场景 4
```yaml
场景地点: 公司楼下 / 车边
出场角色: 沈知意、周承礼
场景目标: 把本章局部胜利压实，并给下一章挂上更清晰的财务追账目标
场景阻力: 周承礼已经知道她开始动“首付款三十万”这层，会立刻试图补口供或抢先和她母亲接触
情节概要: 周承礼在车边再次拦她，这一次不再装和解，而是急着问她到底和谁说了什么。沈知意没有再跟他吵，只把一句“那三十万是我妈给我的”扔回去，看他脸色当场变掉。她因此确认，这条钱线确实踩中了他真正怕的地方。
关键对话: “你刚才怕的不是丢脸，是我妈那笔钱终于被翻出来。” / “周承礼，你最好祈祷这三十万只有一笔。” 
情绪变化: 冷静 -> 试探命中 -> 更深的账被抬起来
场景代价/变化: 她赢下了本章的话语权，但也让对方知道自己已经摸到首付款核心线，后续补口供和转移证据会更快
```

## 推进校验

- 本章关键收益不是吵赢，而是把“夫妻私事”强行改写成“公司和法务都得看得见的现实风险”。
- 本章阻力要同时落在：
  - 前夫 / 婆家的情绪施压
  - 公司对秩序和项目的自保
  - 母亲对“别闹大”的惯性
- 主角代价必须可见：
  - 她主动暴露婚姻切割对工作的影响
  - 她把母亲也正式拖进这笔账
- 本章只赢到：
  - 三十万来源被坐实
  - 经营贷伪签从口角升级成规则风险
- 章末必须把下一战挂到“首付款那笔三十万到底怎么被吞平”的追账线上。

## 关键元素

### 伏笔处理
- `婚房首付款来源异常`: 从“看见一笔不对的钱”推进到“确认三十万来自母亲”
- `前夫真实诉求`: 从“怕离婚”推进到“怕这场账被往更早的时间线翻”

### 新伏笔
- 母亲保留的转账凭证 / 语音记录
- 周承礼会不会先一步去找母亲改口或补口供

### 设定展示
- 现实情感线的反制不是靠大开挂，而是靠：
  - 规则
  - 留痕
  - 把战场改到自己更能控制的地方
- 职场支持不是无条件站队，而是各自也在保项目、保秩序、保合规

### 角色状态变更
- 沈知意：从“切经营贷”推进到“开始翻婚姻最初那笔钱”
- 周承礼：从嘴硬和压人，第一次变成“怕哪条线被翻出来”的现实惊慌
- 母亲：从背景安慰者，变成真正握有关键来源的人

## Continuity Notes

- 这章必须是 `现实情感` 线，不要写成纯离婚吵架章。
- 许雯、领导、HR、母亲都要带自己的现实顾虑，不写成无条件辅助位。
- 章末钩子不是“感情还有没有救”，而是“这三十万到底被谁吞平了”。
