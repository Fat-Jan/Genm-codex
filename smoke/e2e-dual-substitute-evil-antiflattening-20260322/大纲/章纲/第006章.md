# 第006章章纲

## 基础信息

```yaml
章节序号: 第006章
章节标题: 她去拦应岚时，白月光先替她挡了第一道镜头
字数规划: 3200-3800
本章目标: 在应岚离开海城前完成第一次正面截人，让“替换候选”从抽象剧本推进到可被当事人承认的执行逻辑，同时继续压住双女主裂口而不是回甜。
预期爽点: 姜栀不再只拿证据谈判，而是主动追到机场外截人 / 许知微不是递刀工具，而是用自己的方式卡位 / 应岚承认“替换候选”是一套可复制产品，不只是一次偶然操作
本章阻力: 姜栀背着“偷档”热搜和律师函，不能被拍成失控追打；许知微必须先稳住许家那边，不可能全程公开站在姜栀身边；应岚极擅长切词，把自己包装成“只做执行”的乙方，不会轻易吐真正拍板者
本章代价: 姜栀必须顶着第二轮偷拍视频风险去截人；许知微为了卡机场和基金关系线，等于主动把自己继续卷进这场脏局；林特助和小夏都要各自承担被追责的现实后果
本章局部收益: 姜栀拿到应岚一句足以坐实“替换候选不是临时玩笑，而是预设产品”的口供片段，并顺手锁到一个旧项目云盘入口或转账抬头
本章残留风险: 应岚没有吐出最终拍板人，只把两家都往“默认知情”方向推；偷拍视频第二轮仍可能发酵；双女主合作继续成立，但信任并未修复
主动方: 姜栀
防守方: 应岚 / 贺氏法务 / 许家切割线
关键误判: 姜栀起初以为拦住应岚就能拿到清晰答案；许知微起初以为只要替她挡一次镜头，就能把两人的裂口先压住；应岚误判姜栀这次仍会先发火后失控
独立议程: 许知微帮姜栀，不是为了站队，而是为了把“许家只是经手不是立项”这条线往自己能控制的方向收；林特助配合，不是忠诚，而是怕自己也被应岚和许太太一起切出去；小夏继续帮半步，是为了保住品牌团队后路
关系残账: 双女主这次会重新合作，但姜栀不会因为许知微替她挡镜头就原谅她此前的隐瞒；许知微也不会因为姜栀愿意共同行动就交出全部底牌
```

## 场景分解

### 场景 1
```yaml
场景地点: 基金旧馆外 -> 车上
出场角色: 姜栀、许知微、林特助
场景目标: 把“该不该立刻拦应岚”从情绪冲动变成一场带条件的再合作
场景阻力: 姜栀背着偷拍视频和律师函，贸然追人会被直接挂成失控报复；许知微不能公开陪她去拦，否则许家那条线会立刻断；林特助不想同时得罪许家和姜栀
情节概要: 姜栀要立刻去机场拦人，许知微先压她别用昨晚那张图硬爆，而是拿“应岚下午两点四十飞上海”的时间差做局。三人重新分工：姜栀正面追人，许知微用基金和许家关系卡机场接待口，林特助只负责给一次内部通行信息，之后必须切手。
关键对话: “我现在冲过去，外面只会说我狗急跳墙。” / “我替你挡镜头，不等于我替你认账。” / “我只给这一条通行线，出事别把我算进同盟。”
情绪变化: 急躁 -> 再交易 -> 冷着站成同一边
场景代价/变化: 双女主关系不是修复，而是升级成更成熟的利益共同体
```

### 场景 2
```yaml
场景地点: 机场贵宾通道外 / 侧厅
出场角色: 姜栀、许知微、应岚、助理、安保
场景目标: 正面截到应岚，逼她承认“替换候选”是被预设过的执行逻辑
场景阻力: 应岚说话极稳，知道怎么把乙方包装成无辜执行人；安保只要看见姜栀过界，就会把她拖走；许知微不能长时间站在镜头正中
情节概要: 姜栀先拦应岚，果然被对方一句“你是来报复还是来问话”顶回来。许知微在第一轮镜头扑上来时先替她挡半步，把围观角度切掉，让姜栀拿到短短几分钟对话窗口。应岚起初只承认自己做过“归国包装”，直到姜栀把结算联、替换候选和舆论词路连成一线，才失口说出“你不是唯一一个候选，白月光回场前就该有一个女配出口位”。
关键对话: “你做的是包装，不是爱情。” / “替换候选本来就不止你一个，只是最后用你最顺手。” / “所以在你们眼里，我不是人，是方案。”
情绪变化: 对撞 -> 被回刺 -> 失口破口
场景代价/变化: 姜栀拿到了关键口供，但也第一次正面听见自己被当成产品位而不是关系位
```

### 场景 3
```yaml
场景地点: 机场侧厅转角 / 临时休息室门口
出场角色: 姜栀、应岚、许知微、林特助（电话/消息）
场景目标: 在应岚彻底收口前，再抢一层“可继续追”的入口，而不是硬逼最终操盘人
场景阻力: 应岚警觉极高，发现失口后立刻往“你们两家都默认知情”上推，试图让姜栀和许知微先内裂；林特助也只敢通过消息给半条线，不敢全透
情节概要: 应岚在意识到自己说多后，开始反咬两家都拿白月光和女配剧本做过公关试算，谁也别装无辜。姜栀没有被她带去和许知微当场翻脸，而是逼问“那套方案在哪留过底”。林特助这时只发来一个旧项目代号和云盘后缀，姜栀顺势套出应岚曾把简报和替换候选名单挂在“R-Blue”旧项目目录下。
关键对话: “你想我现在和她翻脸？那你就赢了。” / “你们做乙方最喜欢说自己只收钱不写命，那底稿呢？” / “R-Blue，那是你们自己最爱留底的地方。”
情绪变化: 反咬 -> 稳住裂口 -> 抢到下一层入口
场景代价/变化: 许知微必须当场忍下“你们两家都知情”这句脏话；姜栀则拿到更危险也更值钱的云盘入口线
```

### 场景 4
```yaml
场景地点: 机场出发层外
出场角色: 姜栀、许知微、媒体镜头、安保
场景目标: 把本章代价压到公开层，并把下一章明确挂到“R-Blue旧项目目录”
场景阻力: 第二轮偷拍视频和记者镜头已经等在外面，只要姜栀露出一点失控，就会被坐实“偷档报复”叙事
情节概要: 应岚被送进安检线前，姜栀没有再扑上去，而是收住。许知微替她挡了第一轮镜头后立刻收手，两人都清楚这不等于和解。林特助补来最后一句：R-Blue 目录今晚十二点会被远程清理。她们只能先离开机场，再决定谁去抢云盘、谁去挡下一波舆论。
关键对话: “你今天替我挡的是镜头，不是账。” / “够了，先把命口留到晚上。” / “十二点前，谁先摸到R-Blue，谁才算真赢这一轮。”
情绪变化: 短赢 -> 压住火 -> 更大倒计时
场景代价/变化: 双女主仍旧结盟，但裂口没有消失；下一章从“追执行人”转成“抢旧项目底稿”
```

## 推进校验

- 本章关键收益不是自动到手，而是在“偷拍视频第二轮 + 机场安保 + 乙方切词”三重阻力下抢出来的。
- 本章要明确展示：
  - 双女主站在同一边，但不是无条件信任
  - 机构角色和边缘角色各自只愿意帮半步
  - 应岚不是纸片反派，她会主动反咬两家知情
- 主角代价必须可见：
  - 继续背着“偷档报复”风险追人
  - 明知对面在等她失控，还要把火压住
- 本章只赢到：
  - “替换候选是预设产品位”的承认
  - 一个可继续追的 `R-Blue` 旧项目目录入口
- 章末必须把下一战压成倒计时，而不是让机场截人直接变终局。

## 关键元素

### 伏笔处理
- `闻策咨询 / 应岚`: 从外部执行名字推进到会说话、会切词、会反咬的真实角色
- `双女主裂口`: 本章不修复，只证明裂口状态下仍能合作

### 新伏笔
- `R-Blue` 旧项目目录：下一章抢底稿入口
- “你不是唯一一个候选”：说明这套剧本比姜栀想的更工业化

### 设定展示
- 机场不是浪漫追人场，而是带镜头、安保、时间差和舆论代价的现实战场
- 乙方执行链最擅长切词和甩锅，不会轻易把自己写成主谋

### 角色状态变更
- 姜栀：第一次正面听见自己被当成“产品位”
- 许知微：第一次在镜头前替姜栀挡半步，但依然不交全部底牌
- 应岚：从名字升级成有自保逻辑、有职业话术的节点角色
- 林特助：继续只帮半步，明确属于“自保型有限协助”

## Continuity Notes

- 不把许知微写成自动站边的白月光工具人。
- 不把应岚写成一拦就全招的纸片执行人。
- 不让姜栀靠情绪爆发拿到答案，她这章必须靠压火和追问取胜。
- 章末要让读者明确：机场这一战不是终点，真正值钱的是今晚十二点前的 `R-Blue` 目录。
