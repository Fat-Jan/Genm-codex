# 第005章章纲

## 基础信息

```yaml
章节序号: 第005章
章节标题: 借阅记录刚指向许家，她先被扣了偷档的帽子
字数规划: 3400
本章目标: 让B-17借阅记录从“搬运痕迹”推进到“外部执行名字”，同时把反扑真正压到姜栀身上。
预期爽点: 姜栀拿着照片反向卡住许家、双女主重新谈条件、旧样张箱里翻出外部执行线索、幕后人开始主动灭痕。
本章阻力: 贺氏会后舆论口要把姜栀定成偷档和失控的责任人，许家只想先灭火，B-17照片一旦外流就会反噬姜栀自己，许知微也不能把家里那条线全掀开。
本章代价: 姜栀要压住立刻发图反打的冲动，先把证据当谈判筹码；许知微必须交代自己为什么知道B-17，并冒着与母亲一侧彻底撕开的风险带姜栀去看旧样张箱。
本章局部收益: 确认B-17只是归国包装样张目录的一部分，真正执行这套叙事包装的人是外部顾问“闻策咨询”的应岚。
本章残留风险: “姜栀夜闯资料室”的偷拍视频开始流出，贺氏把她往偷档和商业报复方向推；应岚和更上层操盘者会先一步清理外围证据。
```

## 场景分解

### 场景 1
```yaml
场景地点: 次日清晨，姜栀公寓与热搜界面之间
出场角色: 姜栀、小夏（线上）、舆论/公司通告
场景目标: 把反扑从内部处理升级成公开舆论和程序性压迫。
场景阻力: 姜栀手里只有借阅记录照片，不能直接当终局证据；贺氏抢先发停职调查通告，并开始放“夜闯资料室”的偷拍视频。
情节概要: 姜栀醒来就看见停职调查、热搜和偷拍视频预热，小夏发来公司内部风向，提醒她有人在准备把“偷档”帽子扣死。姜栀没有立刻发图，而是先把照片发给许知微，只给一句：想要这张图不爆，就拿答案来换。
关键对话: “他们现在不是想解释昨晚，是想先把我做成贼。” / “这张图我还没发，不代表我不会发。” 
情绪变化: 清醒 → 压怒 → 主动设局。
场景代价/变化: 姜栀忍住立刻反击的冲动，改用照片谈判，意味着她必须暂时吞下一波舆论脏水。
```

### 场景 2
```yaml
场景地点: 许家艺术基金旧展陈库外的接待室
出场角色: 姜栀、许知微、林特助
场景目标: 逼许知微解释B-17的来源，同时测试林特助在许家线里到底站哪一层。
场景阻力: 许知微不能把母亲和家里的旧安排全掀开，林特助只愿意承认自己经手，不愿承认主导；姜栀若说重了，同盟会直接断。
情节概要: 姜栀带着照片赴约，开口就要答案。许知微承认自己早就见过“归国包装”催办单，所以昨晚才能报出B-17；林特助则强调自己只是签收执行，不是立项人。三人对线后，许知微提出交换：她带姜栀去旧展陈库看残留样张箱，姜栀不把借阅照先放出去。
关键对话: “你昨晚报出来的不是编号，是你早就认识这套东西。” / “我签过收，不等于我是写剧本的人。” / “你想要答案，就先跟我进库房，不然今天谁都别查了。”
情绪变化: 冷对峙 → 逼问 → 带条件的再合作。
场景代价/变化: 许知微第一次明确承认自己家里有人碰过这套旧包装；姜栀也被迫暂缓公开亮牌。
```

### 场景 3
```yaml
场景地点: 许家艺术基金旧展陈库
出场角色: 姜栀、许知微、林特助
场景目标: 在旧样张箱里抢出更具体的外部执行线索。
场景阻力: 库房里的核心样张已经被抽走，只剩目录、发票夹和交接便签；林特助不断催时间，因为有人正在往这边来。
情节概要: 三人翻出B-17同批样张夹，正文样张已空，只剩一张结算联和外包沟通便签，显示“闻策咨询-应岚”负责“归国包装/替换候选/舆论词路”。姜栀刚拍下发票编号，楼下就传来人声和脚步，库房被迫中断清查。
关键对话: “他们拿走了样张，只剩付钱的人和做事的人。” / “应岚不是你们家的人。” / “这就对了，脏活从来不写在自己名字上。”
情绪变化: 紧张搜查 → 短促推进 → 危机压近。
场景代价/变化: 线索从家族内部搬运推进到外部执行人，但真正立项者仍未露面。
```

### 场景 4
```yaml
场景地点: 基金楼下与车边
出场角色: 姜栀、许知微、林特助、媒体/公司法务
场景目标: 把本章的代价落到公开层面，并给下一章挂更硬的钩子。
场景阻力: 姜栀刚拿到新线索，贺氏法务就把“夜闯资料室”偷拍视频放给营销号；媒体和法务都开始追她。
情节概要: 三人刚下楼，姜栀手机就弹出偷拍视频热搜，贺氏法务同步发来律师函。许知微想先把她带离现场，林特助却提醒“应岚今天下午就会飞走”。姜栀看着热搜，决定这张锅先背半张，但下一步必须在应岚消失前把人拦住。
关键对话: “他们把我往贼上挂，是想让之后所有证据都像报复。” / “你现在先走，别被他们拍第二轮。” / “走可以，人不能放飞。”
情绪变化: 短暂推进 → 代价砸实 → 下一章强钩子。
场景代价/变化: 姜栀正式被公开扣上“偷档”帽子；下一章目标从内部查档升级为外部追人。
```

| 结尾 | 偷拍视频和律师函落地，应岚即将离场，下一章转入追人。 | 700 | 反噬、强钩子 |

## 推进校验

- 本章不让B-17直接通向最终操盘者，只推进到“外部执行人”这一层。
- 证据不是白送，而是靠姜栀压住公开发图的冲动，用照片换来进旧展陈库的机会。
- 本章代价必须显眼：姜栀被公开往“偷档”上挂，许知微则被迫承认自己早知道B-17这套东西。
- 双女主关系不能回甜，本章只是重新成交，不是消除裂口。
- 章末钩子要落在“应岚要飞走”与“偷拍已挂热搜”这两重时间压力上。

## 关键元素

### 伏笔处理
- `B-17借阅记录`: 从搬运路径推进到结算联和执行顾问名字，证明昨晚抢的不是死线索。
- `许家卷入深度`: 本章把许家定位从“可疑源头”推进成“经手方之一”，但不直接盖章主谋。

### 新伏笔
- `闻策咨询 / 应岚`: 新的外部执行线，后续可追到具体做包装和舆论词路的人。
- `偷拍视频热搜`: 把内部斗争推到公开舆论层，为后续“证据像报复”埋压制风险。

### 设定展示
- `许家艺术基金旧展陈库`: 说明许家一侧也留过归国包装与旧样张，补足许知微为什么知道B-17。
- `外部顾问执行链`: 把豪门关系剧本进一步压成可追责的执行流程，而不是抽象阴谋。

### 角色状态变更
- `姜栀`: 从夜里抢纸面痕迹的人，转成手握半张照片却被公开往“偷档者”上钉的人。
- `许知微`: 首次承认自己早看过这套旧包装的一角，她的隐瞒被正式摆上桌。
- `林特助`: 从名字线索升级为真实出场的执行层角色，但仍保留“只是经手不是立项”的模糊地带。
