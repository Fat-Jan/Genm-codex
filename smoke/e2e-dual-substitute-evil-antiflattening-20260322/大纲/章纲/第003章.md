# 第003章章纲

## 基础信息

```yaml
章节序号: 第003章
章节标题: 他们等她发疯，她和白月光一起换了剧本
字数规划: 3600
本章目标: 在发布会的日光下完成双女主公开联手、撕开替身叙事，且保留代价与后置风险。
预期爽点: 她们联手改流程、证据被现场亮出、媒体开始把目光从“她会疯”转到“谁写剧本”上。
本章阻力: 贺承川与两大家族的旧流程扼杀任何临时变更、台下舆论只等姜栀当众失态、许知微和姜栀之间的联盟仍属试探。
本章代价: 现场曝光会招来董事会质疑、媒体拦截、许知微的基金关系被动暴露，姜栀的个人形象仍需在短时间内恢复操控。
本章局部收益: 公开播放替身筛选简报、在发布会台上成功把旧流程改掉、在记者与股东面前把两人名义对等化。
本章残留风险: 贺承川和后台力量会在会后发动反扑，媒体依旧把“恶毒女配”标签抬出来，匿名操盘者更高一层仍未现身。
```

## 场景分解

### 场景 1
```yaml
场景地点: 发布会后台化妆间
出场角色: 姜栀、许知微、贺承川、助理小夏
场景目标: 压住发布会准备的节奏，展示贺承川试图维持旧流程、姜栀与许知微互相试探、组成临时共识的过程。
场景阻力: 贺承川坚持流程不变，将“白月光回归”“恶毒女配保持克制”当作必须遵守的剧本；台下媒体预期和助理的焦虑加剧。
情节概要: 贺承川再次希望姜栀按部就班致辞；许知微通过新的流程卡打破白月光标签，把两个名字摆在同等位置；姜栀冷静回应，场内压力累积。
关键对话: “今天别再临时起意。” / “我是许家艺术基金负责人。” / “今天谁也别替谁让位。”
情绪变化: 压力 → 紧张 → 暂时被打破的秩序。
场景代价/变化: 贺承川在台前首次被两个女人同时逼到失控边缘，发布会节奏被迫进入非计划路线。
```

### 场景 2
```yaml
场景地点: 发布厅舞台
出场角色: 姜栀、许知微、贺承川、董事/媒体
场景目标: 在公众面前完成台词重写，曝光替身结果，转移注意力。
场景阻力: 灯光、镜头、董事会就在台前，任何情绪失控都能被当作“恶毒女配发疯”；贺承川试图制止，舆论随时可以扭回老路。
情节概要: 姜栀主动把戒指放下，逐字斩断“替身”叙事；许知微走上前宣布自身身份、把两人项目并列；后台大屏直接展示筛选简报，震撼全场。
关键对话: “我不是谁的替身。” / “我回来是和姜栀一起把栖序独立系列拉出来。” / “优先考虑气质冷、轮廓近似许小姐。”
情绪变化: 拉满冷静 → 引爆震撼 → 局面短暂倾斜。
场景代价/变化: 两人被迫暴露身份，恰巧把媒体聚焦拉转，但也让幕后势力看清她们是眼中钉，今后会被更严密盯着。
```

### 场景 3

```yaml
场景地点: 发布会后台
出场角色: 姜栀、许知微、舆情/匿名势力
场景目标: 接到匿名邮件提示还有更高设局人，把风险向后推到第4章。

情节概要: 会后荒乱，姜栀收到“你们今天放出来的只是第一页”的邮件，附带更旧筛选批注；两人面面相觑，意识到顶层操盘者还没现身，必须准备下一波。
关键对话: “你们今天放出来的，只是第一页。” / “看来我们今天掀开的，还不是最上面那层。”
情绪变化: 短暂喘息 → 震惊的冷静 → 决策性的期待。
场景代价/变化: 公开反击带来即时混乱，留下的焦点将她们推向幕后的更强掌控者，也提醒姜栀/许知微必须小心不被新一轮审视碾压。
```

## 推进校验

- 本章关键收益不是“白月光回归”而是把旧流程拆成可质疑的材料——现场公开、流程表、简报都成为证据。
- 本章真正阻力来自贺承川和两家人对媒体节奏的控制、公众对“女配失控”的期待以及董事会在现场的目光。
- 主角付出的代价是被媒体截取“发疯”瞬间、重塑形象被质疑、许知微的身份被挂上“反制”的标签。
- 这次“赢”后更危险的事是贺承川/两大家族会把她们定位成集合体级风险，幕后设局人也会加强调查。
- 如果这是第3章，则必须明确“为什么这不是全赢”：现场虽然联手，但仍留下媒介质疑、幕后风险、机构反扑的痕迹。

## 关键元素

### 伏笔处理
- `替身筛选简报`: 现场投屏后将是后续调查的直接证据素材，档案页的出处会成为第4章继续追查的线索。
- `发布会流程备注`: 显示旧流程被重新协调，为将来某方偷偷换线留下痕迹。

### 新伏笔
- `匿名邮件“你们今天放出来的只是第一页”`: 预示幕后还藏着更早的操作批注、一位真正挑人者仍在场。
- `媒体/董事的反应被控制`: 说明会后评估不会简单落幕后，她们的每一个动作都有人在监督。

### 设定展示
- `栖序珠宝春季发布会`: 通过舞台、流程卡、投屏，全方位呈现贺家与许家构建的公共场域，后续可以用作真假代言翻牌的对照。
- `白月光与恶毒女配的故事线`: 通过台上台下的冲突、镜头捕捉、舆论期待，展示这个故事为什么需要被“换剧本”。

### 角色状态变更
- `姜栀`: 从被动拉扯的女配，升级为敢在台前撕掉戒指与旧叙事的自我掌控者，但会被公司/媒体标记为“危险”。
- `许知微`: 从回归白月光的身份转为主动破掉标签的操盘者，但她的资历与基金身份被摊到台面，今后容易被排查。
