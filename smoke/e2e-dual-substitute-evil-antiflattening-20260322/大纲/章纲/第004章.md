# 第004章章纲

## 基础信息

```yaml
章节序号: 第004章
章节标题: 她们刚赢一场，公司先收了她的门禁
字数规划: 3400
本章目标: 写出发布会后的第一轮反扑，让双女主同盟第一次明显失衡，并抢到一张只够继续追查的借阅记录。
预期爽点: 姜栀不肯把解释权交回去、贺氏第一时间反扑、资料室借阅记录被抢出来、旧剧本第一次露出更具体的搬运路径。
本章阻力: 董事办与公关口要把发布会定义成姜栀个人失控，许家和贺家同时开始切割风险，替身档案正在被连夜清理，姜栀和许知微的策略首次出现分歧。
本章代价: 姜栀必须交出工牌和工作机、面临停职调查，还得冒着被监控抓现行的风险去抢资料；许知微要在公开场合先和姜栀拉开半步，保住自己继续查档案的入口。
本章局部收益: 拿到旧档案借阅记录和文件编号，确认替身筛选文件曾在贺氏董事办与许家办公室之间被调动过。
本章残留风险: 借阅记录直接指向许家一侧，同盟出现第一道裂口；姜栀失去公司内权限；她们拿到的仍只是搬运痕迹，不是最终操盘者。
```

## 场景分解

### 场景 1
```yaml
场景地点: 发布会后台与临时会议区
出场角色: 姜栀、许知微、贺承川、罗秘书、小夏、品牌公关
场景目标: 把发布会后的第一轮反扑落地，展示贺氏要立刻收走姜栀的解释权和内部权限。
场景阻力: 董事办把现场公开文件定性为违规披露，公关团队急着找替罪羊，许知微不能在此刻和姜栀完全站成一排。
情节概要: 会后混乱里，罗秘书代表董事办要求姜栀交出工作机和工牌，接受内部调查；贺承川试图让她签“个人情绪失控说明”止损。许知微表面切开半步，只要求所有档案移交法务封存，姜栀当场意识到这场合作不是无条件站边。
关键对话: “你们想收的不是工牌，是我的解释权。” / “今晚这件事，贺氏必须先有统一口径。” / “我只能帮你保住档案入口，不能替你把今晚一起认下来。”
情绪变化: 会后余震 → 被围堵的冷怒 → 第一次明显失衡。
场景代价/变化: 姜栀被迫交出工牌和工作机，停职风险落地；许知微和她之间第一次出现公开距离。
```

### 场景 2
```yaml
场景地点: 贺氏大楼地下停车场
出场角色: 姜栀、许知微
场景目标: 把“合作”从爽感拉回交易，明确两人第一次策略分歧，并锁定下一步行动目标。
场景阻力: 姜栀刚被切走权限，直觉上认为许知微在拿她做靶子；许知微必须保住许家那边仅剩的档案入口，不能把全部底牌交出来。
情节概要: 姜栀在停车场拦住许知微，直接追问她为什么当场拉开距离。许知微解释，若她也被一起钉成“内部泄密人”，许家会立刻封死文化基金和母亲办公室的旧档渠道。她给出一条线索：旧档案编号是“B-17”，今晚若不去贺氏资料室看借阅登记，明早就会被抽干净。两人争执后达成新交换：姜栀去抢借阅记录，许知微去拖住许家与董事办的会后封口。
关键对话: “站边可以快，解释权不能一起丢。” / “你刚才不是保我，是保你自己的门。” / “对，我先保门，不然我们连第二页都摸不到。”
情绪变化: 受伤与怀疑 → 对刺 → 再次成交但更冷。
场景代价/变化: 双女主同盟升级为明确交易关系；姜栀接下夜里抢档案的风险。
```

### 场景 3
```yaml
场景地点: 贺氏品牌资料室
出场角色: 姜栀、小夏、安保
场景目标: 让姜栀用冒风险的行动抢到第一张“够用但不够终结”的纸面证据。
场景阻力: 资料室正在被封存，门禁即将注销，安保和董事办的人已经开始逐层清点；小夏不敢完全站队，只能帮半步。
情节概要: 小夏出于自保和保团队，悄悄提醒姜栀资料室还没封死。姜栀赶在门禁失效前进资料室，却发现核心文件已被抽走，只剩借阅登记和碎纸桶里的半页目录。她拍下借阅记录，看到“B-17归国包装样张”曾被罗秘书调出，并经“许太太办公室林特助”签收。安保追到门口，姜栀只能抢下一张照片，工牌当场被注销。
关键对话: “姜总，我不是帮你，我是不想整条品牌线陪葬。” / “他们拿走了正文，只给我剩下一行签收。” / “够了，先拿活口，再想怎么审。”
情绪变化: 潜入的紧绷 → 抢到痕迹的短促爽点 → 更深的怀疑与危机。
场景代价/变化: 姜栀彻底失去公司权限；借阅记录把许家一侧卷进来，同盟出现第一道真正裂口。
```

## 推进校验

- 本章关键收益不是自动送证据，而是姜栀在停职风险落地后，冒着权限注销和被抓现行的代价去抢借阅记录。
- 本章真正阻力来自董事办、公关口、许家封口需求，以及双女主第一次真实的策略分裂。
- 主角为这次收益付出的代价是交出工牌和工作机、被停职调查、失去公司内部安全位置。
- 这次赢到的只是“档案搬运路径”，不是最终操盘者；证据足够把下一章推下去，不足以终结主线。
- 章末要把风险留在“借阅记录指向许家一侧，同盟是否还稳”上，而不是把答案全给出来。

## 关键元素

### 伏笔处理
- `匿名邮件“只是第一页”`: 本章转化为更具体的文件编号与借阅路径，证明匿名人不是随口挑拨。
- `双女主同盟是否稳定`: 本章第一次落成结构性裂口，不再只是口头互防。

### 新伏笔
- `B-17归国包装样张`: 指向更完整的旧包装文件，后续可继续追到操盘链上层。
- `许太太办公室林特助`: 把许家一侧卷入，但仍未说明是参与者、搬运者，还是替别人背签的人。

### 设定展示
- `贺氏内部收口逻辑`: 通过董事办、公关、法务封存，展示现实情感/豪门关系线里的机构性反扑。
- `公司资料室与借阅登记`: 把“替身剧本”从抽象关系叙事，压成具体可追的旧包装流程。

### 角色状态变更
- `姜栀`: 从“发布会上赢了一轮的人”变成“失去内部权限、只能冒险自己抢证据的人”。
- `许知微`: 从可并肩的搭档变成必须先保住许家入口的合作者，她的独立利益第一次压过关系爽点。
- `小夏`: 从纯助理升级为有自保动机的边缘参与者，她帮的是团队生路，不是单纯站队。
