# 第003章章纲

## 基础信息

```yaml
章节序号: 3
章节标题: 太子只问一句：谁敢再换她的婚书
字数规划: 3200-3800
本章目标: 兑现“太子开口”的书名承诺，但把收益控制在‘规矩层被改写’而非‘旧案答案到手’
预期爽点: 东宫试问 / 口谕落地 / 柳氏第一次真正压不住
本章阻力: 东宫不替她私人伸冤、伯府仍想压成内宅误会、旧人口随时会被灭口
本章代价: 谢云昭必须主动把局从婚书局抬成规矩局，也因此把沈氏旧账和崔妈妈一起推到危险区
本章局部收益: 拿到东宫口谕与令签，婚书和账册暂由她自持，三日内可带旧人回话
本章残留风险: 柳氏会先一步去庄子灭口；东宫给的是门，不是包办到底；旧案真正入口才刚打开
```

## 场景分解

### 场景 1
```yaml
场景地点: 东宫宫门外 -> 偏殿
出场角色: 谢云昭、伯爷、柳氏、谢明珠、年长嬷嬷、内侍
场景目标: 先堵死伯府“长辈替答”的路
场景阻力: 伯爷想跟进、柳氏想隔门控局、东宫不天然站谢云昭
情节概要: 东宫只点二姑娘入内，把伯爷和柳氏压在门外。谢云昭因此必须独自承担成败，不能靠任何人兜话。
关键对话: “殿下没点旁人。” / “安平伯若觉得规矩能改，尽可自己进宫门试试。”
情绪变化: 压迫 -> 被单独拎上桌 -> 稳住
场景代价/变化: 她失去所有“长辈在场帮她缓冲”的余地
```

### 场景 2
```yaml
场景地点: 东宫偏殿案前
出场角色: 谢云昭、内侍、屏风后之人
场景目标: 把“被夺婚事的伯府二姑娘求做主”改写成“谁敢坏规矩、吞旧账”
场景阻力: 东宫不要哭诉，只要有价值的局；她若只讲婚事，会被压回内宅争风
情节概要: 谢云昭主动说自己不是来求东宫替她争男人，而是来问：婚书和账册为何能一起被人吞下。她把婚书局抬成规矩局，赌东宫在意的不是她这个人，而是伯府有没有人敢借婚书吞旧账。
关键对话: “臣女不敢求东宫替臣女争男人。” / “臣女争的不是婚事本身，是谁也不能把规矩踩成一张废纸。”
情绪变化: 试探 -> 顶住 -> 说动上位规则
场景代价/变化: 她把局抬大了，也自断了“回去再慢慢哭婚事”的退路
```

### 场景 3
```yaml
场景地点: 偏殿 -> 宫门外
出场角色: 谢云昭、内侍、屏风后之人、伯爷、柳氏、谢明珠、青枝
场景目标: 给出本卷第一份能落回家门的硬保护，并立刻挂出庄子旧人线
场景阻力: 口谕并不等于东宫替她走完路；她还缺旧人和活证
情节概要: 太子开口，婚书与账册暂由谢云昭自持，三日内把崔妈妈带来回话。她因此第一次拿到真正能带回伯府的刀。可刚出宫门，她就得知柳氏的人已经赶往庄子，说明这份收益本身就把旧人口推成了生死线。
关键对话: “婚书既写她名，谁再换，按欺东宫论。” / “下一次再进东宫，带实证，不带眼泪。”
情绪变化: 短促硬爽 -> 更冷的清醒 -> 立刻转急
场景代价/变化: 她拿到了门，也把旧人口和沈氏账线彻底暴露了
```

## 推进校验

- 本章硬收益不是自动到账，而是谢云昭主动把自己从“哭婚事的人”改写成“替规矩发问的人”才抢来的。
- 本章真正的阻力来自：东宫不管私人伸冤、伯府想压回内宅、旧人线随时会被灭口。
- 主角的代价是：她把局抬大了，也把崔妈妈和沈氏旧账一起推到危险区。
- 本章不是全赢：她拿到了门，但还没拿到人，更没拿到旧案答案。
- 如果这是第3章：硬兑现是“令签到手”，不是“旧案查清”。

## 关键元素

### 伏笔处理
- 东宫为何介入：从婚书异动升级成“规矩和旧账一起被吞”
- 沈氏旧账：第一次被东宫认可为继续往下查的价值线

### 新伏笔
- 崔妈妈的生死赛跑：本章章末立刻接入 -> 第004章回收
- 东宫给门不替走：后续旧档局和更大旧案入口 -> 第009章后深化

### 设定展示
- 东宫权力边界：给规则、给门、给令签，但不替她私人伸冤到底
- 谢云昭的打法：不求怜悯，只求把别人最不敢上桌的东西一起拖上桌

### 角色状态变更
- 谢云昭：从伯府受压的二姑娘，升级为可持东宫口谕回府的人
- 伯爷：第一次意识到事情已无法按家法收口
- 柳氏：从压婚书，转为抢时间灭口旧人

## Continuity Notes

- 延续 `恶女 x 宫斗宅斗` 路线：不求别人替她做主，而是借规矩反压伯府。
- 本章收益必须控制在“规则升级 + 入口打开”，不能直接把旧案线打穿。
- 章末要非常明确：真正的新战场不是东宫，而是庄子和旧人口。
