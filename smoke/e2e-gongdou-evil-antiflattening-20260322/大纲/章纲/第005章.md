# 第005章章纲

## 基础信息

```yaml
章节序号: 5
章节标题: 她带着东宫令签，直接踩进了外院账房
字数规划: 3200-3800
本章目标: 把战场从庄子旧人线切回外院账房线，抢下顾家旧回帖暗记并第一次把怀疑抬向伯爷
预期爽点: 东宫令签压外院 / 火盆前抢回帖 / 周管事第一次失口
本章阻力: 周管事正准备烧账封口，伯爷会护外院账房，谢云昭一旦慢半步，证据就没了
本章代价: 谢云昭必须把崔妈妈和东宫令签一起拖进外院男人账线，等于把父亲那层也彻底拖下水
本章局部收益: 抢下顾家旧回帖暗记，坐实外院线留有真凭据，并逼出“账页已送进伯爷书房”
本章残留风险: 真正那页账仍不在手上，且伯爷已从幕后怀疑对象变成正面风险
```

## 场景分解

### 场景 1
```yaml
场景地点: 安平伯府外院侧门 -> 外院账房院
出场角色: 谢云昭、青枝、崔妈妈、小厮、周管事、账房先生
场景目标: 抢在账房烧净前直接撞进去
场景阻力: 外院不是她平时能进的门；账房已开始烧纸；下人仍先看周管事眼色
情节概要: 谢云昭不先安顿人，直接带车撞外院，逼小厮开门，又在账房院里当场看见火盆和待烧纸堆。她必须用东宫令签把“账房男人们的场子”硬压成自己能翻的地方。
关键对话: “不敢就把门开大。” / “今日我拿着这块牌子站在这里，不是来和你们商量。”
情绪变化: 抢时间 -> 压场 -> 账房秩序翻面
场景代价/变化: 她为了抢证据，主动把崔妈妈与东宫令签一起拖进外院男人账线，今后再难把父亲那层摘干净
```

### 场景 2
```yaml
场景地点: 外院账案前
出场角色: 谢云昭、周管事、崔妈妈、账房众人
场景目标: 在火盆边抢出回帖，并逼周管事露第一道口风
场景阻力: 顾家旧回帖混在待烧废纸里；周管事会抢、会赖、会把一切都推成旧人疯话
情节概要: 崔妈妈当场认出回帖暗记的落点，谢云昭借她这句“旧仆认暗记”的权威，在火盆边一张张翻纸，硬抢出旧回帖。周管事急到伸手去抢，反而暴露自己怕的不是她发疯，而是那页账和回帖对上。
关键对话: “来人，把火盆端开。” / “你上月翻我床板夹层，抢走的那页单纸，送去哪了？”
情绪变化: 绷死 -> 抢到回帖 -> 周管事第一次被逼失态
场景代价/变化: 谢云昭抢到了回帖，却也让外院所有人都明白她现在盯的已不是后宅小婚事
```

### 场景 3
```yaml
场景地点: 账房院内
出场角色: 谢云昭、周管事、伯爷、崔妈妈
场景目标: 借伯爷到场，把怀疑第一次从柳氏抬到家主本人
场景阻力: 伯爷会本能护外院；周管事也会试探哪句话能保命
情节概要: 伯爷赶到后想收场，谢云昭却不再只问“回帖在不在”，而是顺着周管事的眼神去盯“那页账送去哪了”。周管事在伯爷和她之间摇摆，最终失手吐出“那页早送进老爷书房暗匣”，把真正下一战钩到伯爷身上。
关键对话: “女儿若今日不闹，父亲明日拿什么去和东宫回话？” / “那页不在小的手里，送进老爷书房暗匣了。”
情绪变化: 对峙 -> 试探 -> 失口抬层
场景代价/变化: 她没拿到账页本身，但从这一刻起，父亲不再只是默认压制者，而是局中人
```

| 结尾 | 下一战从后宅与外院，正式推向伯爷书房 | 250-350 | 更险 |

## 推进校验

- 本章关键收益不是自动到账，而是在“火盆马上烧净”的时钟下抢回来的。
- 本章阻力来自账房封口、伯爷护外院、周管事边赖边试探。
- 主角代价是：为了拿回帖，她把父亲这层也彻底拖下水。
- 本章只赢到“回帖在手 + 伯爷书房浮上台面”，没有拿到那页账。
- 这次赢完以后，更危险的是：家主本人已经成了下一层正面风险。

## 关键元素

### 伏笔处理
- 顾家旧回帖暗记：从“知道有暗记”推进到“暗记真被抢到手”
- 外院账房线：从庄子旧人线接上男人账线

### 新伏笔
- 伯爷书房暗匣：第006章主战场
- 外院并非柳氏一人能控：说明婚书局背后还有更大默认者

### 设定展示
- 东宫令签能压账房秩序，但不能自动生出证据
- 崔妈妈不仅是旧仆，更是当年真正会认账的人

### 角色状态变更
- 谢云昭：第一次把刀从后宅抬到外院男人账线上
- 周管事：从模糊的脏手，升级为可被撬口的中层经手人
- 伯爷：从护脸面，升级为真正的可疑对象

## Continuity Notes

- 延续 `恶女 x 宫斗宅斗` 路线：先抢证据，再逼人失态，最后顺势抬高对手级别。
- 本章爽点不是训下人，而是“她带着东宫令签硬翻男人们的账”。
- 章末必须清楚告诉读者：真正要抢的那页账，还在伯爷的书房门后面。
