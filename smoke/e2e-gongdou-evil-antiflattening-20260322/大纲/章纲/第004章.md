# 第004章章纲

## 基础信息

```yaml
章节序号: 4
章节标题: 东宫令签压下来，先跪的不是我
字数规划: 3000-3600
本章目标: 把东宫给的“规矩保护”第一次兑换成现实收益，抢下活证人崔妈妈
预期爽点: 庄子压人反被压 / 东宫令签第一次落地 / 崔妈妈活着被带走
本章阻力: 柳氏的人先到庄子，庄头见风使舵，崔妈妈自己也不敢轻易开口
本章代价: 谢云昭必须公开把东宫令签压到庄子和伯府家丁头上，等于彻底暴露自己正在抢活证
本章局部收益: 抢下崔妈妈，确认关键账页曾被人先拿走，并把怀疑推向外院周管事
本章残留风险: 账页不在庄子，且对方已抢先一步把关键东西送回城中
```

## 场景分解

### 场景 1
```yaml
场景地点: 城郊庄子门外 -> 院内
出场角色: 谢云昭、青枝、庄头、管事娘子、粗使婆子
场景目标: 先把“挪院静养”的假话撕开
场景阻力: 柳氏的人已先一步到位，庄头只认谁势大，所有人都想装糊涂
情节概要: 谢云昭赶到庄子时，崔妈妈已被往后山旧屋方向抬。她不能光靠喝止，必须先当众拆穿“静养”是假、“送死”是真，再亮东宫令签，把庄头和管事娘子一起钉在规矩下。
关键对话: “谁准你们抬人？” / “后山旧屋什么时候开始住活人了？”
情绪变化: 紧急 -> 拦截 -> 现场倒向谢云昭
场景代价/变化: 谢云昭第一次把东宫令签公开压到伯府庄子线上，也因此让柳氏知道她已真在抢活证
```

### 场景 2
```yaml
场景地点: 庄子病屋
出场角色: 谢云昭、崔妈妈、青枝
场景目标: 从崔妈妈口中抢到第一层有效信息
场景阻力: 崔妈妈病弱、惊惧、怕自己今晚活不过去，不会一次把底牌全说完
情节概要: 谢云昭不先安慰，而是先说明自己今天不是空手来。她逼崔妈妈在“现在说，或今晚就死”之间作选择，抢出两条关键信息：一页实账被人先拿走、顾家旧回帖上还有一道能对上账页编号的暗记。
关键对话: “我今日不是空手来的。” / “那页不在我手里，回帖上还有记。”
情绪变化: 绷死 -> 犹疑 -> 被逼开口
场景代价/变化: 信息抢出来了，但崔妈妈的存在也因此变成必须立刻转移的硬负担
```

### 场景 3
```yaml
场景地点: 庄子院门 -> 马车
出场角色: 谢云昭、庄头、小厮、伯府家丁
场景目标: 在伯府家丁赶到前，先把活证人和主动权一起带走
场景阻力: 伯府家丁已赶到，领头的是周管事的人，名义上是“带回去问话”，实际是趁乱抹口
情节概要: 谢云昭不能留下来和伯府来人讲理，只能借东宫令签和庄头怕事的本能，逼庄子后门先开，把崔妈妈先装车带走。她因此第一次清楚：现在要咬的，不只是柳氏，还有外院里那条更脏的手。
关键对话: “今日这庄子里，你听谁的？” / “谁敢拦，就把名字记下来带去东宫。”
情绪变化: 更急 -> 压人 -> 抢路成功
场景代价/变化: 她抢走了人，却也让对方知道自己已经顺着庄子线摸到了外院线
```

## 推进校验

- 本章关键收益不是自动到账，而是人命和信息一起抢出来的。
- 本章阻力来自庄头见风使舵、柳氏先到一步、伯府家丁紧跟着来。
- 本章代价是谢云昭第一次公开把东宫令签压到庄子与伯府家丁头上，彻底暴露自己在抢活证。
- 本章只赢到“人抢回来 + 线索抢出来”，并没有拿到账页本身。
- 这次赢完更危险的是：对方知道她已摸到外院账线，回城后的封口一定更快。

## 关键元素

### 伏笔处理
- 沈氏旧账：从“旧账存在”推进到“账页 + 回帖暗记两段式证据”
- 东宫令签：第一次从口头保护兑换成现实抢人收益

### 新伏笔
- 周管事：第一次被明确钉成庄子线背后的下一层手 -> 第005章回收
- 顾家旧回帖暗记：成为下一章抢账房证据的入口

### 设定展示
- 伯府庄子线真正听的不是规矩，而是谁现在更大
- 东宫令签的作用是借规矩压人，不是自动替主角完成所有动作

### 角色状态变更
- 谢云昭：第一次从“拿到刀”变成“真用刀的人”
- 崔妈妈：从死口，变成必须立刻护送的活证
- 柳氏：从内宅压人，升级为开始失去线头的人
