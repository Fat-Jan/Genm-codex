# 第002章章纲

## 基础信息

```yaml
章节序号: 2
章节标题: 东宫问婚，先低头的不是我
字数规划: 3000-3600
本章目标: 把婚书争夺从“后宅姐妹争婚”推进成“必须当众开匣验真的家门公案”
预期爽点: 柳氏急压失手 / 原婚书归手 / 东宫正式点名
本章阻力: 柳氏想压消息，伯爷想保脸面，原婚书和旧册都不在谢云昭手里
本章代价: 谢云昭必须当众撕开家门体面，把自己推上无法回撤的明面位置
本章局部收益: 原婚书归属被坐实，谢云昭拿到次日入东宫回话的资格
本章残留风险: 柳氏今夜会去掐旧人口，婚书局开始转成旧账局
```

## 场景分解

### 场景 1
```yaml
场景地点: 安平伯府正厅
出场角色: 谢云昭、柳氏、谢明珠、伯爷、族叔、管家
场景目标: 让东宫来人的压力先压乱伯府内部节奏
场景阻力: 柳氏想把谢云昭按回屋，伯爷想把局压成私下解释
情节概要: 东宫口谕一到，柳氏急着说“冷静”，伯爷急着护住伯府体面。谢云昭不抢着哭诉，而是先把最怕的人和最怕见光的东西钉出来。
关键对话: “冷静？母亲是想让我再病一场？” / “东宫点名，已经不是家里误会。”
情绪变化: 慌乱 -> 急压 -> 谢云昭把节奏抢回来
场景代价/变化: 她主动顶撞柳氏，也等于切断了“回房私了”的退路
```

### 场景 2
```yaml
场景地点: 正厅案前
出场角色: 谢云昭、柳氏、谢明珠、伯爷、族叔、媒人
场景目标: 把“婚书原件必须开匣”变成当场无法回避的动作
场景阻力: 原件和账册都在伯府手里；若压不住场，谢云昭会被重新打回“庶女胡闹”
情节概要: 谢云昭借族叔在场和东宫点名的压力，逼伯爷和族叔一起开匣。柳氏试图用礼数和“姑娘讲钱丢脸”压回去，反而坐实她最怕人看见的不是婚事，而是账。
关键对话: “请族叔和父亲一起开匣。” / “母亲既嫌我讲钱，为何又拿这些东西定我的死活？”
情绪变化: 卡住 -> 绷紧 -> 开匣见真
场景代价/变化: 家门体面被当众撕开，伯爷第一次正面意识到谢云昭不是不会争，而是在等时机
```

### 场景 3
```yaml
场景地点: 正厅门口
出场角色: 东宫内侍、谢云昭、柳氏、谢明珠、伯爷
场景目标: 给出本章硬收益，同时立刻挂出更危险的新战场
场景阻力: 东宫不会替她伸冤到底，只会按自己的规矩点名办事
情节概要: 东宫内侍亲自进门，重申次日巳时前请二姑娘携婚书与信物入东宫回话，并把旧账一起带上。谢云昭因此第一次拿到真正落回自己手里的主动，但也清楚今夜之后旧人一定会被先下手。
关键对话: “既然姑娘会问这句，那就都带上。” / “既然点的是我，那就由我自己去。”
情绪变化: 硬爽 -> 更冷的清醒
场景代价/变化: 谢云昭拿到入宫资格，但沈氏旧账和旧人口也因此一起暴露
```

| 结尾 | 资格到手，但旧人口当夜就成高危对象 | 250-350 | 更险 |

## 推进校验

- 本章关键收益不是自动到账，而是借东宫点名、族叔在场和当众开匣硬抢回来的。
- 本章真正的阻力来自柳氏压消息、伯爷保体面、原件不在手上。
- 主角为了这次收益付出的代价，是把家门体面和自己一起推上明面。
- 这次赢完以后，更危险的是今夜旧人口和旧账会被反扑。
- 如果这是第2章：卡点就在“能不能让婚书原件真的见光”。

## 关键元素

### 伏笔处理
- 生母旧账：从旧抄延伸到账册与聘银单据，说明婚书和账不是两件事
- 东宫介入：从传话升级成正式点名回话

### 新伏笔
- 旧人口的生死赛跑：本章末立起 -> 第003-004章承接
- 东宫不只看婚书：旧案入口打开 -> 第003章深化

### 设定展示
- 伯府家门规矩第一次在东宫口谕前失灵
- “二姑娘”的位置第一次被正式点给谢云昭

### 角色状态变更
- 谢云昭：从被动挨压，升级为正式被东宫点名回话的人
- 柳氏：从内宅掌控局势，转为急于当夜补最后一道手尾
- 伯爷：从护脸面，转为被拖下水

## Continuity Notes

- 延续 `恶女 x 宫斗宅斗` 路线：不撒泼，不硬撞，用礼法、账目、话柄和时机反杀。
- 本章收益要控制在“婚书归属坐实 + 入东宫资格”，不能把旧案直接掀完。
- 章末必须让读者明确：最危险的不是明日入东宫，而是今夜谁会先去灭旧人。
