# 第006章章纲

## 基础信息

```yaml
章节序号: 6
章节标题: 那页账不认柳氏，认的是父亲的书房锁
字数规划: 3200-3800
本章目标: 用顾家旧回帖暗记和崔妈妈证词，正式把婚书局从柳氏线咬成伯爷线
预期爽点: 周管事塌口 / 书房暗匣被逼开 / 伯爷第一次真正失去遮掩
本章阻力: 伯爷会护书房和外院，周管事会拼命把账推回柳氏，家门礼法和“不孝”名头都会反压主角
本章代价: 谢云昭为了打开书房，必须当众把父亲拖进局，并彻底撕掉“还给家门留脸”的可能
本章局部收益: 周管事塌口、伯爷知情被坐实、书房暗匣里真有旧账痕迹
本章残留风险: 真正完整账页仍未到手，且说明还有第三只手先一步摸过书房和旧账
```

## 场景分解

### 场景 1
```yaml
场景地点: 正院花厅
出场角色: 谢云昭、伯爷、柳氏、谢明珠、族叔、周管事、崔妈妈
场景目标: 把外院账房线从“下人脏手”推进成“家主默认”
场景阻力: 伯爷会本能护账房，柳氏会继续把一切往后宅女人手上按，周管事会等最后一道保命符
情节概要: 谢云昭不再满足于“周管事有问题”，而是借回帖暗记和崔妈妈口证逼问：周管事为什么敢动沈氏旧账，为什么被逼急时第一反应是把那页账供进伯爷书房暗匣。
关键对话: “周管事一个外院账房，为何敢动母亲旧账？” / “不是我放肆，是有人把沈氏留下的东西锁进了家主书房。”
情绪变化: 对峙 -> 抬层 -> 伯爷第一次真正慌
场景代价/变化: 谢云昭把“父亲”从模糊保护伞变成正面被问责对象
```

### 场景 2
```yaml
场景地点: 伯爷书房
出场角色: 伯爷、谢云昭、族叔、柳氏、周管事
场景目标: 把“书房暗匣里到底有没有东西”从猜测变成见光事实
场景阻力: 家主书房是家门核心禁地；一旦开匣，伯爷的脸和权威一起掉
情节概要: 在族叔压场下，伯爷不得不亲自去开暗匣。匣中并非完整账册，却压着足够坐实“顾家初聘银子曾被外院与书房一起挪用”的单纸和私印。到这里，柳氏再也不能单独背锅，伯爷线被正式写死。
关键对话: “父亲若真无辜，开匣让族叔看一眼就是。” / “这也叫暂存？”
情绪变化: 绷死 -> 匣开见真 -> 满厅失声
场景代价/变化: 书房这道门一开，伯府家门秩序被撕裂到不可回缝
```

### 场景 3
```yaml
场景地点: 花厅 -> 门外
出场角色: 谢云昭、周管事、伯爷、柳氏、管家
场景目标: 兑现阶段胜利，同时明确“账页不全，第三只手存在”
场景阻力: 周管事塌口后仍想把最关键的半页账留成死结；柳氏会拼命把这一切压回“周转误会”
情节概要: 周管事终于承认，顾家那笔不入礼单的银最初是为伯府外院周转，后面柳氏接手中馈后才知道这笔脏账。可真正那页完整实账已经不全，说明书房之外还有第三只手先一步摸过这条线。紧接着门外抓到翻窗婆子，证明今夜还有人要补最后一道手尾。
关键对话: “那笔银最初不是夫人的主意。” / “看来今夜，谁都别想睡了。”
情绪变化: 口供塌陷 -> 更深的缺口露出 -> 新危机压近
场景代价/变化: 她赢下了“伯爷不干净”这层，但真正最关键的完整账页仍不在手里
```

## 推进校验

- 本章关键收益不是自动到账，而是借族叔压场、回帖暗记和崔妈妈证词层层逼出来的。
- 本章真正阻力来自家主书房禁地、伯爷护脸、周管事拼命甩锅和“不孝”反扣。
- 主角的代价是：她为了开书房，彻底把父亲拖进局，往后再也没有“还给家门留脸”的退路。
- 本章不是全赢：只坐实了伯爷线和外院线，真正完整账页仍未到手。
- 章末必须让读者明确：现在危险的不是“有没有脏账”，而是“谁在今夜补最后一道漏”。

## 关键元素

### 伏笔处理
- 顾家旧回帖暗记：从账房线推进到书房暗匣线
- 伯爷书房暗匣：从怀疑升级成见真

### 新伏笔
- 第三只手：说明旧账线远不止柳氏、伯爷、周管事三人
- 翻窗婆子：直接挂到第007章夜抢/先夺暗匣线

### 设定展示
- 家主的权威不是无敌，只是在被上位规矩和族规一起压时才会裂
- 书房和外院不是两套干净线，而是同一笔脏账的前后台

### 角色状态变更
- 谢云昭：第一次把“父亲”也从伦理压制位拖成可被审视的局中人
- 伯爷：从模糊纵容者，升级为知情并受益的高风险对象
- 柳氏：从单线主压制者，退成家门旧账链的一环

## Continuity Notes

- 本章是“柳氏线咬成伯爷线”的关键过渡，不可写成简单打脸。
- 爽点来自层级抬升和家门秩序开裂，不来自口头赢一场。
- 章末必须给第007章留下直接行动目标：夜抢或先夺伯爷书房暗匣后的最后一层线头。
