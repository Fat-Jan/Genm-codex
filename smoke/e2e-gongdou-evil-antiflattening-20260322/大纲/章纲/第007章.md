# 第007章章纲

## 基础信息

```yaml
章节序号: 7
章节标题: 翻窗的人不认柳氏，认的是父亲身边那只最干净的手
字数规划: 3200-3800
本章目标: 顺着翻窗婆子这条线，把“第三只手”从模糊存在推进成伯爷身边的具体经手人，并逼出半页实账现在还没出府
预期爽点: 夜审翻窗婆子 / 伯爷身边最干净的人先露破绽 / 账页去向第一次缩到具体范围
本章阻力: 翻窗婆子怕死也怕儿子被扣，伯爷会用家门体面压夜审，族叔和二房老婶只想尽快止损，不会无条件站在谢云昭这边
本章代价: 谢云昭为了继续追账，必须让族内更多人知道伯府男人线也烂了，并且得暂时放走一个自己最想当场摁死的人
本章局部收益: 锁定第三只手是伯爷身边常管书信与钥匙的冯忠，确认半页实账仍在府内，而且最可能藏在正院佛堂和账匣来往线之间
本章残留风险: 冯忠已经察觉自己暴露，可能会连夜转移账页；族里也开始有人觉得谢云昭查得太深，会伤整个谢家脸面
主动方: 谢云昭
防守方: 伯爷 / 冯忠 / 柳氏
关键误判: 伯爷以为翻窗婆子只会咬柳氏；翻窗婆子以为只要不供冯忠，自己儿子就能活；谢云昭一开始也误以为“第三只手”仍是内宅女眷
独立议程: 族叔支持继续查，不是为谢云昭，而是为了抢在东宫发难前先把谢家族面保下来；二房老婶配合夜审，是为了切割二房和脏账，不是为了替谢云昭撑腰
关系残账: 谢云昭和族叔暂时站在同一边，但双方目标不同；她借二房老婶的切割心态拿到一时支持，后续也要付出让二房自保的代价
```

## 场景分解

### 场景 1
```yaml
场景地点: 正院偏厅
出场角色: 谢云昭、伯爷、柳氏、谢明珠、族叔、二房老婶、翻窗婆子、青枝
场景目标: 先把翻窗婆子从“抓到的小偷”钉成“知道账页去向的人”
场景阻力: 伯爷想把人拖去家法处置，柳氏想把口供重新摁回“内宅奴婢乱窜”，翻窗婆子死咬自己只是偷银
情节概要: 谢云昭不先问账页，而是先问谁能调正院后窗钥匙、谁知道书房翻窗后该走哪条暗廊，逼翻窗婆子露出她不可能只是普通偷银贼。族叔和二房老婶一个为族面、一个为切割，都被迫点头让夜审继续。
关键对话: “偷银的人不知道后窗第二道锁怎么开。” / “今夜若只当她是贼，明日东宫问的就是谢家为何替贼遮账。”
情绪变化: 乱 -> 被压住 -> 口供被重新定性
场景代价/变化: 谢云昭把夜审扩大到族中旁支都看得见，等于主动扩大父亲丢脸的范围
```

### 场景 2
```yaml
场景地点: 正院偏厅内侧 / 临时刑凳前
出场角色: 谢云昭、翻窗婆子、青枝、崔妈妈、族叔
场景目标: 从翻窗婆子嘴里撬出“第三只手”的名字和账页仍在府中的证据
场景阻力: 翻窗婆子真正怕的是儿子契身在外院小厮房，一旦供出冯忠，儿子先死；崔妈妈也不敢把自己知道的“佛堂经匣”线全说满
情节概要: 谢云昭不用重刑，而是先当众点破翻窗婆子儿子的契书如今不在柳氏手里，而在外院钥匙房的旧簿里，说明她真正听的是冯忠。崔妈妈补出当年沈氏怕账页被人翻走，曾把半页实账在佛堂经匣和账匣之间来回换藏的习惯，翻窗婆子这才失口认出冯忠让她去“摸匣角，不摸经卷”。
关键对话: “你怕的不是夫人，是你儿子的契书。” / “他说只摸匣角，不准翻经卷。”
情绪变化: 绷住 -> 被拆软肋 -> 失口
场景代价/变化: 谢云昭拿到名字，却也暴露自己知道契书和钥匙房这层老规矩，以后很难再隐藏她对府内旧秩序的掌握程度
```

### 场景 3
```yaml
场景地点: 正院回廊 -> 佛堂外
出场角色: 谢云昭、冯忠、伯爷、二房老婶、族叔、柳氏
场景目标: 兑现阶段胜利，但不在本章直接拿到账页
场景阻力: 冯忠警觉极快，一察觉翻窗婆子失口就准备转移；伯爷会死保冯忠这只“最干净的手”；二房老婶只愿帮忙封门，不愿替她彻查佛堂
情节概要: 谢云昭借翻窗婆子那句“摸匣角，不摸经卷”顺势压向佛堂，正撞见冯忠试图先一步换匣。她没当场搜出半页实账，只在匣底找到账灰和被折过又抽走的空缝，确认账页仍在府内且与佛堂来往线有关。族叔下令先封佛堂和钥匙房，二房老婶则当众要求二房的人全部退开切割风险。冯忠暂时被扣下，但真正的半页账仍没落手。
关键对话: “你看着最干净，所以才轮得到你摸钥匙。” / “今日先封门，不代表谢家明日就能把这张脸捡回来。”
情绪变化: 抢先一步 -> 短赢 -> 更大的撕裂被亮出来
场景代价/变化: 谢云昭赢到“冯忠暴露 + 佛堂线锁死”，却也让族里更多人意识到她会继续往家门最深处掘，这会反过来制造新的阻力
```

## 节奏安排

| 段落 | 内容 | 字数 | 情绪 |
|------|------|------|------|
| 开头 | 抓到翻窗婆子后，先争“她到底是贼还是知情人” | 400-550 | 紧绷 |
| 中段1 | 借钥匙、暗廊、契书把婆子定性并拆掉伯爷的家法收口 | 850-1050 | 压场 |
| 中段2 | 顺着儿子契书和崔妈妈旧记，逼出冯忠和佛堂匣角 | 900-1100 | 逼供硬爽 |
| 高潮 | 冲到佛堂外截冯忠，抢到空缝和账灰，确认账页还在府里 | 800-950 | 硬赢但不全赢 |
| 结尾 | 族内支持开始分裂，下一战转成“天亮前谁先拿到账页” | 250-350 | 更险 |

## 推进校验

- 本章关键收益不是自动到账，而是靠夜审、拆软肋和抢路线硬逼出来的。
- 本章阻力不是单一反派阻拦，而是伯爷护脸、冯忠保命、族叔止损、二房切割各有各的算计。
- 本章代价是谢云昭主动把族内更多人拖进局，未来会遇到“你不能再查了”的集体反压。
- 本章只赢到“第三只手有名有姓，账页仍在府内”，没有直接拿到账页。
- 章末必须把下一战推成“天亮前先抢账页”，而不是继续停在夜审余波。

## 关键元素

### 伏笔处理
- 第三只手：从模糊存在推进到具体经手人冯忠
- 半页实账：从“可能丢了”推进到“仍在府内、仍可抢”

### 新伏笔
- 佛堂经匣与账匣来往线 -> 第008章主战场
- 二房和族叔的支持开始分裂 -> 后续家族内部阻力升级

### 设定展示
- 伯府真正的“干净手”是掌钥匙、管书信、能进佛堂和书房的人
- 族内长辈支持继续查，不等于支持谢云昭翻到底

### 角色状态变更
- 谢云昭：从“把父亲拖下水”进入“开始调动族内不同自保力量”
- 族叔：从压场人变成有限盟友，他更关心族面不是谢云昭
- 二房老婶：第一次明确站出来切割风险，她不是盟友，只是暂时不想陪柳氏一起死
- 冯忠：从无名背景人升级为真正会咬出下一层的大节点

## Continuity Notes

- 本章必须让读者看见：支持谢云昭的人不是天然站她，而是各有算盘。
- 不允许把翻窗婆子写成一撬就全招的纸人，她必须有“儿子契书”这层真实软肋。
- 冯忠不能当场全盘崩掉，否则第008章没法继续抢账页。
- 章末要把危机压到“天亮前”，让下一章天然带倒计时。
