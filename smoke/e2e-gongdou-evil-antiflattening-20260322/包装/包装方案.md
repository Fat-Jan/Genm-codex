# 《庶妹换我婚书那夜，太子先开了口》包装方案

## 项目定位

- 平台：番茄
- 题材：宫斗宅斗
- 内容桶：宫斗宅斗
- 核心标签：`家门压制` `换婚书` `继母陷害` `后宅翻盘`
- 核心情绪链路：被压制 -> 不认命 -> 当众反咬 -> 更大局入场

## 推荐主方案

### 主方向

婚书被换 + 继母压制 + 伯府二姑娘反咬翻盘

### 推荐书名

《婚书被换那夜，太子先开了口》

### 运行中书名

《庶妹换我婚书那夜，太子先开了口》

### 一句话卖点

人人都以为安平伯府的二姑娘谢云昭该认命，可婚书被换那夜，她反手掀出账本旧线，把继母和伯府旧账一起逼上了台面。

### 推荐简介

继母要她认命，继母那一线的女儿要踩着她的婚书嫁进高门，满府上下都等着看谢云昭低头。  
可她偏偏不认。  
婚书被换那一夜，她当众掀出一笔没人敢细查的旧账，从被压着喘不过气的伯府二姑娘，一步步把自己推到了谁也不敢再随便处置的位置。  
只是她咬开的，不止是一桩婚事，还有伯府上一代没埋干净的血账。

## 包装约束

- 第一屏必须见压制关系
- 标题要有人物关系和高反转
- 简介先抛压制，再抛反咬，再挂更大局
- 不写成纯情爱线，核心是府门权力和翻盘
- 当前 `market_adjustments` 已新增：
  - `scan-surface-hook`
  - `scan-frontload-conflict`
  - `scan-kinship-truth-check`
- 因为家族真值目前仍是 `draft`：
  - “庶妹”暂不算已锁定关系词
  - 当前主书名可继续作为运行方案，但不建议直接视为最终冻结主方案
  - 当前更安全的冻结主方案是《婚书被换那夜，太子先开了口》
  - 若后续真值确认谢明珠并非稳定“庶妹”口径，优先改回不带风险关系词的标题方向
