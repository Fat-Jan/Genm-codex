# 《闪婚那天，我把京圈太子爷认成司机》包装方案

## 项目定位

- 平台：番茄
- 题材：言情
- 内容桶：现言甜宠
- 核心标签：`闪婚` `误认身份` `豪门反差` `先婚后爱`
- 核心情绪链路：被逼婚 -> 误认 -> 关系绑死 -> 高糖反差升级

## 推荐主方案

### 主方向

闪婚误认 + 豪门反差 + 先婚后爱

### 推荐书名

《闪婚那天，我把京圈太子爷认成司机》

### 一句话卖点

被前任和家里同时逼到墙角的林栀，在民政局门口随手拉错了一个“司机”闪婚，结果对方转头就把她带进了京圈最不能惹的名字里。

### 推荐简介

前任回头、家里逼婚、住处也快保不住的那天，林栀在民政局门口抓错了人。  
她原本只想找个陌生人陪自己演一场戏，结果这一演，直接把自己演进了结婚证里。  
更离谱的是，她以为自己闪婚的是个司机，对方却偏偏是整个海城最惹不起的那位。  
从那天起，她的人生不再只剩逃跑，还多了一个总在关键时刻替她撑腰的人。
