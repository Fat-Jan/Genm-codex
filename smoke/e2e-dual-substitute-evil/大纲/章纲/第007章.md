# 第007章章纲

## 基础信息

```yaml
章节序号: 第007章
章节标题: 她们刚堵到柏宁，酒店先把她当成来闹事的
字数规划: 3400
本章目标: 让柏宁线从“接口人名字”推进到“归位计划3.2 会签页”，并把酒店外部阻断真正压到动作层。
预期爽点: 姜栀顶着第二轮热搜去堵柏宁、许知微公开认领她、柏宁失口承认“双总办会签”、会签页被抢到半张。
本章阻力: 西岸酒店已经被人提前布了保安、法务和镜头，姜栀一出现就会被按成闹事者；柏宁知道自己只是接口层，一被堵就会先删记录、先喊保安；林特助的内部门卡也开始失效。
本章代价: 姜栀要在公开场域承受“酒店堵人”的第二轮舆论；许知微必须第一次在外人面前明确站到姜栀这边，等于主动烧掉家里给她留的缓冲；林特助会因为这次带路被当场撤掉权限。
本章局部收益: 抢到一页“归位计划3.2”会签页的半张照片，确认项目走的是“双总办会签”，且批注里仍写着“姜线保留女配出口”。
本章残留风险: 会签页只有部门和批注，没有最终拍板人名字；柏宁当场删机并报警式求保，酒店偷拍视频会把姜栀继续往“失控追人”上推；今夜原定的终稿归档会被提前转移。
```

## 场景分解

### 场景 1
```yaml
场景地点: 西岸酒店外与侧门入口
出场角色: 姜栀、许知微、林特助、媒体、酒店安保
场景目标: 把外部阻断落到动作层，而不是只做背景压迫。
场景阻力: 酒店门口已经有镜头和法务跟口，姜栀一下车就会被认出来；林特助原本准备的员工门卡被系统临时冻结。
情节概要: 三人刚到酒店，外面的偷拍视频词条已经升级成“姜栀酒店堵人”。林特助试图带她们走侧门，却发现门卡被撤；安保开始询问身份，镜头往这边扑。许知微只能当场说姜栀是她带来的，先把人从侧门接进去。
关键对话: “你现在再被拍到一次，他们会把‘偷档’改写成‘四处闹场’。” / “她跟我一起的，出事我认。” / “林特助，你的权限刚刚被停了。”
情绪变化: 高压 → 失算 → 被迫明站边。
场景代价/变化: 许知微第一次公开把姜栀认进自己的线，林特助失去内部通道能力。
```

### 场景 2
```yaml
场景地点: 酒店会务层外廊
出场角色: 姜栀、许知微、柏宁、酒店秘书、安保
场景目标: 正面堵住柏宁，并让他在高压下露出“双总办会签”这一层。
场景阻力: 柏宁比应岚更滑，他第一反应不是说话，而是删机、叫保安、把她们定义成骚扰；许知微和姜栀还在录音问题上带裂口。
情节概要: 柏宁刚从小会议室出来，就被姜栀和许知微堵住。姜栀用应岚录音里的“柏宁”开门，许知微再用“归位计划3.2”压他。柏宁先否认，随后在删手机和催保安的动作里失口说出“我只是替双总办收版本”。姜栀追问“双总办”时，柏宁更慌，转头就喊安保。
关键对话: “你不是甲方，你是替甲方接脏稿的人。” / “我只收版本，不做判断。” / “双总办会签的东西，为什么要我背？”
情绪变化: 逼问 → 慌乱 → 失口。
场景代价/变化: 她们确认项目并非单线私人恩怨，而是走部门会签；但柏宁迅速启动安保反制。
```

### 场景 3
```yaml
场景地点: 小会议室门口与文件推车旁
出场角色: 姜栀、许知微、柏宁、会务秘书、安保
场景目标: 抢到会签页半张，完成本章局部收益。
场景阻力: 安保已经逼近，柏宁在删手机，会务秘书想先把文件车推走；时间只够看一眼。
情节概要: 姜栀趁混乱掀开文件推车上的会签夹，快速拍下半张页眉：`归位计划3.2`、`双总办会签`、批注 `姜线保留女配出口`。柏宁扑上来抢文件，会务秘书把夹子抽走，只留半张图。许知微为了护姜栀，正面拦下安保和秘书，公开说“这件事我会亲自对接”，把自己彻底卷进来。
关键对话: “别动那份。” / “拍到了就够了。” / “她今天不是无关人员，她跟我走同一条线。”
情绪变化: 混乱 → 抢到半张图的短促爽点 → 代价砸实。
场景代价/变化: 许知微公开护人，代价真正落地；会签页只拍到半张，没有人名。
```

### 场景 4
```yaml
场景地点: 酒店后梯与回程车内
出场角色: 姜栀、许知微、林特助
场景目标: 把本章的残留风险收紧，并挂第008章钩子。
场景阻力: 外面第二轮偷拍视频和酒店报警式投诉正在发酵；林特助接到撤岗通知，不能再继续做稳定通道。
情节概要: 三人从后梯撤走，林特助收到被撤出许家工作群和门禁系统的通知。姜栀翻看半张会签页，发现今夜八点有“终稿归档”字样，却没有地点。许知微看着那页纸，第一次承认：真正握“归位计划”终稿的人，已经不是她能隔空拖住的层级。
关键对话: “我这次是真的没门了。” / “半张也够，至少知道他们不是临时起意。” / “今晚八点之前，我们得先知道终稿被送去哪。” 
情绪变化: 得手后的冷静 → 代价落地 → 更大门槛。
```

## 推进校验

- 柏宁不能一堵就全说，只能失口露出“双总办会签”和接口层焦虑。
- 外部阻断必须产生真实动作后果：安保、镜头、门卡冻结、撤岗，不只是氛围。
- 本章的关键收益只能是“半张会签页”，不能拿到完整终稿。
- 许知微的代价要真正落地到公开护人，而不是只在私下嘴硬。
- 章末钩子要从“抓谁”升级到“今夜终稿归档去哪”。

## 关键元素

### 伏笔处理
- `柏宁接口线`: 从名字推进到实际出场和“会签层”口风。
- `偷拍视频热搜`: 本章继续升级成“酒店堵人”，保证反咬没有消失。
- `林特助功能桥问题`: 本章通过撤岗和门卡失效，让他付出真实成本，不再只是平滑报点。

### 新伏笔
- `双总办会签`: 指向两家共同利益层，而不是单边私心。
- `终稿归档`: 为第008章进入真正设局层预留更高门槛。

### 设定展示
- `西岸酒店会务层`: 展示项目、会签、会务秘书、文件车这类现实执行链条。
- `半张会签页`: 把旧包装进一步压成可追的文书链，而不是纯口供。

### 角色状态变更
- `姜栀`: 从手握录音的人升级为手握半张会签页的人，但代价是酒店“闹事”标签继续加码。
- `许知微`: 第一次在公开场域明确站到姜栀这一边，她和家里那条缓冲线正式烧起来。
- `林特助`: 终于不再只是功能桥，他这次帮完就失去权限和岗位安全。
