# 第008章章纲

## 基础信息

```yaml
章节序号: 第008章
章节标题: 终稿还没进柜，她先在流转单上看见了承川的体面
字数规划: 3500
本章目标: 通过更高门槛的流转日志与交接箱信息，确认“归位计划3.2”今晚要去的真实场域，并首次露出更上层的会签名字。
预期爽点: 姜栀不用对手失口也能顺着格式和流转码追到终稿路径、许知微被迫签联合责任书、交接箱标签露出“承川线体面保留”、更上层名字第一次浮面。
本章阻力: 第007章后对方已经知道她们会追终稿，文书寄存中心和外部档服会先锁权限、先清日志；姜栀还挂在酒店堵人热搜上，一露面就会被当高风险访客；许知微公开站边后，家里那侧开始直接切她权限。
本章代价: 许知微必须动用最后一条基金外包链，为姜栀签联合责任书；林特助的个人设备会被远程抹除，连最后那点格式记忆都可能丢；姜栀需要放弃“马上公开会签页”的短平快反打，先赌一把今晚真正的交接路线。
本章局部收益: 通过流转日志与交接箱标签，确认终稿会在今晚七点四十送往“潮汐会馆闭门会”，并第一次看到更上层会签名：贺氏董事办主任沈蔚、许太太办公室。
本章残留风险: 她们拿到的是交接与会签路径，不是终稿正文；对方已经开始远程清日志、抹设备，说明今晚闭门会会提前加固；许知微与家里的对立正式公开化，后续不再有缓冲层。
```

## 场景分解

### 场景 1
```yaml
场景地点: 回程车内与临时落脚咖啡馆
出场角色: 姜栀、许知微、林特助
场景目标: 用现有半张会签页倒推“终稿归档点”的格式规律，避免再走同场掉纸的老路。
场景阻力: 会签页只有半张、格式不完整；林特助设备正在被远程停用，能提供的内部记忆越来越少；双女主还在为录音与公开站边僵着。
情节概要: 姜栀把会签页放大分析，认出页脚归档码不是酒店会务格式，而是外部文书寄存商的交接码。许知微用自己曾经批过的基金外包单比对，确认这套码属于“岚序档服”。林特助刚想补一条路径，平板就被远程抹除，只来得及说出“北港文书中心”几个字。
关键对话: “这不是会务层的页脚，是外包档服的交接码。” / “你不是只会掀桌，你连别人怎么装柜都记。” / “快记，抹机前我只认得出这半段。”
情绪变化: 紧绷 → 推理式推进 → 被时间追着跑。
场景代价/变化: 线索不再来自对手失口，而来自格式识别与残余专业经验；林特助的内部能力继续被抽干。
```

### 场景 2
```yaml
场景地点: 北港文书中心大厅
出场角色: 姜栀、许知微、林特助、前台、安保、法务联络人
场景目标: 把“想看终稿路径”变成一场需要正式签责的高门槛动作。
场景阻力: 北港文书中心已收到风险访客提醒，姜栀名字一录入就跳红；许知微的许家权限被切，不能再直接调外包系统；安保和法务要求必须有联合责任签字才能查交接流转。
情节概要: 三人到大厅即被识别为高风险访客。前台拒绝放行，法务视频连线要求“基金侧负责人 + 文件相关当事人”共同签责任书才能调当天交接流转。许知微在大厅当着所有人把字签了，等于公开承认自己与姜栀站到同一调查线上。
关键对话: “现在不是她一个人的风险，是我带来的。” / “你签下去，就等于承认你和这份文件存在利害关系。” / “我本来就在里面。”
情绪变化: 压迫 → 被逼签责 → 代价落地。
场景代价/变化: 许知微失去最后一层家里留给她的模糊地带；外部阻断继续存在但被制度性签责打穿。
```

### 场景 3
```yaml
场景地点: 文书中心流转室与监控盲区边缘
出场角色: 姜栀、许知微、值班管理员、安保
场景目标: 在有限查看权限内抢到“交接箱标签 + 流转日志”这份非正文证据。
场景阻力: 系统只开放五分钟、且只能看当日交接箱编号；管理员全程盯着，不能拍完整屏；对方已经远程撤回一部分日志。
情节概要: 姜栀和许知微在五分钟内只能翻物流流转，不给看终稿正文。姜栀借品牌供应链经验，先找“会签项目优先移交”的标签，再在交接箱标签上看到“归位计划3.2-F”“潮汐会馆闭门会”“承川线体面保留”。流转室屏幕上的发件责任栏一闪而过，姜栀只来得及拍到两行：`贺氏董事办主任 沈蔚`、`许太太办公室`。
关键对话: “不给正文，就看它怎么走。” / “终稿不能看，箱签可以看。” / “承川线体面保留，这不是爱情，是包装保护。”
情绪变化: 冷静抢时间 → 短促爽点 → 更大真相浮头。
场景代价/变化: 她们第一次看到更上层名字，但证据仍停留在流转和箱签，不足以直接一锤定音。
```

### 场景 4
```yaml
场景地点: 文书中心外侧通道与夜色下的回程车
出场角色: 姜栀、许知微、林特助
场景目标: 收紧本章残留风险，并把第009章目标抛成“潮汐会馆闭门会”。
场景阻力: 对方发现她们查过流转后，北港文书中心开始回收查看权限；林特助设备被远程清空，无法继续做格式比对；外面酒店堵人热视频与文书中心签责记录开始形成叠加风险。
情节概要: 三人刚离开流转室，系统查看权限就被回收。林特助的设备被正式抹空，他只能确认“潮汐会馆”一直是两家处理体面事务的旧场。姜栀看着那两行会签名，第一次明确意识到贺承川更像被保下来的项目成分，而不是唯一写剧本的人。她决定第009章直接去潮汐会馆堵闭门会。
关键对话: “他们保的不是承川的人，是承川这条线的体面。” / “今晚七点四十之前，潮汐会馆。” / “这次我们去的不是接口层，是写结局的人坐的那张桌子。”
情绪变化: 得手后的冷静 → 真相转向 → 临战。
场景代价/变化: 真正设局层第一次从概念变成具体会场；第009章转入闭门会硬碰。
```

## 推进校验

- 本章禁止再用“对手失口 + 现场掉纸”同场推进，改用格式识别、责任签字和流转日志。
- 关键收益只能是“交接箱标签 + 流转日志 + 会签名”，不是终稿正文。
- 许知微的代价要继续真实落地到公开签责，而不是只靠态度说站边。
- 林特助这章继续承担成本，但不再负责顺滑送答案，只给半段格式记忆。
- 章末必须把第009章目标明确锁到“潮汐会馆闭门会”。

## 关键元素

### 伏笔处理
- `终稿归档点`: 本章推进成具体会场 `潮汐会馆闭门会`。
- `双总办会签`: 本章第一次带出更上层实际名字与签发链。
- `林特助功能桥问题`: 通过远程抹机与权限清零，压住他继续无成本送线索的风险。

### 新伏笔
- `潮汐会馆闭门会`: 第009章硬碰的真正场域。
- `承川线体面保留`: 说明男方本身也被包装进项目，不只是单纯的操控者。

### 设定展示
- `北港文书中心`: 让证据链从人和文件，升级到正式的流转、签责和寄存体系。
- `交接箱标签 / 流转日志`: 用更硬的执行链证明旧包装项目真实存在。

### 角色状态变更
- `姜栀`: 从手握半张会签页的人，升级为能从文书流转里倒推出真正会场的人。
- `许知微`: 公开站边后再签联合责任书，已经没有家里缓冲层可退。
- `林特助`: 设备被彻底抹空，他从半吊子桥梁退成“只剩记忆碎片”的高成本证人。
