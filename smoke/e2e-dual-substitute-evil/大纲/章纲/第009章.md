# 第009章章纲

## 基础信息

```yaml
章节序号: 第009章
章节标题: 她没进闭门会，先在门外听见了谁该去演女配
字数规划: 3600
本章目标: 在潮汐会馆闭门会外完成“真正设局层浮面”，让许太太与沈蔚从责任栏上的名字变成当场可确认的人。
预期爽点: 姜栀不靠口供而靠追交接箱逼近闭门会、许知微正面拖住母亲线、终稿封页与现场对话同时把“女配出口”坐实、真正设局层第一次浮面。
本章阻力: 潮汐会馆今晚已经提前清场，前门宾客验证、后门交接路线、会馆安保都在防外人；姜栀挂着酒店堵人热搜，正门一露面就会被直接拦下；林特助权限已空，只能给零碎旧记忆；许知微若公开顶撞母亲，就等于彻底撕破家里最后一层遮羞布。
本章代价: 姜栀要放弃“直接闯进去”的冲动，改走后场追箱；许知微必须前门亮相，把母亲与沈蔚从桌上拖出来；林特助要冒着被认出和被追责的风险，用旧记忆带她们找到会馆服务梯。
本章局部收益: 抢到终稿封页照片，确认版本标题里直接写着“知微归位 / 姜线退场”；并通过门外录音与对话，确认许太太与沈蔚就是共同设局层。
本章残留风险: 终稿正文仍未完整拿到；潮汐会馆内部已经启动版本替换和灭痕；姜栀与许知微虽然站到了同一边，但接下来一旦公开反打，双方都要正面承受各自家族与舆论的反噬。
```

## 场景分解

### 场景 1
```yaml
场景地点: 去潮汐会馆的路上
出场角色: 姜栀、许知微、林特助
场景目标: 把第008章查到的流转路径压成具体行动方案。
场景阻力: 潮汐会馆前门一定被盯死，后门交接时间又很短；三人现在没有任何正规权限。
情节概要: 姜栀根据交接箱编号和会馆常用路线，判断终稿不会从大堂走，而会走服务梯进二层小宴厅。许知微提出由她从前门进场，主动把母亲线拖出来；姜栀走后场追箱。林特助只提供一条旧记忆：服务梯会先停酒水层再上二层。
关键对话: “正门是脸，后门才是文件。” / “你去追箱，我去拖桌上的人。” / “我现在只剩这点旧路能认，错了就真进不去了。”
情绪变化: 冷静排路 → 分工 → 临战。
场景代价/变化: 双女主首次在公开与暗线两端同步配合，但风险也被同时拉满。
```

### 场景 2
```yaml
场景地点: 潮汐会馆正门与前厅
出场角色: 许知微、许太太、沈蔚、贺承川、宾客/迎宾
场景目标: 让许知微把真正设局层从闭门桌上逼到门边。
场景阻力: 许太太与沈蔚都想维持体面，不会在前厅直接承认任何东西；贺承川此刻更像被保护起来的门面。
情节概要: 许知微前门亮相，直接要求见母亲。许太太和沈蔚先后现身，试图把她带回里厅“私下谈”。贺承川也在场，但全程被压着做体面接口。许知微一句句把“归位计划”“姜线出口”往台面上拽，逼得两人情绪失衡。
关键对话: “今天你们是叫我来归位，还是叫她退场？” / “知微，别在外面说这些。” / “承川线要保，体面就不能先碎。”
情绪变化: 克制 → 针锋相对 → 体面开始裂。
场景代价/变化: 许知微与家里彻底撕破；沈蔚和许太太第一次同时出现在同一个场域里。
```

### 场景 3
```yaml
场景地点: 潮汐会馆后场服务梯与二层宴厅门外
出场角色: 姜栀、林特助、交接员、安保
场景目标: 不靠对手失口，而靠追交接箱抢到终稿封页与门外录音。
场景阻力: 交接箱只会在服务梯停留几十秒；安保巡查更紧，林特助一旦被认出就会被立刻带走。
情节概要: 姜栀和林特助在后场等到A-17交接箱，姜栀借酒水推车遮挡追到服务梯口，只在箱子换签时抢到一张终稿封页照：标题写着“知微归位 / 姜线退场”。与此同时，门里传出许太太和沈蔚被许知微逼出来的争执声，姜栀顺手录下“女配出口必须留”“承川线先保体面”等关键句。
关键对话: “别碰那箱。” / “我只要第一页。” / “姜线必须有出口，不然今晚这桌怎么收。” / “先保承川线的体面，再谈知微怎么回位。”
情绪变化: 紧绷潜入 → 抢到封页的短促爽点 → 真相钉死。
场景代价/变化: 姜栀确认真正剧本写法，但林特助因护她被后场安保认出。
```

### 场景 4
```yaml
场景地点: 会馆后廊与撤离车内
出场角色: 姜栀、许知微、林特助
```

## 推进校验

- 本章允许“真正设局层浮面”，但不允许直接全结清。
- 关键收益必须分成两半：
  - 封页证明剧本写法
  - 门外录音证明谁在拍板
- 不让任何一个角色直接口头全招；真相由封页、对话、场域共同坐实。
- 贤于上层的阻力必须真实：清场、巡查、后场交接、版本替换、安保认人。
- 章末必须把第010章锁成“公开反打”，而不是继续偷偷追文书。

## 关键元素

### 伏笔处理
- `终稿归档点`: 本章兑现为潮汐会馆闭门会。
- `沈蔚会签线`: 本章从责任栏名字升级成真人现身。
- `承川线体面保留`: 本章从箱签字样变成桌上原则。

### 新伏笔
- `版本替换`: 说明原件会被快速回收，下一章不能再迷信“追原件”。
- `公开反打`: 第010章的主目标，不再是潜入式取证。

### 设定展示
- `潮汐会馆闭门会`: 两家处理“体面事务”的真正场。
- `服务梯交接箱`: 用会馆后场执行链把终稿送达桌面。

### 角色状态变更
- `姜栀`: 从追证据的人变成手握封页与录音、准备把标签还回去的人。
- `许知微`: 从公开站边升级到正面顶撞母亲线，她的退路几乎被烧光。
- `林特助`: 从高成本证人进一步滑向“会被彻底弃掉的人”，他能给的信息到此基本见底。
