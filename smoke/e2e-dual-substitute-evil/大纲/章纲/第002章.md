# 第002章章纲

## 基础信息

```yaml
章节序号: 第002章
章节标题: 白月光没踩她，反而递来了一张替身简报
字数规划: 3200
本章目标: 把白月光与恶毒女配的旧对立翻面，在试探、卡牌展示与条件交换里搭建起暂时同盟。
预期爽点: 证据被摊在桌面、替身剧本第一次露出真实面目、两个原本对立的女人暂时合流改写规则。
本章阻力: 姜栀习惯把被安排的身体状态当作成本，贺承川与两家还在控制话语权，许知微必须保留自己的真正筹码。
本章代价: 姜栀要承认自己是被挑选的替身、把不甘心的源头暴露出来，许知微需要把“替身简报”“流程备注”等底牌当作试探。
本章局部收益: 揭开替身筛选简报、确认许知微不是来“收场”、达成条件明确的临时合作。
本章残留风险: 证据亮出后，幕后来操盘的力量会把两人当成更危险的对手；第二页简历仍然被隐瞒，释放会场只掀开一层；两人仍需维护各自立场，信任随时可能被旧叙事拉扯回去。
```

## 场景分解

### 场景 1
```yaml
场景地点: 发布会后台休息室的化妆台区域
出场角色: 姜栀、许知微
场景目标: 让姜栀确认自己被筛选替身/恶毒女配的事实，同时试探许知微到底要怎么用这份资料。
场景阻力: 姜栀把“让位”“闭嘴”当作工作常态，信号太弱看不出这是设计；许知微刚回国需要保持姿态，不能露出全部意图。
情节概要: 许知微把三年前的旧邮件和简历摆出来，姜栀从“只是长相像你”到“我被选中来了”再到“你到底要干什么”逐步质问，两人在空气中反复试探、逼出互相信任的冰点。
关键对话: “你不是意外站到那个位置上的。” / “这份简报才是这个剧本的底牌。” / “你比我想象的还难合作。”
情绪变化: 冷静试探 → 震惊与警惕 → 暗含讽刺的缓和。
场景代价/变化: 姜栀不得不承认自己不是“自己选的位置”，许知微把“替身简报”转为彼此之间的卡牌，关系从对打进入微妙的谈判状态。
```

### 场景 2
```yaml
场景地点: 休息室的桌边，处在发布会档案和流程表之间
出场角色: 姜栀、许知微
场景目标: 明确当晚发布会的分工，把“你要我退位”“我只配做背景”的旧逻辑换成彼此条件。
场景阻力: 姜栀不想当垫脚石，担心再次被拿来证明“恶毒女配会发疯”；许知微要把“栖序发布会”资源当作筹码又不想暴露计划。
情节概要: 许知微展示饭局流程备注，要求姜栀“别退”，姜栀先列出三条条件：一起改台词、不能被用来翻白月光牌、不要再接手不完整旧物。两人把各自文档当成卡牌交换，初步达成“暂时合作但各自保留退路”。
关键对话: “你站在台上，把订婚戒退干净，也别替我说话。” / “等这件事掀开后，你手里还有什么旧东西，不能再只让我接半张。” / “明天谁都别替谁让位。”
情绪变化: 怀疑 → 互相试探 → 微妙的默契与局部爽点。
场景代价/变化: 条件交换把两人都拉进了更公开的战场，姜栀把脆弱的身份交出来换取主动，许知微也把自己回国的筹码当作赌注。
```

### 场景 3
```yaml
场景地点: 休息室门口/走廊，发布会即将开场前的短暂独处
出场角色: 姜栀
场景目标: 对接匿名警示，把新的风险线索带入下一章。
场景阻力: 短暂的静默提供不了反应时间，只能靠直觉判断是否继续信任许知微。
情节概要: 许知微走后，姜栀接到陌生短信：三年前替她改简历的人也会在发布会现场，暗示还有更高层在看着她的每一步。
关键对话: （短信）“三年前替你改简历的人，明天也会在发布会现场。”
情绪变化: 微笑转为凝神警戒。
场景代价/变化: 本章的合作立刻带上更高的代价——另一个操盘者还没出现，明天不仅要改剧本，还得先把人抓出来。
```

## 推进校验

- 本章关键收益不是自动到账，而是通过证据展示与条件交换，把“替身剧本”变成可谈判的筹码。
- 本章真正的阻力来自贺承川与两家的旧叙事、大众期待“白月光回归”、以及姜栀自身的防备心。
- 主角为了这次收益付出了暴露替身身份、分享此刻的脆弱以及以旧文件作为信任牌的代价。
- 这次赢完以后，更危险的在于暴露出来的证据把两个女人推向更高阶的对抗目标，幕后操盘者会增加审视和反击。
- 如果这是第2章的位置，卡住的就是“我到底要相信谁？”——两人完成暂时同盟前必须先互相试探、交换条件，不能一拍即合。

## 关键元素

### 伏笔处理
- `替身筛选简报`: 由许知微公开，姜栀看到特征描述后不再把自己当偶然，后续可放在发布会现场被再度翻出。
- `发布会流程备注`: 作为条件交换的“卡牌”之一，说明原本有人要统一叙事，以后某一方偷换流程会留下痕迹。

### 新伏笔
- `匿名短信“替你改简历的人也会在现场”`: 明示还有更上层的设局者，发布会主线已经激怒了更大的手。

### 设定展示
- `栖序珠宝春季发布会`: 通过流程表、档案记录，明确这是贺氏与许家一起构建的新场域，也是两人走到台前的战场。
- `替身文学的运作逻辑`: 电子简报与流程备注展示贺氏如何把“白月光”“女配”写进剧本，为后续“谁写剧本”线做铺垫。

### 角色状态变更
- `姜栀`: 从拒绝相信升级为主动设条件的合作者，身份从“被动承受者”变成把证据当武器的参与者。
- `许知微`: 从“白月光”标签的被动载体转向愿意换名字、换标题的行动者，但仍保留“我还在试探你是不是一样危险”的警惕性。
