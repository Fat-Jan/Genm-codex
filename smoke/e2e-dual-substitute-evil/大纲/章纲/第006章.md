# 第006章章纲

## 基础信息

```yaml
章节序号: 第006章
章节标题: 她还挂在偷档热搜上，先去机场堵写台词的人
字数规划: 3400
本章目标: 在偷拍视频和律师函继续反咬的前提下，拦住应岚，抢到一段只够推进下一层的口供录音。
预期爽点: 姜栀顶着偷档热搜直接追人、应岚被当场堵住、归国包装第一次被应岚亲口承认为“项目”、柏宁这个接口人被抛出来。
本章阻力: 机场是公开场域，偷拍视频热搜还在发酵，姜栀一露面就会被继续拍；许知微不能再让许家旧馆被拖下水，因此策略更保守；应岚是职业执行人，知道怎么只说半句。
本章代价: 姜栀必须放弃立刻做舆论自救，顶着“偷档报复”标签去追人；许知微要动用自己并不想再碰的家里通道，把两人送进机场贵宾通道；两人会因为是否公开录音再起一次冲突。
本章局部收益: 抢到应岚亲口承认“B-17只是Q版样张”“归位计划3.2”真实存在，并拿到接口人名字“柏宁”。
本章残留风险: 录音不够直接指向最终设局人，且机场偷拍视频会把姜栀往“追着人炒作”方向再推一轮；许知微会对姜栀私开录音这件事不满，同盟裂口继续扩大。
```

## 场景分解

### 场景 1
```yaml
场景地点: 去机场的车上
出场角色: 姜栀、许知微、林特助
场景目标: 把第005章的锅和时间压力同时压实，明确这次追人本身就是一笔代价。
场景阻力: 热搜和律师函已经挂出去，姜栀一旦露面就会被继续钉在“偷档报复”标签上；许知微坚持这次只能低调接触，不能再在公开场域硬掀。
情节概要: 车上三人对策略起冲突。许知微想先见应岚、姜栀留在车里，姜栀拒绝，直说自己已经被挂成贼，再藏只会坐实。林特助补充：应岚已经改走贵宾通道，能接触的时间只剩十分钟。
关键对话: “我现在缩回去，不叫止损，叫认罪。” / “你要的是答案，我要的是人别在机场炸开。” / “你们再吵五分钟，人就上安检了。”
情绪变化: 高压 → 对刺 → 被时间逼着统一动作。
场景代价/变化: 姜栀决定顶着热搜去堵人，许知微则被迫动用家里通道资源。
```

### 场景 2
```yaml
场景地点: 机场贵宾通道外与休息廊
出场角色: 姜栀、许知微、林特助、机场工作人员、跟拍媒体
场景目标: 展示外部舆论如何继续反咬，并让“追人”本身变成公开风险。
场景阻力: 外面已经有媒体和代拍守着，林特助拿来的临时通道只能保七分钟；姜栀一旦被认出来，偷拍视频就会升级成“追到机场炒作”。
情节概要: 三人从侧通道进贵宾区，外侧仍有镜头扑过来。姜栀差点被认出，只能摘下耳饰和外套临时遮挡。许知微低声提醒她别再硬冲，姜栀嘴上答应，手里却悄悄打开录音。
关键对话: “你再上一次镜头，他们就能把‘偷档’写成‘追着人不放’。” / “我知道，所以这次我不抢镜头，我抢话。”

场景代价/变化: 姜栀为这次接触主动承受第二轮舆论风险，并埋下对许知微的隐瞒。
```

## 推进校验

- 本章不能让应岚一堵就全说，只能给版本号、接口名和项目性质。
- 本章的代价必须同时落在两头：姜栀继续背“偷档报复”热搜，许知微继续承担被家里切开的风险。
- 机场公开场域要真实制造压迫，不能写成两人随便进去就聊完。
- 新证据以录音形式存在，但录音本身也会成为后续关系风险。
- 章末要明确：执行人已经抓到半口供，下一层目标转成接口人柏宁。

## 关键元素

### 伏笔处理
- `应岚执行线`: 本章兑现为项目版本号与接口人，不让她停留在名字层。
- `偷拍视频热搜`: 继续反咬，不许它只在前章出现一次就消失。

### 新伏笔
- `归位计划3.2`: 说明归国包装不是零散操作，而是明确项目。
- `柏宁`: 作为应岚的接口人，被抛成下一章的新门。

### 设定展示
- `机场贵宾通道`: 让公开场域和阶层通道一起产生压迫，体现“追人”不是私密对话而是带舆论风险的行动。
- `项目执行链`: 从家族经手 -> 外部执行 -> 接口人，逐步压实旧包装的流程感。

### 角色状态变更
- `姜栀`: 从手握照片的人变成手握半段录音的人，但她也因此把自己推到更危险的公开位置。
- `许知微`: 再次站到“要答案”和“要保门”之间，她对姜栀的戒心继续加深。
- `应岚`: 从文件里的名字升级成真正会说话的人，但她仍牢牢把自己留在“执行层”安全线内。
