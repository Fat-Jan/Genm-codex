## 快速创建

### 修仙世界模板
```yaml
力量体系: 炼气→筑基→金丹→元婴→化神→大乘
主要势力: 正道宗门、魔道、散修联盟
核心资源: 灵石、功法、丹药
特殊规则: 渡劫、心魔、寿元限制
```

### 都市异能模板
```yaml
力量来源: 基因觉醒/外星科技/灵气复苏
组织分布: 官方异能局、地下组织、独行异能者
社会形态: 隐藏于常人之中/半公开
核心矛盾: 异能者与普通人/不同势力之间
```

### 末世废土模板
```yaml
灾难原因: 病毒/核战/陨石/灵气复苏
生存环境: 辐射区/安全区/移动基地
资源稀缺: 食物、水、干净空气
核心冲突: 生存竞争、人性考验、重建秩序
```
