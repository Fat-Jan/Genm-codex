```yaml

英文名/别名:
性别:
年龄:
身份/职业:
阵营:
```

## 外貌特征

```yaml
身高:
体型:
发型/发色:
眼睛:
显著特征:
穿着风格:
```

## 性格设定

```yaml
核心性格:
表面性格:
内心性格:
说话方式:

口头禅:
```

## 背景故事

```yaml
出身背景:
```


```yaml

```

## 剧情状态

```yaml
当前状态:
所在位置:
近期目标:
伏笔关联:
出场章节: [1, 5, 10]
```

---

## 快速创建

### 主角模板
```yaml
角色名: [姓名]
年龄: 20-25岁
身份: 普通人/学生/打工人
核心性格: 坚韧不拔/机智过人/善良正直
核心动机: 改变现状/保护重要之人/追求真相
金手指: [特殊能力或机遇]
```

### 反派模板
```yaml
角色名: [姓名]
身份: [高位者/同行者/隐藏者]
核心性格: 野心勃勃/冷酷无情/偏执极端
核心动机: 权力/复仇/理想
与主角关系: [对立原因]
```

### 女主/重要配角模板
```yaml
角色名: [姓名]
与主角关系: [恋人/伙伴/导师/对手]
核心性格: [补充主角的特质]
作用: [情感线/助攻/制衡]
```
