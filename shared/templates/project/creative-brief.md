## 简报模板

```markdown

示例：一个被判定为废材的少年，意外获得上古传承，从此踏上逆袭之路
```

## 基于简报自动生成

### 角色卡生成

从主角人设关键词自动扩展：

```yaml
性格:坚毅 →
  - 面对困难不轻易放弃
  - 有强烈的自尊心和责任感
  - 可能过于固执

背景:废材 →
  - 被家族/宗门轻视
  - 有隐藏的潜力或特殊体质
  - 逆袭空间大

金手指:系统 →
  - 系统类型：任务/签到/抽奖/兑换
  - 系统限制：防止过于无敌
  - 成长节奏：前期辅助，后期融合
```

### 世界观匹配

根据题材自动选择世界观模板：

| 题材 | 默认世界观模板 |
|------|---------------|
| 修仙 | build/templates/genres/修仙.md |
| 玄幻 | build/templates/genres/西幻.md |
| 都市 | build/templates/genres/都市日常.md |
| 历史 | build/templates/genres/历史古代.md |
| 悬疑 | build/templates/genres/悬疑灵异.md |

### 总纲规划

基于目标字数自动生成卷纲：

```yaml
30万字（短篇）:
  - 第1卷: 觉醒崛起 (1-30章)
  - 第2卷: 初露锋芒 (31-60章)
  - 第3卷: 终局之战 (61-100章)

100万字（中篇）:
  - 第1卷: 初入江湖 (1-50章)
  - 第2卷: 崭露头角 (51-150章)
  - 第3卷: 名震一方 (151-250章)
  - 第4卷: 登顶之路 (251-350章)
  - 第5卷: 终章 (351-500章)

200万字+（长篇）:
  - 增加更多副本、地图、境界层次
  - 每卷约50-100章
  - 预留支线扩展空间
```

## 使用流程

### Step 1: 填写简报

复制本模板，用一句话和关键词快速填写。

### Step 2: 运行初始化

```bash
/novel-init --brief creative-brief.md
```

### Step 3: 自动生成

系统基于简报自动生成：

1. 完整角色卡（主角+主要配角）
2. 匹配题材的世界观设定
3. 3-5卷总纲规划
4. 前10章详细章纲

### Step 4: 人工调整

基于生成内容进行微调和个性化修改。

## 示例简报

### 示例：废材修仙流

**一句话概括**：被判定为废灵根的世家子弟，意外获得上古丹道传承，以丹入道，逆天改命。

**题材类型**：修仙 - 凡人流
**目标字数**：200万字
**目标平台**：番茄小说

**关键爽点**：

- 废材逆袭
- 金手指（丹道传承）
- 越级战斗
- 装X打脸
- 升级突破

**主角人设**：

- 性格：坚毅 + 腹黑
- 背景：废材（伪灵根）
- 金手指：上古丹方传承
- 目标：证道长生，查明父母失踪真相

**预期风格**：热血激昂 + 智斗
