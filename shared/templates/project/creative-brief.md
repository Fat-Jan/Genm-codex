> 快速启动新书的极简创作文档

## 简报模板

### 核心概念

**一句话概括**：

```markdown
[主角] + [核心冲突] + [独特卖点]

示例：一个被判定为废材的少年，意外获得上古传承，从此踏上逆袭之路
```

### 基础设定

| 项目 | 内容 |
|------|------|
| **题材类型** | □ 修仙 □ 玄幻 □ 都市 □ 历史 □ 科幻 □ 悬疑 □ 言情 □ 其他：___ |
| **子流派** | □ 凡人流 □ 系统流 □ 无敌流 □ 种田流 □ 签到流 □ 其他：___ |
| **目标平台** | □ 番茄 □ 起点 □ 七猫 □ 飞卢 □ 晋江 □ 其他：___ |
| **目标字数** | □ 30万（短篇） □ 100万（中篇） □ 200万+（长篇） |

### 母题选择

如果你现在还只有一个模糊想法，先选：

- `1` 个世界观母题
- `0-1` 个机制母题
- `0-1` 个包装 / 关系母题

参考：

- `shared/references/writing/worldview-motif-catalog.md`

建议不要在立项阶段一次叠超过 `3` 个母题。

| 项目 | 内容 |
|------|------|
| **世界观母题** | 如：时间循环 / 末日乌托邦 / 神话复苏 / 虚拟现实 / AI崛起 / 梦境入侵 / 万物有灵 |
| **机制母题** | 如：规则怪谈 / 无限流副本 / 倒计时 / 系统绑定 / 快穿 / 穿书 / 记忆缺失 |
| **包装 / 关系母题** | 如：历史脑洞 / 古言脑洞 / 现言脑洞 / 女频悬疑 / 科幻末世 / 游戏体育 / 娱乐圈 / 双主角 |
| **核心异常** | 这个世界最反常、最值得点开的地方是什么 |
| **明面规则** | 读者在前3章内应该知道的规则 |
| **关键代价** | 主角每次利用这个设定要付出什么 |
| **隐藏真相** | 这个母题后面真正的大秘密是什么方向 |
| **首屏世界观钩子** | 前100-300字能抛出的最强异常 |

如果这里填不顺：

- 不要急着锁 `总纲`
- 不要急着冻结开篇包装
- 先把母题和代价写顺，再往下走

## 基于简报自动生成

### 角色卡生成

从主角人设关键词自动扩展：

```yaml
性格:坚毅 →
  - 面对困难不轻易放弃
  - 有强烈的自尊心和责任感
  - 可能过于固执

背景:废材 →
  - 被家族/宗门轻视
  - 有隐藏的潜力或特殊体质
  - 逆袭空间大

金手指:系统 →
  - 系统类型：任务/签到/抽奖/兑换
  - 系统限制：防止过于无敌
  - 成长节奏：前期辅助，后期融合
```

### 世界观匹配

根据题材和已选母题，选择最接近的世界观模板：

| 题材 | 默认世界观模板 |
|------|---------------|
| 修仙 | shared/templates/genres/修仙.md |
| 玄幻 | shared/templates/genres/西幻.md |
| 都市 | shared/templates/genres/都市日常.md |
| 历史 | shared/templates/genres/历史古代.md |
| 悬疑 | shared/templates/genres/悬疑灵异.md |

### 总纲规划

基于目标字数自动生成卷纲：

```yaml
30万字（短篇）:
  - 第1卷: 觉醒崛起 (1-30章)
  - 第2卷: 初露锋芒 (31-60章)
  - 第3卷: 终局之战 (61-100章)

100万字（中篇）:
  - 第1卷: 初入江湖 (1-50章)
  - 第2卷: 崭露头角 (51-150章)
  - 第3卷: 名震一方 (151-250章)
  - 第4卷: 登顶之路 (251-350章)
  - 第5卷: 终章 (351-500章)

200万字+（长篇）:
  - 增加更多副本、地图、境界层次
  - 每卷约50-100章
  - 预留支线扩展空间
```

## 使用流程

### Step 1: 填写简报

复制本模板，用一句话和关键词快速填写。

如果一句话 premise 还不稳，先回填“母题选择”这一段，再写一句话概括。

### Step 2: 运行初始化

```bash
/novel-init --brief creative-brief.md
```

### Step 3: 自动生成

系统基于简报自动生成：

1. 完整角色卡（主角+主要配角）
2. 匹配题材的世界观设定
3. 3-5卷总纲规划
4. 前10章详细章纲

### Step 4: 人工调整

基于生成内容进行微调和个性化修改。

## 示例简报

### 示例：废材修仙流

**一句话概括**：被判定为废灵根的世家子弟，意外获得上古丹道传承，以丹入道，逆天改命。

**题材类型**：修仙 - 凡人流
**目标字数**：200万字
**目标平台**：番茄小说

**关键爽点**：

- 废材逆袭
- 金手指（丹道传承）
- 越级战斗
- 装X打脸
- 升级突破

**主角人设**：

- 性格：坚毅 + 腹黑
- 背景：废材（伪灵根）
- 金手指：上古丹方传承
- 目标：证道长生，查明父母失踪真相

**预期风格**：热血激昂 + 智斗
