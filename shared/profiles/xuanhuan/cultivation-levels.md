# 境界体系设计指南

本文档定义玄幻/修仙题材的境界体系设计规则。

---

## 境界体系原则

### 核心原则

1. **层次分明**：每个境界有明确的实力差距
2. **压制合理**：大境界压制不可轻易逾越
3. **进阶有据**：突破需要资源、感悟、机缘
4. **成长可见**：境界提升带来明显变化

### 境界差距规则

| 差距 | 战斗结果 | 越级可能性 |
|------|----------|------------|
| 同境界 | 胜负五五开 | 正常 |
| 小境界差距 | 优势明显 | 需特殊手段 |
| 大境界差距 | 压制碾压 | 极难，需重大代价 |
| 两个大境界 | 必败 | 几乎不可能 |

## 经典境界体系

### 传统修仙体系

```markdown
凡人境界：
  炼气期 → 筑基期 → 金丹期 → 元婴期 → 化神期 → 炼虚期 → 合体期 → 大乘期 → 渡劫期

每个大境界细分：
  初期 → 中期 → 后期 → 圆满/巅峰
```

### 境界特征

| 境界 | 寿命 | 核心能力 | 标志性特征 |
|------|------|----------|------------|
| 炼气期 | 100-150年 | 灵气入体 | 飞行术、基础法术 |
| 筑基期 | 200-300年 | 筑建道基 | 神识初成、御剑飞行 |
| 金丹期 | 500-800年 | 金丹大道 | 寿元大增、法宝操控 |
| 元婴期 | 1000-2000年 | 元婴出窍 | 神识化形、瞬移 |
| 化神期 | 3000-5000年 | 元婴化神 | 沟通天地、领域雏形 |
| 炼虚期 | 5000-10000年 | 炼虚合道 | 空间操控 |
| 合体期 | 数万年 | 身魂合一 | 创造法则 |
| 大乘期 | 十万年+ | 圆满之境 | 掌控规则 |
| 渡劫期 | 不死 | 渡过天劫 | 飞升在即 |

## 境界突破设计

### 突破要素

1. **资源积累**：丹药、灵石、天材地宝
2. **感悟提升**：战斗、历练、顿悟
3. **心境磨砺**：心魔、执念、道心
4. **机缘巧合**：传承、秘境、奇遇

### 突破场景模板

```markdown
# 突破场景模板

## 铺垫阶段（1-3章前）
- 发现瓶颈
- 寻找突破契机
- 准备资源/功法

## 突破前奏（同章）
- 进入关键状态
- 调动全身资源
- 面临心魔/考验

## 突破时刻
- 瓶颈碎裂
- 天地异象
- 实力飞跃

## 突破后
- 巩固境界
- 展示新能力
- 引发后续事件
```

### 突破描写示例

```markdown
林轩盘坐于青石之上，丹田处灵气疯狂旋转。

这一刻，他感觉自己像是一个即将爆炸的气球。金丹后期的瓶颈，如同一道无形的墙，将他牢牢困住。

"不够...还不够..."

林轩咬牙，将那株千年灵芝的最后一丝药力炼化。天地灵气如潮水般涌入，在他体内形成一道巨大的漩涡。

"破！"

随着一声低喝，林轩只觉识海中传来一声脆响——那道桎梏，碎了。

天地灵气疯狂涌入，丹田处金光大盛，一道冲天光柱直破云霄。

林轩缓缓睁开眼，眸中金芒流转。

金丹圆满，终于成了。
```

## 境界体系禁忌

### 绝对禁止

1. **无铺垫突破**：突破必须有前置铺垫
2. **随意打破压制**：大境界压制必须有合理理由
3. **境界倒退**：除非特殊设定，否则不允许
4. **境界混乱**：前后文境界要一致

### 需要预警

1. **快速连升**：短时间内连续突破
2. **完美突破**：每次突破都是完美状态
3. **无代价越级**：越级战斗没有代价
