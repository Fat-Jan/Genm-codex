# 剧情模式设计指南

本文档定义玄幻/修仙题材的常用剧情模式和设计方法。

---

## 黄金三章设计

### 第一章：开篇定调

```yaml
# 第一章必含要素

核心任务:
  - 建立主角形象（姓名、身份、目标）
  - 展示世界观一角
  - 暗示/展现金手指
  - 制造第一个冲突
  - 章末留钩子

开篇钩子类型:
  危机式: 主角面临生死危机
  重生式: 带着前世记忆重生
  废材式: 被嘲笑欺辱，为打脸铺垫
  神秘式: 发现神秘物品/传承

字数要求: 4000-5000字
```

**第一章模板**：

```markdown
## 第一章：废材逆袭式开篇

【场景】宗门考核场
【人物】林轩（主角）、路人甲（嘲笑者）、路人乙（同情者）

【结构】
1. 开场：考核结果公布，林轩倒数第一（200字）
2. 冲突：被人嘲讽欺辱（800字）
3. 展示：主角内心坚韧，不甘屈服（500字）
4. 转折：金手指暗示（神秘玉佩发光）（300字）
5. 钩子：章末暗示机缘将至（200字）

【开头示例】
"林轩，外门考核你又是倒数第一，再这样下去，迟早被逐出宗门。"

林轩低着头，没有反驳。没人知道，他丹田中那枚神秘玉佩，今夜突然发出了微弱的光芒...
```

### 第二章：展开剧情

```yaml
# 第二章必含要素

核心任务:
  - 金手指首次展现
  - 制造资源争夺冲突
  - 主角获得初步提升
  - 铺垫即将到来的危机

字数要求: 4000-6000字
```

### 第三章：首个小高潮

```yaml
# 第三章必含要素

核心任务:
  - 第一次小打脸或突破
  - 展示更大的世界观
  - 铺垫新的危机或机缘
  - 建立主角初步威望

字数要求: 4000-6000字
```

## 常用剧情模式

### 1. 宗门考核模式

```yaml
# 宗门考核剧情结构

阶段一：考核前
  - 铺垫考核重要性
  - 展示对手实力
  - 主角被轻视

阶段二：考核中
  - 意外状况
  - 主角展现实力
  - 众人震惊

阶段三：考核后
  - 地位提升
  - 获得资源/机会
  - 引发新的冲突
```

### 2. 秘境探索模式

```yaml
# 秘境探索剧情结构

阶段一：入秘境
  - 秘境开启
  - 规则介绍
  - 队伍分组

阶段二：探索中
  - 遇到危险
  - 发现机缘
  - 队友冲突/合作

阶段三：秘境高潮
  - 核心宝物争夺
  - 强敌出现
  - 主角突破/收获

阶段四：离开秘境
  - 收获盘点
  - 引发后续影响
```

### 3. 拍卖会模式

```yaml
# 拍卖会剧情结构

阶段一：拍卖前
  - 主角需要某物
  - 资源不足
  - 结识关键人物

阶段二：拍卖中
  - 竞价过程
  - 意外状况
  - 最终获得

阶段三：拍卖后
  - 被人盯上
  - 冲突爆发
  - 展示实力
```

### 4. 比武擂台模式

```yaml
# 比武擂台剧情结构

阶段一：赛前
  - 比武原因
  - 对手介绍
  - 主角被看衰

阶段二：比赛中
  - 上半场劣势
  - 中场转折
  - 下半场反杀

阶段三：比赛后
  - 众人反应
  - 获得奖励
  - 新的挑战
```

## 剧情节奏控制

### 单章情绪曲线

```yaml
# 章节情绪设计

标准型:
  铺垫: 30%
  冲突: 40%
  高潮: 20%
  钩子: 10%

高潮型:
  快速铺垫: 15%
  连续冲突: 50%
  高潮爆发: 30%
  悬念: 5%

过渡型:
  日常描写: 40%
  小冲突: 30%
  线索铺垫: 20%
  钩子: 10%
```

### 卷级节奏规划

```yaml
# 一卷（约30章）节奏规划

第1-3章：黄金三章，开篇定调
第4-8章：日常铺垫，建立关系
第9-15章：第一个大事件
第16-20章：过渡调整，埋伏笔
第21-28章：卷末高潮，大冲突
第29-30章：收尾，衔接下卷
```

## 冲突设计

### 冲突类型

| 类型 | 说明 | 解决方式 |
|------|------|----------|
| 资源冲突 | 争夺修炼资源 | 实力比拼、智谋较量 |
| 感情冲突 | 感情纠葛 | 沟通解决、剧情推动 |
| 势力冲突 | 宗门/家族争斗 | 集体对抗、个人突破 |
| 理念冲突 | 道不同 | 辩论、战斗 |
| 利益冲突 | 利益分配 | 妥协、对抗 |

### 冲突升级模板

```markdown
## 冲突升级七步法

1. **初始冲突**：小摩擦，口头冲突
2. **冲突升级**：利益相关，态度强硬
3. **冲突爆发**：正面交锋，有输赢
4. **余波处理**：一方不服，寻求报复
5. **二次冲突**：更大规模，更多人物
6. **高潮对决**：决定性一战
7. **后续影响**：格局改变，新平衡
```

## 伏笔设计

### 伏笔类型

| 类型 | 揭晓时机 | 示例 |
|------|----------|------|
| 短线伏笔 | 3-15章 | 副本内埋设，副本结束回收 |
| 中线伏笔 | 30-80章 | 金手指来源、宗门秘密 |
| 长线伏笔 | 80-200章 | 主角身世、世界观真相 |

### 伏笔埋设规则

```yaml
# 伏笔埋设节奏

原则: 每18章埋设1个新伏笔，每50章回收1个旧伏笔

状态流转:
  - 🟢潜伏：已埋设，未引起注意
  - 🟡铺垫：开始有线索指向
  - 🟠可回收：时机成熟，可以揭晓
  - ⚪已回收：已经揭晓，尘埃落定

示例:
  短线伏笔: "副本入口处的奇怪符文" → 10章后揭示是封印
  中线伏笔: "师父眼神中的异样" → 50章后揭示师父有隐情
  长线伏笔: "主角胎记的形状" → 100章后揭示是古老传承印记
```
