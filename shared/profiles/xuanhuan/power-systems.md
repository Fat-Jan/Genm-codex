# 力量体系设计指南

本文档定义玄幻/修仙题材的力量体系设计规则。

---

## 力量体系分类

### 1. 内在力量

| 类型 | 说明 | 代表 |
|------|------|------|
| 灵力 | 修仙者基础能量 | 灵气、真元 |
| 神识 | 精神力量 | 神识扫描、传音 |
| 肉身 | 体修力量 | 气血、筋骨 |
| 神魂 | 灵魂力量 | 元婴、神魂攻击 |

### 2. 外在力量

| 类型 | 说明 | 获取方式 |
|------|------|----------|
| 法宝 | 炼制宝物 | 炼器、机缘 |
| 阵法 | 天地之力 | 布阵、传承 |
| 符箓 | 封印之力 | 绘制、购买 |
| 丹药 | 辅助力量 | 炼丹、交易 |

### 3. 特殊力量

| 类型 | 说明 | 限制条件 |
|------|------|----------|
| 天赋神通 | 血脉能力 | 觉醒条件 |
| 传承之力 | 前人遗留 | 接受传承 |
| 诅咒之力 | 代价力量 | 负面效果 |
| 信仰之力 | 众生意念 | 需要信徒 |

## 金手指设计

### 金手指类型

```yaml
# 金手指分类

成长型:
  - 系统空间：随主角成长解锁功能
  - 神秘传承：逐步觉醒的上古功法
  - 天赋觉醒：血脉力量逐渐显现

固定型:
  - 空间农场：种田流核心
  - 回溯能力：时间类金手指
  - 预知能力：看见未来片段

限制型:
  - 寿命消耗：使用力量消耗寿命
  - 反噬风险：每次使用有代价
  - 条件触发：特定条件才能使用
```

### 金手指设计规则

1. **有限制**：不能无限制使用
2. **有成长**：随主角成长而进化
3. **有代价**：使用需要付出代价
4. **有来源**：来源要合理，最好有悬念

### 金手指禁忌

| 禁忌 | 说明 | 严重程度 |
|------|------|----------|
| 无限制使用 | 想用就用，毫无代价 | critical |
| 一开始全开 | 所有功能开局就有 | warning |
| 无成长性 | 从头到尾一样 | warning |
| 来源不明 | 没有任何解释 | warning |

## 功法体系

### 功法等级

```markdown
品级划分：
  黄阶 → 玄阶 → 地阶 → 天阶 → 圣阶 → 神阶

每个品级细分：
  下品 → 中品 → 上品 → 极品
```

### 功法类型

| 类型 | 说明 | 修炼要点 |
|------|------|----------|
| 主修功法 | 核心修炼法门 | 决定境界上限 |
| 辅助功法 | 辅助修炼 | 加速修炼、辅助战斗 |
| 战斗功法 | 攻击防御 | 招式、秘术 |
| 特殊功法 | 特殊能力 | 炼丹、炼器、阵法 |

### 功法设计模板

```markdown
## 功法模板

**名称**：青云剑诀
**品级**：天阶上品
**来源**：青云宗镇派之宝
**修炼条件**：
  - 境界要求：筑基期以上
  - 资质要求：木灵根
  - 其他条件：需得青云宗核心弟子身份

**功法层次**：
  - 第一层：青云剑气（筑基期）
  - 第二层：剑分化影（金丹期）
  - 第三层：万剑归宗（元婴期）
  - 第四层：青云化界（化神期）

**特殊能力**：
  - 剑心通明：感知危险
  - 御剑飞行：消耗低
  - 剑气护体：被动防御

**修炼风险**：
  - 需以本命精血祭炼
  - 修炼过程中可能走火入魔
  - 若心术不正，剑气反噬
```

## 法宝体系

### 法宝等级

```markdown
品级划分：
  法器 → 灵器 → 法宝 → 仙器 → 神器

每个品级细分：
  下品 → 中品 → 上品 → 极品
```

### 法宝类型

| 类型 | 说明 | 代表 |
|------|------|------|
| 攻击型 | 主攻杀伐 | 飞剑、宝塔 |
| 防御型 | 主防御 | 护盾、铠甲 |
| 辅助型 | 功能性 | 储物袋、飞舟 |
| 特殊型 | 特殊能力 | 传讯、追踪 |

### 法宝炼制

```markdown
## 法宝炼制要素

1. **材料**：
   - 基础材料：凡铁、灵矿
   - 核心材料：天材地宝
   - 辅助材料：灵液、符文

2. **炼制者**：
   - 自身境界决定法宝上限
   - 炼器造诣影响成功率

3. **炼制过程**：
   - 熔炼材料
   - 刻录符文
   - 打入神识烙印
   - 渡劫（高阶法宝）

4. **法宝认主**：
   - 滴血认主
   - 神识烙印
   - 契约绑定
```

## 战斗力量体系

### 战斗要素

```markdown
## 战斗力量层次

1. **灵力对抗**
   - 灵力强度
   - 灵力储备
   - 恢复速度

2. **功法对抗**
   - 功法品级
   - 修炼层次
   - 组合运用

3. **法宝对抗**
   - 法宝品级
   - 认主程度
   - 配合默契

4. **战斗意识**
   - 战斗经验
   - 临场反应
   - 战术运用

5. **特殊因素**
   - 地利
   - 人和
   - 天时
```

### 越级战斗设计

```yaml
# 越级战斗条件

合理越级因素:
  功法压制:
    - 高阶功法压制低阶
    - 克制属性

  法宝优势:
    - 法宝品级碾压
    - 本命法宝加成

  特殊能力:
    - 金手指爆发
    - 血脉觉醒
    - 传承之力

  外部因素:
    - 阵法加成
    - 丹药辅助
    - 队友配合

越级代价:
  - 燃烧寿元
  - 法宝受损
  - 修养恢复
  - 资源消耗
```

## 力量体系一致性检查

### 检查清单

```markdown
[ ] 力量来源有合理解释
[ ] 力量使用有明确限制
[ ] 力量成长有逻辑路径
[ ] 力量体系前后一致
[ ] 金手指有代价和成长性
[ ] 功法等级与效果匹配
[ ] 法宝能力与品级对应
[ ] 战斗结果符合力量对比
```

### 常见错误

| 错误 | 示例 | 修正 |
|------|------|------|
| 力量混乱 | 一会用灵力一会用魔力 | 统一力量体系 |
| 忘记限制 | 空间无限存放 | 设定容量限制 |
| 无代价使用 | 金手指随便用 | 添加使用代价 |
| 等级失衡 | 黄阶功法打天阶 | 遵守等级压制 |
