# 晋江文学城平台风格指南

## 平台概述

晋江文学城是一个以**感情流、CP互动、细腻描写**为核心特征的 female-oriented 阅读平台。该平台用户追求**情感共鸣和人物魅力**，喜欢**细腻的情感描写**。

## 核心特征

### 1. 情感要求
- **感情细腻**：情感描写要细腻入微
- **CP感强**：主角之间的化学反应要强烈
- **情感发展**：感情发展要自然合理
- **心理描写**：内心活动要丰富详细

### 2. 节奏特点
- **情感节奏**：以情感发展为主线
- **张弛有度**：紧张与轻松情节交替
- **细节铺陈**：重视细节的积累和铺垫
- **情感爆发**：情感高潮要足够动人

### 3. 内容偏好
- **甜宠情节**：甜蜜宠溺的互动
- **虐心情节**：适度的心痛情节
- **误会冲突**：因误会产生的矛盾
- **和解甜蜜**：误会解除后的甜蜜

## 与其他平台对比

| 特征 | 晋江 | 起点 | 番茄 |
|------|------|------|------|

## 情感发展模式

### 一见钟情
```
1. 初次相遇：印象深刻的第一面
2. 心动瞬间：某个细节让人心动
3. 情感确认：意识到自己喜欢对方
4. 感情发展：开始主动接近对方
```

### 日久生情
```
1. 日常相处：频繁的日常接触
2. 了解加深：逐渐了解对方的优点
3. 情感萌芽：在不经意间产生感情
4. 感情确认：确认自己的心意
```

### 欢喜冤家
```
1. 初次冲突：一开始就产生矛盾
2. 持续斗嘴：经常吵架斗嘴
3. 情感转变：在冲突中产生感情
4. 心意相通：最终确认彼此心意
```

### 破镜重圆
```
1. 曾经相爱：过去有美好的回忆
2. 因故分离：因为某些原因分开
3. 重逢相遇：多年后再次相遇
4. 重新开始：克服困难重新在一起
```

3. **成长轨迹**：人物要有成长和变化

## 避免雷区

### 情感雷区
- **感情突兀**：感情发展过于突兀
- **人设崩塌**：人物性格突然改变
- **过度虐心**：虐心情节过度
- **感情廉价**：感情发展过于廉价

### 人物雷区
- **圣母女主**：过分圣母的女主角
- **渣男男主**：真正的渣男男主角
- **恶毒女配**：过分恶毒的女配角
- **完美人设**：过分完美不真实

### 情节雷区
- **强行误会**：为了误会而误会
- **狗血情节**：过分狗血的情节
- **拖沓节奏**：节奏过于拖沓
- **烂尾结局**：结局处理不当
