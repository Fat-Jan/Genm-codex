# 起点中文网平台风格指南

## 平台概述

起点中文网是一个以**深度设定、逻辑严谨、智斗布局**为核心特征的付费阅读平台。该平台用户追求**合理性和深度**，喜欢**有逻辑的爽感**。

## 核心特征

### 1. 设定要求
- **世界观完整**：需要有完整的世界观设定
- **力量体系**：力量体系要严谨，有明确的等级划分
- **逻辑严密**：情节发展要符合逻辑，不能突兀
- **细节丰富**：各种设定要详细，让读者信服

### 2. 节奏特点
- **张弛有度**：有紧张的战斗，也有轻松的日常
- **铺垫充分**：重要情节需要充分铺垫
- **爆发有力**：铺垫后的爆发要足够有力
- **长线布局**：要有长远的情节规划

### 3. 内容偏好
- **智斗为主**：智慧较量比单纯武力更重要
- **策略布局**：主角要有深谋远虑
- **合理升级**：实力提升要合理，不能过快
- **人物成长**：主角要有明显的成长轨迹

## 设定规范

### 世界观设定
```
1. 世界背景：完整的世界历史和现状
2. 地理设定：详细的地理位置和特点
3. 势力分布：各方势力的分布和关系
4. 文化体系：独特的文化和社会制度
```

### 力量体系
```
1. 等级划分：明确的力量等级体系
2. 进阶条件：每个等级的提升条件
3. 能力特点：每个等级的能力特征
4. 瓶颈设定：合理的瓶颈和突破方式
```

### 修炼体系
```
1. 修炼方法：具体的修炼功法和步骤
2. 资源需求：修炼需要的各种资源
3. 时间成本：修炼需要的时间和努力
4. 风险设定：修炼可能遇到的风险
```

## 写作原则

### 合理性原则
1. **实力匹配**：角色实力与行为要匹配
2. **逻辑推理**：情节发展要符合逻辑
3. **因果关系**：行为和结果要有因果关系
4. **设定一致**：不能违反自己设定的规则

### 智斗原则
1. **策略优先**：优先使用智慧解决问题
2. **布局深远**：要有长远的眼光和布局
3. **信息优势**：利用信息差获得优势
4. **心理博弈**：重视心理层面的较量

### 成长原则
1. **循序渐进**：实力提升要循序渐进
2. **努力付出**：成长需要努力和付出
3. **机缘辅助**：机缘可以加速但不可替代努力
4. **瓶颈突破**：要有合理的瓶颈和突破

## 章节结构

### 标准结构（3000-5000字）
- **铺垫**（500-800字）：承接上文，引出新的情节
- **发展**（1500-2000字）：情节推进，矛盾升级
- **高潮**（800-1200字）：冲突爆发，智斗展开
- **解决**（500-800字）：问题解决，收获成果
- **伏笔**（200-500字）：为后续情节埋下伏笔

### 节奏控制
- **紧张情节**：战斗、智斗、危机处理
- **轻松情节**：日常、感情、幽默桥段
- **过渡情节**：修炼、探索、信息收集
- **高潮情节**：大事件、重要战斗、重大突破

## 人物设定

### 主角特征
```
智商：高智商，善于思考和规划
性格：沉稳冷静，不轻易冲动
能力：全面发展，不依赖单一能力
成长：循序渐进，有明确的成长轨迹
```

### 反派特征
```
智商：在线，有一定的智慧和能力
威胁：对主角构成真正的威胁
动机：有合理的动机和目标
成长：也会成长和进步
```

### 配角特征
```
个性：各有特色，不脸谱化
作用：在关键时刻起到重要作用
成长：也有自己的成长轨迹
关系：与主角的关系要合理发展
```

## 智斗模板

### 信息战
```
1. 信息收集：主角收集对手信息
2. 信息分析：分析对手的弱点和习惯
3. 信息误导：故意放出假信息
4. 信息利用：利用信息差制定策略
```

### 心理战
```
1. 心理分析：分析对手的心理状态
2. 心理施压：通过各种方式给对手压力
3. 心理诱导：诱导对手做出错误判断
4. 心理崩溃：让对手心理崩溃
```

### 策略战
```
1. 目标设定：明确要达成的目标
2. 方案设计：设计多个备选方案
3. 风险评估：评估每个方案的风险
4. 方案执行：选择最佳方案执行
5. 应变调整：根据情况调整策略
```

## 常见题材

### 修仙玄幻
- **优势**：设定丰富，力量体系完整
- **重点**：修炼体系、等级制度、势力争斗
- **代表**：《斗破苍穹》《完美世界》

### 都市异能
- **优势**：贴近现实，容易产生共鸣
- **重点**：异能的合理运用、都市生活的结合
- **代表**：《全球高武》《大王饶命》

### 历史军事
- **优势**：历史背景深厚，策略性强
- **重点**：历史事件的改编、军事策略
- **代表**：《赘婿》《庆余年》

### 科幻未来
- **优势**：想象力丰富，科技设定
- **重点**：科技原理、未来社会、星际战争
- **代表**：《深空彼岸》《我们生活在南京》

## 成功要素

### 设定要素
1. **完整性**：世界观要完整，不留漏洞
2. **独特性**：要有自己的特色和创新
3. **一致性**：设定要前后一致，不自相矛盾
4. **详细性**：重要设定要详细说明

### 情节要素
1. **逻辑性**：情节发展要符合逻辑
2. **连贯性**：情节之间要有连贯性
3. **节奏感**：节奏要张弛有度
4. **悬念感**：要有足够的悬念吸引读者

### 人物要素
1. **立体性**：人物要立体，不脸谱化
2. **成长性**：人物要有明显的成长
3. **合理性**：人物行为要符合设定
4. **魅力性**：人物要有吸引人的特质

## 避免雷区

### 设定雷区
- **力量失衡**：主角实力提升过快
- **设定矛盾**：前后设定不一致
- **逻辑漏洞**：情节发展不合逻辑
- **信息缺失**：重要信息没有交代

### 情节雷区
- **主角降智**：主角做出不符合智商的行为
- **反派弱智**：反派智商过低
- **强行转折**：情节转折过于突兀
- **注水情节**：无关紧要的拖沓情节

### 人物雷区
- **性格突变**：人物性格突然改变
- **能力bug**：人物能力超出设定范围
- **关系混乱**：人物关系发展不合理
- **配角抢戏**：配角过于突出影响主角
