## 一、钩子类型分类 (Hook Types)

### 1. 危机钩 (Crisis Hook)

**定义**: 以迫在眉睫的危机或威胁作为章末锚点，制造"必须知道结果"的紧迫感。

**强度判定**:
| 强度 | 特征 | 持续期待 |
|------|------|----------|
| strong | 生死攸关、不可逆转的倒计时 | 读者必须立即看下一章 |
| medium | 明确威胁但后果可控 | 读者会记住并在意 |
| weak | 隐性危机、缺乏紧迫感 | 读者可能忽略 |

**示例**:
```markdown
[strong] 他脚下一滑，整个人向悬崖坠去。下面是万丈深渊。

[medium] "明天正午，生死台上见。"她留下这句话，转身离去。

[weak] 隐约感觉有什么不好的事情要发生，但他说不上来是什么。
```

**题材适配**:
- 玄幻/武侠: 高频使用，战斗危机
- 都市: 中频，商业/情感危机
- 言情: 低频，情感危机为主

---

### 2. 悬念钩 (Mystery Hook)

**定义**: 以未解之谜或信息缺失作为章末锚点，激发读者的好奇心。

**强度判定**:
| 强度 | 特征 | 信息缺口 |
|------|------|----------|
| strong | 核心谜团、颠覆性信息即将揭示 | 读者迫切想知道答案 |
| medium | 支线谜团、有趣但非关键 | 读者好奇但可等待 |
| weak | 装饰性悬念、与主线关联弱 | 读者可能不在意 |

**示例**:
```markdown
[strong] 那块玉佩上刻的，竟然是她失散多年的亲生妹妹的名字。

[medium] 这个盒子里装着什么？他正要打开，门被敲响了。

[weak] 他总觉得在哪里见过这个人，但想不起来。
```

**题材适配**:
- 悬疑/推理: 核心手法，高频使用
- 玄幻: 中频，身世/宝物谜团
- 言情: 低频，身份/关系谜团

---

### 3. 渴望钩 (Desire Hook)

**定义**: 以主角的目标或期待作为章末锚点，让读者期待目标的达成。

**强度判定**:
| 强度 | 特征 | 目标价值 |
|------|------|----------|
| strong | 核心目标、巨大收益、突破机会 | 读者与主角共情渴望 |
| medium | 支线目标、有一定收获 | 读者期待但不迫切 |
| weak | 小目标、收获不明显 | 读者兴趣有限 |

**示例**:
```markdown
[strong] 突破金丹期的机缘，就在前方。他已等待这一刻整整三年。

[medium] 这次拍卖会上的那把剑，他志在必得。

[weak] 明天的比试，他应该能赢。
```

**题材适配**:
- 玄幻/武侠: 高频，升级/突破/夺宝
- 都市: 中频，事业/财富目标
- 言情: 转化为情感目标

---

### 4. 情绪钩 (Emotion Hook)

**定义**: 以情感张力或情绪悬念作为章末锚点，让读者为角色的情感状态牵挂。

**强度判定**:
| 强度 | 特征 | 情感张力 |
|------|------|----------|
| strong | 情感转折点、重大关系变化 | 读者情绪高度投入 |
| medium | 情感波动、关系小变化 | 读者有所触动 |
| weak | 情绪平淡、无变化 | 读者情感投入低 |

**示例**:
```markdown
[strong] 她转身离去，没有回头。这一别，不知何时才能再见。

[medium] 他不知道该不该告诉她真相。说还是不说？

[weak] 今天的约会很开心，他心情不错。
```

**题材适配**:
- 言情/甜宠: 核心手法，高频使用
- 都市: 中频，感情与事业交织
- 玄幻: 低频，作为节奏调节

---

### 5. 选择钩 (Choice Hook)

**定义**: 以关键抉择作为章末锚点，让读者期待主角的决定及其后果。

**强度判定**:
| 强度 | 特征 | 选择代价 |
|------|------|----------|
| strong | 两难困境、重大代价、不可逆选择 | 读者紧张期待 |
| medium | 有明显优劣但都不完美 | 读者有所关注 |
| weak | 选择无实质差异或代价低 | 读者兴趣有限 |

**示例**:
```markdown
[strong] 救母亲，还是救挚爱？他只能选一个。

[medium] 是接受宗门的招揽，还是继续独行？各有利弊。

[weak] 他决定明天再去看看那家店。
```

**题材适配**:
- 玄幻: 中频，道心/阵营选择
- 都市: 中频，事业/道德抉择
- 言情: 低频，情感选择

## 二、钩子强度评估矩阵

### 综合评分公式

```markdown
钩子强度分 = 紧迫度(1-10) × 信息缺口(1-10) × 情感投入(1-10) / 50
```

### 评级标准

| 分数范围 | 强度 | 读者反应 |
|---------|------|----------|
| 7-10 | strong | 必须点下一章 |
| 4-6 | medium | 会记住并继续阅读 |
| 1-3 | weak | 可能流失 |

### 快速诊断清单

```markdown
[ ] 是否有明确的时间压力？
[ ] 是否有读者迫切想知道的信息？
[ ] 是否有读者关心的角色状态？
[ ] 是否有让读者紧张的选择？
[ ] 是否有让读者期待的目标？

满足 4-5 项 → strong
满足 2-3 项 → medium
满足 0-1 项 → weak
```

### CP-01 装逼打脸 (Flex & Counter)

**结构**:
```markdown
对方轻视 → 主角展示实力 → 对方震惊/后悔
```

**必要元素**:
1. 铺垫: 反派/对手的轻视态度
2. 对照: 配角/路人的吹捧对比
3. 爆发: 主角实力的惊艳展示
4. 收尾: 对方的震惊反应 + 实质收获

**示例**:
```markdown
王浩嗤笑："就凭你也敢挑战我？"
[铺垫: 轻视]

萧炎没有回答。他抬手，一道剑气破空而出。
[爆发: 展示]

王浩的笑容僵在脸上。筑基期的威压...这怎么可能？
[收尾: 震惊]
```

**题材适配**: 都市(财力/人脉)、玄幻(战力)、职场(能力)

---

### CP-02 扮猪吃虎 (Underdog Reveal)

**结构**:
```markdown
表面弱小 → 关键时刻爆发 → 众人惊艳
```

**必要元素**:
1. 隐藏: 主角真实实力被隐藏
2. 信息差: 读者知道但角色不知道
3. 挑衅: 对方基于表象的轻视
4. 爆发: 关键时刻的惊人表现
5. 反应: 众人的震惊和态度转变

**示例**:
```markdown
[隐藏] 林动看似普通弟子，实则已突破元婴。

[挑衅] "一个小小的练气期，也敢来此？"李长老不屑。

[爆发] 林动一步踏出，元婴威压席卷全场。

[反应] "元婴！他竟然是元婴强者！"全场哗然。
```

**题材适配**: 重生、隐藏身份、低调流

---

### CP-03 越级反杀 (Underdog Victory)

**结构**:
```markdown
实力差距明显 → 主角逆袭 → 敌人不可置信
```

**必要元素**:
1. 差距: 明确的实力对比
2. 困境: 主角处于劣势
3. 奇迹: 翻盘的关键
4. 震撼: 敌人/观众的反应

**示例**:
```markdown
[差距] 金丹期的威压压得他喘不过气。

[困境] 他已被逼入死角，退无可退。

[奇迹] 他体内沉睡的力量突然苏醒...

[震撼] "不可能！你明明只是筑基期！"
```

**题材适配**: 玄幻/武侠(战斗)、竞技(比赛)、商战(逆袭)

---

### CP-04 打脸权威 (Authority Challenge)

**结构**:
```markdown
权威质疑 → 主角用实力证明 → 权威认可/尴尬
```

**必要元素**:
1. 权威: 建立权威的可信度
2. 质疑: 对主角的否定/否定判断
3. 证明: 主角用事实/实力打脸
4. 结果: 权威的态度转变

**示例**:
```markdown
[权威] 宗门首席炼丹师，炼丹界泰斗级人物。

[质疑] "这丹方根本不可能成功，简直是天方夜谭！"

[证明] 丹炉开启，一颗完美的丹药呈现在众人面前。

[结果] 老者沉默良久："...天才。真正的天才。"
```

**题材适配**: 学院、职场、技术流

---

### CP-05 反派翻车 (Villain Downfall)

**结构**:
```markdown
反派得意 → 计划破产 → 反派狼狈
```

**必要元素**:
1. 铺垫: 反派的阴谋/得意
2. 转折: 计划的关键破绽
3. 破产: 阴谋败露/计划失败
4. 结局: 反派的狼狈下场

**示例**:
```markdown
[铺垫] "我已经布下天罗地网，他插翅难逃！"反派得意大笑。

[转折] 然而，他不知道的是，计划早已泄露。

[破产] 当他踏入陷阱时，发现等待他的是——主角早已设好的埋伏。

[结局] "怎么会...这不可能！"他狼狈逃窜，颜面尽失。
```

**题材适配**: 所有题材

---

### CP-06 甜蜜超预期 (Sweet Surprise)

**结构**:
```markdown
平淡日常 → 意外惊喜 → 情感升温
```

**必要元素**:
1. 铺垫: 平淡或略带担忧的日常
2. 惊喜: 超出预期的浪漫
3. 升温: 关系的实质性进展

**示例**:
```markdown
[铺垫] 她以为他忘记了这个日子，有些失落。

[惊喜] 推开门，满屋的蜡烛和玫瑰。他站在中央。

[升温] "这是我为你准备的。"她扑入他的怀抱。
```

**题材适配**: 言情、甜宠

---

### CP-07 迪化误解 (Misunderstanding Elevation)

**结构**:
```markdown
误解 → 放大 → 意外好处
```

**必要元素**:
1. 误解: 角色被误认为比自己实际更强/更厉害
2. 放大: 误解不断被加深
3. 意外: 主角获得意外的好处/尊重
4. 维持: 误解继续存在

**示例**:
```markdown
[误解] 他只是随手一挥，对方却以为他用了绝世神通。

[放大] "此子深不可测！"消息在宗门内传开。

[意外] 他莫名其妙获得了长老的关注和资源倾斜。

[维持] 他只能装作高深莫测，"是的，正如你们所想。"
```

**题材适配**: 喜剧、无敌流、迪化流

---

### CP-08 身份掉马 (Identity Reveal)

**结构**:
```markdown
隐藏身份 → 被迫揭示 → 周围震撼
```

**必要元素**:
1. 隐藏: 主角有隐藏身份
2. 触发: 被迫揭示的契机
3. 揭示: 身份曝光的瞬间
4. 震撼: 周围人的反应

**示例**:
```markdown
[隐藏] 他化名进入宗门，无人知道他的真实身份。

[触发] 宗门危机，只有他的真实身份才能化解。

[揭示] 他摘下面具，"我乃...北域圣子。"

[震撼] 全场死寂。然后是——"圣子殿下？！"
```

**题材适配**: 马甲文、重生文、权谋

---

### CP-09 逆境翻盘 (Turn the Tables)

**结构**:
```markdown
跌入谷底 → 绝地反击 → 逆转翻盘
```

**必要元素**:
1. 谷底: 主角陷入绝境，看似毫无希望
2. 反击: 主角找到突破口，发起反攻
3. 逆转: 局势彻底翻转，主角获得胜利
4. 震撼: 众人/敌人的不可置信反应

**强度判定**:
| 强度 | 特征 | 谷底深度 |
|------|------|----------|
| strong | 生死绝境、万念俱灰后的绝地反击 | 读者绝望后狂喜 |
| medium | 困境明显、反击有一定难度 | 读者期待但不太紧张 |
| weak | 困境不深、反击较为轻松 | 读者情绪波动小 |

**示例**:
```markdown
[谷底] 他经脉尽断，灵力全失，被敌人踩在脚下。"这就是与我作对的下场。"

[反击] 他嘴角微微上扬，"你以为我毫无准备？"体内沉睡的力量突然觉醒。

[逆转] 一道金光冲天而起，将他震飞。他重新站起，威压比之前更强十倍。

[震撼] "这不可能！你明明已经废了！"敌人惊恐后退。
```

**题材适配**: 玄幻/武侠(修炼突破)、都市(商战逆袭)、竞技(逆转胜利)

---

### CP-10 获得认可 (Earn Respect)

**结构**:
```markdown
被质疑/忽视 → 证明价值 → 获得尊重/接纳
```

**必要元素**:
1. 质疑: 周围人对主角的否定/轻视/忽视
2. 证明: 主角用实际行动证明自己的价值
3. 认可: 他人态度的转变，从质疑到尊重
4. 升华: 关系的实质性改善

**强度判定**:
| 强度 | 特征 | 认可程度 |
|------|------|----------|
| strong | 全盘否定到高度认可，关系重大转变 | 读者情感强烈共鸣 |
| medium | 质疑消除，获得一定尊重 | 读者感到欣慰 |
| weak | 小误解消除，认可度一般 | 读者反应平淡 |

**示例**:
```markdown
[质疑] "他一个废物，凭什么进入核心弟子班？"众人都对他投来鄙夷的目光。

[证明] 大比之上，他一路过关斩将，击败所有质疑他的人。

[认可] "是我看走眼了。"曾经质疑他的长老主动道歉，"你是真正的天才。"

[升华] 他不仅获得了核心弟子的身份，更赢得了所有人的尊重。
```

**题材适配**: 学院流、职场文、竞技文、重生复仇

---

### CP-11 机缘获得 (Fortune Falls)

**结构**:
```markdown
意外机遇 → 获得 → 巨大收获
```

**必要元素**:
1. 意外: 机遇的偶然性、不可预期性
2. 过程: 获得机缘的过程（可简可繁）
3. 收获: 明确的收益，超出预期
4. 展望: 收获带来的未来可能性

**强度判定**:
| 强度 | 特征 | 收获价值 |
|------|------|----------|
| strong | 改变命运的机缘、巨大收益 | 读者狂喜，期待后续 |
| medium | 有价值的收获、一定惊喜 | 读者满意 |
| weak | 小机缘、收益有限 | 读者兴趣一般 |

**示例**:
```markdown
[意外] 他为躲避追杀误入一处秘境，本以为是绝路。

[过程] 秘境中央，一株散发着七彩光芒的灵草静静生长。

[收获] 他获得了传说中的七彩仙草，服用后不仅伤势痊愈，更突破了瓶颈。

[展望] 他感受到体内澎湃的力量，"有了这个，我就能参加宗门大比了。"
```

**题材适配**: 玄幻/武侠(修炼机缘)、都市(意外财富/机会)、系统文(系统奖励)

## 四、微兑现类型 (Micro-Fulfillment Types)

### 定义

微兑现是指章节中部分满足读者期待的小型奖励点，与爽点(大型兑现)不同，微兑现在是日常性的、持续性的满足感来源。

### 类型分类

| ID | 类型 | 定义 | 典型表现 |
|----|------|------|----------|
| MF-01 | 信息兑现 | 揭示部分悬念/谜团 | 得到线索、揭示部分真相 |
| MF-02 | 关系兑现 | 关系出现进展 | 好感上升、敌意下降、新关系建立 |
| MF-03 | 能力兑现 | 实力有所提升 | 获得技能、突破、感悟 |
| MF-04 | 资源兑现 | 获得物质收益 | 获得金钱、宝物、地位 |
| MF-05 | 认可兑现 | 获得他人认可 | 获得尊重、赞扬、地位提升 |
| MF-06 | 情绪兑现 | 情绪获得满足 | 报仇、平反、情感释放 |
| MF-07 | 线索兑现 | 获得重要线索 | 发现踪迹、找到证据 |

### 微兑现已爽点的关系

```markdown
爽点 = 多个微兑现的组合爆发
微兑现 = 爽点的前置积累

例如：
- 信息微兑现 × 3 + 能力微兑现 × 2 = 身份掉马爽点
- 关系微兑现 × 4 + 情绪微兑现 × 2 = 甜蜜超预期爽点
```

### 微兑现频率建议

| 题材类型 | 建议每章微兑现数 | 说明 |
|---------|-----------------|------|
| 传统爽文 | 2-3个 | 持续满足感 |
| 正剧向 | 1-2个 | 重质量而非数量 |
| 言情甜宠 | 1-2个 | 以情感兑现为主 |
| 悬疑推理 | 1个(线索兑现) | 推动剧情 |

## 五、追读力综合评分

### 微兑现评分公式

```markdown
微兑现分 = 微兑现数量 × 平均质量(1-5) / 5
```

**质量评估标准**:
| 质量等级 | 分值 | 特征 |
|---------|------|------|
| 高质量 | 5 | 收获超出预期、有惊喜感、推动主线 |
| 中上 | 4 | 收获明确、有一定满足感 |
| 中等 | 3 | 收获一般、符合预期 |
| 中下 | 2 | 收获较小、满足感不足 |
| 低质量 | 1 | 收获微弱、几乎无满足感 |

### 综合追读力得分

```markdown
追读力总分 = (钩子强度分 × 0.4) + (微兑现分 × 0.3) + (爽点强度 × 0.3)
```

**爽点强度评分**:
| 爽点表现 | 分值 |
|---------|------|
| 完整执行 strong 爽点 | 10 |
| 完整执行 medium 爽点 | 7 |
| 完整执行 weak 爽点 | 4 |
| 爽点执行不完整 | 评分减半 |
| 本章无爽点 | 0 |

### 评级标准

| 总分 | 评级 | 含义 | 建议操作 |
|------|------|------|----------|
| 8-10 | S级 | 读者必追 | 保持当前节奏 |
| 6-7 | A级 | 读者持续关注 | 可适当优化 |
| 4-5 | B级 | 读者可能继续 | 需要改进 |
| 1-3 | C级 | 读者可能流失 | 必须修改 |

### 快速诊断清单

```markdown
[ ] 钩子强度是否达到 medium 以上？
[ ] 本章是否有至少 1-2 个微兑现？
[ ] 是否有完整执行的爽点？
[ ] 微兑现质量是否达到中等以上？
[ ] 爽点强度是否与章节定位匹配？

满足 4-5 项 → A级以上
满足 2-3 项 → B级
满足 0-1 项 → C级，需要修改
```

## 六、约束分级体系 (Constraint Levels)

### Hard 约束 (必须执行)

| ID | 约束名称 | 定义 | 违反后果 |
|----|---------|------|----------|
| HARD-001 | 可读性底线 | 关键信息缺失导致看不懂 | 读者流失，必须修改 |
| HARD-002 | 承诺违背 | 上章钩子完全不兑现 | 信任崩塌，必须修改 |
| HARD-003 | 节奏灾难 | 连续3章无任何推进 | 读者厌弃，必须修改 |
| HARD-004 | 冲突真空 | 整章无问题/目标/代价 | 内容空洞，必须修改 |
| HARD-005 | 设定冲突 | 与已建立设定矛盾 | 体系崩塌，必须修改 |
| HARD-006 | 逻辑断裂 | 行为/因果不符合逻辑 | 可信度丧失，必须修改 |

### Soft 约束 (建议执行)

| ID | 约束名称 | 定义 | 违反后果 |
|----|---------|------|----------|
| SOFT-001 | 章末钩子不足 | 钩子强度 weak | 读者继续阅读意愿降低 |
| SOFT-002 | 微兑现缺失 | 本章无微兑现 | 读者满足感不足 |
| SOFT-003 | 情绪曲线平淡 | 全章情绪无起伏 | 读者注意力分散 |
| SOFT-004 | 模式重复疲劳 | 连续3章同类爽点 | 审美疲劳 |
| SOFT-005 | 信息过载 | 单章新信息过多 | 读者消化不良 |
| SOFT-006 | 对话功能性过强 | 对话只为推进剧情 | 角色扁平化 |

### Style 约束 (可选强化)

| ID | 约束名称 | 定义 | 违反后果 |
|----|---------|------|----------|
| STYLE-001 | AI痕迹明显 | 使用禁用词汇/模式 | 出戏感 |
| STYLE-002 | 节奏单一 | 句式/段落模式重复 | 阅读疲劳 |
| STYLE-003 | 过度解释 | 读者已理解仍解释 | 信任度下降 |

### 约束检查优先级

```markdown
Hard > Soft > Style

Hard 约束: 必须修复后才能继续
Soft 约束: 建议修复，可权衡
Style 约束: 视情况优化
```
