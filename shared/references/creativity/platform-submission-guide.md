## 一、番茄小说投稿规范

### 1.1 基本要求

| 项目 | 要求 |
|------|------|
| 最低字数 | 2万字起可申请签约 |
| 章节字数 | 建议 2500-4000 字/章 |
| 更新频率 | 日更 4000 字以上为佳 |
| 完结字数 | 男频 100-300 万字，女频 80-150 万字 |
| 首发要求 | 非独家首发可，独家首发权益更高 |

### 1.2 内容规范

**题材限制**：
- ❌ 禁止：涉政、涉黄、涉暴、迷信、邪教
- ❌ 禁止：现实人物映射、敏感历史事件
- ❌ 禁止：未成年人不良行为引导
- ⚠️ 限制：灵异恐怖（需有科学解释或正义战胜邪恶）
- ⚠️ 限制：官场题材（避免腐败描写正面化）

**女频特色要求**：
- 情感线必须有，但不过度描写亲密行为
- 男主不能有严重道德瑕疵（如家暴、出轨洗白）
- 重生复仇需有底线，避免极端暴力

**男频特色要求**：
- 后宫文限制严格，建议单女主或双女主
- 打脸要有逻辑，避免无脑爽
- 权谋需合理，避免过于阴暗

### 1.3 审核标准

| 审核项 | 通过标准 | 驳回常见原因 |
|--------|---------|-------------|
| 黄金三章 | 前3章有明确钩子 | 开篇平淡，无冲突 |
| 主角人设 | 性格鲜明，行为合理 | 人设崩塌，行为不合逻辑 |
| 金手指 | 有吸引力且合理 | 过强导致无悬念，或无意义 |
| 节奏把控 | 爽点密度 1.2-1.5/千字 | 节奏拖沓或过快 |
| 文笔质量 | 流畅通顺，无明显错字 | 语病多，AI痕迹重 |

### 1.4 签约标准

```yaml
番茄签约核心指标:
  追读率: >15%（优秀），>10%（合格）
  收藏率: >5%（优秀），>3%（合格）
  完读率: >30%（前10章）

加分项:
  - 开篇强钩子，前500字有冲突
  - 第1-3章节奏紧凑
  - 金手指设定新颖
  - 主角人设有记忆点

减分项:
  - 开篇慢热，前3章无冲突
  - AI 写作痕迹明显
  - 题材严重同质化
  - 人设老套无新意
```

### 1.5 番茄特色建议

**番茄读者偏好**：
- 女频强势，甜宠、重生复仇、古言热门
- 男频偏都市、系统流
- 节奏要求快，开篇即入题
- 爽点密集，情绪价值优先

**番茄爆款特征**：
```yaml
开篇模式:
  女频:
    - 重生复仇式：前世惨死 → 重生 → 开始反击
    - 甜宠式：初遇男主 → 误会/冲突 → 暧昧升温
    - 穿书式：穿成恶毒女配 → 改变命运

  男频:
    - 系统觉醒式：获得系统 → 第一次使用 → 震惊全场
    - 都市异能式：异能觉醒 → 第一次战斗 → 发现秘密
    - 重生逆袭式：前世失败 → 重生 → 改变命运
```

## 二、七猫小说投稿规范

### 2.1 基本要求

| 项目 | 要求 |
|------|------|
| 最低字数 | 1-2 万字可申请签约 |
| 章节字数 | 建议 3000-5000 字/章 |
| 更新频率 | 日更 4000 字以上 |
| 完结字数 | 男频 150-300 万字，女频 100-200 万字 |
| 首发要求 | 支持非独家 |

### 2.2 内容规范

**七猫特色**：
- 下沉市场为主，爽文导向
- 节奏最快，爽点密度要求最高
- 金手指必须明显，打脸要直接

**七猫红线**：
```yaml
禁止内容:
  - 黄赌毒正面描写
  - 政治敏感内容
  - 封建迷信宣扬
  - 未成年人犯罪引导
  - 现实人物影射

限制内容:
  - 后宫文（建议单女主）
  - 灵异题材（需有合理结局）
  - 官场权谋（避免过于阴暗）
```

### 2.3 审核标准

| 审核项 | 七猫标准 |
|--------|---------|
| 爽点密度 | 1.3-1.8/千字（最高要求） |
| 开篇节奏 | 第1章必须有冲突或危机 |
| 金手指 | 必须在3章内展现 |
| 打脸节奏 | 第3章前必须有第一次小打脸 |
| 文风 | 简单直白，避免晦涩 |

### 2.4 七猫爆款公式

```yaml
七猫成功公式:
  开篇: 危机/羞辱 → 金手指/机缘
  节奏: 每500字一个小爽点，每章一个大爽点
  人物: 主角必须果断，不圣母不纠结
  冲突: 密集且直接，小人要脸谱化
  打脸: 要狠要快，不留余地

热门标签:
  男频: 系统、重生、都市、无敌、签到
  女频: 重生复仇、甜宠、打脸、逆袭
```

## 三、起点中文网投稿规范

### 3.1 基本要求

| 项目 | 要求 |
|------|------|
| 最低字数 | 3-5 万字可申请签约 |
| 章节字数 | 建议 4000-6000 字/章 |
| 更新频率 | 日更 6000 字以上为佳 |
| 完结字数 | 男频 200-500 万字 |
| 首发要求 | 独家首发（可多站分发但起点首发） |

### 3.2 内容规范

**起点特色**：
- 核心读者群体，付费意愿高
- 题材创新重要，世界观要完整
- 允许慢热，但深度必须有
- 爽点密度可低于免费平台

**起点红线**：
```yaml
禁止内容:
  - 违法犯罪正面描写
  - 政治敏感内容
  - 黄赌毒宣扬
  - 封建迷信
  - 严重三观不正

起点特有要求:
  - 世界观逻辑自洽
  - 设定要有创新点
  - 人物智商在线
  - 避免无脑打脸
```

### 3.3 审核标准

| 审核项 | 起点标准 |
|--------|---------|
| 创新度 | 题材或设定需有创新点 |
| 世界观 | 需完整且有深度 |
| 节奏 | 可慢热，但要有钩子 |
| 爽点密度 | 0.6-1.0/千字（低于免费平台） |
| 文笔 | 要求较高，AI痕迹明显会被扣分 |

### 3.4 起点签约公式

```yaml
起点成功要素:
  题材创新: 新题材或老题材新写法
  世界观: 完整、自洽、有深度
  人物: 群像塑造能力，配角有灵魂
  节奏: 可慢热，但每章要有看点
  文笔: 流畅、有画面感、不俗套

起点读者偏好:
  - 玄幻/奇幻: 世界观是核心
  - 科幻: 设定严谨，逻辑自洽
  - 都市: 现实感与爽感平衡
  - 无限流: 副本设计有创意
```

## 四、晋江文学城投稿规范

### 4.1 基本要求

| 项目 | 要求 |
|------|------|
| 最低字数 | 3 万字可申请签约 |
| 章节字数 | 建议 3000-5000 字/章 |
| 更新频率 | 日更 3000-4000 字 |
| 完结字数 | 30-80 万字（偏短篇） |
| 首发要求 | 独家首发 |

### 4.2 内容规范

**晋江特色**：
- 女性读者为主，情感细腻
- 人设和情感线是核心
- 文笔要求最高
- OOC 零容忍

**晋江红线**：
```yaml
禁止内容:
  - 三观不正
  - 涉黄涉暴
  - 政治敏感
  - 现实人物影射

晋江特有要求:
  - 情感描写要有深度
  - 人设不能崩
  - 文笔要过关
  - 避免狗血俗套
```

### 4.3 审核标准

| 审核项 | 晋江标准 |
|--------|---------|
| 情感线 | 核心必须有，描写要细腻 |
| 人设 | 性格鲜明，行为逻辑自洽 |
| 文笔 | 要求最高，AI痕迹会被扣分 |
| OOC | 零容忍，人设崩=拒签 |

### 4.4 晋江签约公式

```yaml
晋江成功要素:
  情感: 情感线是核心，必须有
  人设: 男主/女主都要有记忆点
  文笔: 流畅、细腻、有感染力
  节奏: 情感节奏比剧情节奏重要
  创新: 反套路、新人设受欢迎

晋江读者偏好:
  - 言情: 情感深度是关键
  - 古言: 文风要古韵，人设要丰满
  - 现代: 现实感与浪漫平衡
  - 耽美: 人设和情感是核心
```

## 五、通用审核红线

### 5.1 绝对禁区

| 类型 | 具体内容 | 后果 |
|------|---------|------|
| 政治敏感 | 反动言论、政治隐喻、领导人影射 | 直接封禁 |
| 色情低俗 | 详细性描写、露骨情节 | 直接封禁 |
| 暴力血腥 | 过度暴力描写、血腥细节 | 封禁或要求修改 |
| 迷信邪教 | 宣扬迷信、邪教思想 | 直接封禁 |
| 未成年人保护 | 未成年人犯罪、不良行为引导 | 直接封禁 |

### 5.2 敏感词规避

```yaml
政治类:
  - 国家领导人姓名（避免影射）
  - 政府机构名称（虚构化处理）
  - 敏感历史事件（避开或虚构化）

色情类:
  - 露骨描写词汇（用暗示替代）
  - 具体性行为描写（用留白替代）
  - 未成年相关（绝对禁止）

暴力类:
  - 过度血腥描写（淡化处理）
  - 酷刑细节（避免详细描写）
  - 未成年人暴力（绝对禁止）

迷信类:
  - 算命、风水等（科学化或虚构化）
  - 鬼怪（要有合理解释或正义结局）
```

### 5.3 平台差异敏感度

| 内容类型 | 番茄 | 七猫 | 起点 | 晋江 |
|---------|------|------|------|------|
| 后宫文 | 限制 | 限制 | 较宽松 | 禁止 |
| 灵异恐怖 | 需解释 | 需解释 | 可接受 | 需解释 |
| 官场权谋 | 限制 | 限制 | 可接受 | 限制 |
| 暴力描写 | 严格 | 较宽松 | 较宽松 | 严格 |
| AI写作 | 开始检测 | 开始检测 | 检测 | 检测 |

## 六、投稿自查清单

### 6.1 投稿前必检

```yaml
内容合规:
  - [ ] 无政治敏感内容
  - [ ] 无色情低俗描写
  - [ ] 无过度暴力血腥
  - [ ] 无迷信邪教宣扬
  - [ ] 无未成年人不良引导

黄金三章:
  - [ ] 第1章有开篇钩子
  - [ ] 主角前300字出场
  - [ ] 金手指/机缘暗示
  - [ ] 第3章有第一次小高潮
  - [ ] 每章末有悬念钩子

平台匹配:
  - [ ] 爽点密度匹配平台要求
  - [ ] 章节字数匹配平台要求
  - [ ] 题材符合平台偏好
  - [ ] 更新频率能满足要求
```

### 6.2 签约自查

```yaml
硬性指标:
  - [ ] 字数达标
  - [ ] 更新稳定
  - [ ] 无违规内容

软性指标:
  - [ ] 开篇有吸引力
  - [ ] 人设有记忆点
  - [ ] 金手指有新意
  - [ ] 节奏合适
  - [ ] 文笔流畅
```
