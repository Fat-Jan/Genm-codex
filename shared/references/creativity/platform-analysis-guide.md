## 七猫小说

### 平台特点

- **商业模式**: 免费阅读 + 广告变现
- **读者画像**: 下沉市场、年轻读者、快节奏偏好
- **内容偏好**: 爽文为主，节奏快，爽点密集

### 分析重点

```yaml
关键指标:
  爽点密度: 1.3-1.8/千字（高于行业平均）
  章节字数: 3000-5000字（偏短）
  开篇节奏: 快节奏（第一章必有爽点）

题材偏好:
  男频:
    - 都市异能
    - 重生逆袭
    - 系统流
  女频:
    - 甜宠
    - 重生复仇
    - 穿书穿脑洞

流行标签:
  - 系统
  - 金手指
  - 重生
  - 打脸
  - 无敌

黄金三章模式:
  第一章: 开局即冲突 + 金手指觉醒
  第二章: 第一次反击 + 震惊全场
  第三章: 小胜 + 树立主要敌人
```

### 数据采集策略

- 搜索关键词: "七猫小说排行榜 免费 2026"
- 关注榜单: 热榜、新书榜、完结榜
- 免费章节: 前 10-20 章（大部分作品免费）

## 起点中文网

### 平台特点

- **商业模式**: 付费订阅（VIP 章节）
- **读者画像**: 核心读者、付费意愿高、口味刁钻
- **内容偏好**: 创新重要、世界观完整、慢热可接受

### 分析重点

```yaml
关键指标:
  爽点密度: 0.6-1.0/千字（低于免费平台）
  章节字数: 4000-6000字
  开篇节奏: 慢热可接受（但需要钩子）

题材偏好:
  男频主导:
    - 玄幻
    - 奇幻
    - 科幻
    - 都市
  特点: 题材创新重要

流行标签:
  - 系统流（但需要创新）
  - 无限流
  - 克苏鲁
  - 诡秘

黄金三章模式:
  第一章: 世界观展示 + 主角处境 + 隐藏钩子
  第二章: 金手指暗示 + 第一次小冲突
  第三章: 金手指确认 + 第一个小高潮

关键差异:
  - 读者更看重世界观和设定
  - 允许慢热但需要深度
  - 创新比套路更受欢迎
```

### 数据采集策略

- 搜索关键词: "起点中文网 月票榜 2026"
- 关注榜单: 月票榜、推荐榜、新人榜
- 免费章节: 通常前 5-10 章

## 晋江文学城

### 平台特点

- **商业模式**: 付费订阅
- **读者画像**: 女性读者为主、情感细腻、付费意愿高
- **内容偏好**: 情感驱动、人设重要、文笔要求高

### 分析重点

```yaml
关键指标:
  爽点密度: 0.8-1.2/千字（情感爽点为主）
  章节字数: 3000-5000字
  开篇节奏: 情感驱动（不强求快节奏）

题材偏好:
  女频核心:
    - 言情
    - 耽美
    - 古代言情
    - 现代言情
  特点: 情感深度重要

流行标签:
  - 甜宠
  - 虐恋
  - 古言
  - 穿越
  - 重生

黄金三章模式:
  第一章: 人物出场 + 情感张力
  第二章: 关系发展 + 第一次冲突/互动
  第三章: 情感转折 + 小高潮

关键差异:
  - 情感描写深度重要
  - 人设不能崩
  - 文笔要求高
  - 读者对OOC容忍度低
```

### 数据采集策略

- 搜索关键词: "晋江文学城 金榜 2026"
- 关注榜单: 金榜、月榜、半年榜
- 免费章节: 通常前 3-5 章

## 注意事项

1. **数据来源**: 仅分析公开的免费章节，不涉及付费内容
2. **时效性**: 市场趋势变化快，数据有效期 7-14 天
3. **平台差异**: 不同平台读者群体不同，注意区分应用场景
4. **人工判断**: AI 分析仅供参考，创作者需根据自身情况调整
