# Truth Files Spec

这份兼容文档用于承接旧引用。

在 `Genm-codex` 里，项目级 truth files / truth sheets 的权威入口主要是：

- [ancient-household-truth-sheet.md](../templates/project/ancient-household-truth-sheet.md)
- [ancient-office-truth-sheet.md](../templates/project/ancient-office-truth-sheet.md)
- [setting-gate-policy.json](../../docs/setting-gate-policy.json)
- [strong-quality-gate-policy.json](../../docs/strong-quality-gate-policy.json)

最小原则：

- 家族/宅门真值优先落在项目内 `设定集/家族/宅门真值表.md`
- 官制真值优先落在项目内 `设定集/官制/官职真值表.md`
- 写作和收口阶段的硬阻断，以 `setting gate` 与 `strong-quality-gate-policy` 为准
