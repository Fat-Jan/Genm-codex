## 核心规则速查

处理文本时，牢记这 5 条核心原则：

1. **删除填充短语** - 去除开场白和强调性拐杖词
2. **打破公式结构** - 避免二元对比、戏剧性分段、修辞性设置
3. **变化节奏** - 混合句子长度。两项优于三项。段落结尾要多样化
4. **信任读者** - 直接陈述事实，跳过软化、辩解和手把手引导
5. **删除金句** - 如果听起来像可引用的语句，重写它

## 一、个性与灵魂

避免 AI 模式只是工作的一半。无菌、没有声音的写作和机器生成的内容一样明显。好的写作背后有一个真实的人。

### 缺乏灵魂的写作迹象（即使技术上"干净"）：

- 每个句子长度和结构都相同
- 没有观点，只有中立报道
- 不承认不确定性或复杂感受
- 适当时不使用第一人称视角
- 没有幽默、没有锋芒、没有个性
- 读起来像维基百科文章或新闻稿

### 如何增加语调：

**有观点。** 不要只是报告事实——对它们做出反应。"我真的不知道该怎么看待这件事"比中立地列出利弊更有人味。

**变化节奏。** 短促有力的句子。然后是需要时间慢慢展开的长句。混合使用。

**承认复杂性。** 真实的人有复杂的感受。"这令人印象深刻但也有点不安"胜过"这令人印象深刻"。

**适当使用"我"。** 第一人称不是不专业——而是诚实。"我一直在思考……"或"让我困扰的是……"表明有真实的人在思考。

**允许一些混乱。** 完美的结构感觉像算法。跑题、题外话和半成型的想法是人性的体现。

**对感受要具体。** 不是"这令人担忧"，而是"凌晨三点没人看着的时候，智能体还在不停地运转，这让人不安"。

### 改写前（干净但无灵魂）：
> 实验产生了有趣的结果。智能体生成了 300 万行代码。一些开发者印象深刻，另一些则持怀疑态度。影响尚不明确。

### 改写后（鲜活）：
> 我真的不知道该怎么看待这件事。300 万行代码，在人类大概睡觉的时候生成的。开发社区有一半人疯了，另一半人在解释为什么这不算数。真相可能在无聊的中间某处——但我一直在想那些通宵工作的智能体。

## 二、内容模式（6种）

### 模式1：过度强调意义、遗产和更广泛的趋势

**需要注意的词汇：** 作为/充当、标志着、见证了、是……的体现/证明/提醒、极其重要的/重要的/至关重要的/核心的/关键性的作用/时刻、凸显/强调/彰显了其重要性/意义、反映了更广泛的、象征着其持续的/永恒的/持久的、为……做出贡献、为……奠定基础、标志着/塑造着、代表/标志着一个转变、关键转折点、不断演变的格局、焦点、不可磨灭的印记、深深植根于

**问题：** LLM 写作通过添加关于任意方面如何代表或促进更广泛主题的陈述来夸大重要性。

**改写前：**
> 加泰罗尼亚统计局于 1989 年正式成立，标志着西班牙区域统计演变史上的关键时刻。这一举措是西班牙全国范围内更广泛运动的一部分，旨在分散行政职能并加强区域治理。

**改写后：**
> 加泰罗尼亚统计局成立于 1989 年，负责独立于西班牙国家统计局收集和发布区域统计数据。

**网文示例：**
```markdown
❌ 林天突破筑基期，标志着他修仙道路上的关键转折点，为他日后的辉煌成就奠定了坚实基础。
✅ 林天突破了。筑基期。
   他看了看自己的手，力量感涌动。
```

### 模式2：过度强调知名度和媒体报道

**需要注意的词汇：** 独立报道、地方/区域/国家媒体、由知名专家撰写、活跃的社交媒体账号

**问题：** LLM 反复强调知名度主张，通常列出来源而不提供上下文。

**改写前：**
> 她的观点被《纽约时报》、BBC、《金融时报》和《印度教徒报》引用。她在社交媒体上拥有活跃的存在，拥有超过 50 万粉丝。

**改写后：**
> 在 2024 年《纽约时报》的采访中，她认为 AI 监管应该关注结果而不是方法。

**网文示例：**
```markdown
❌ 林天的名声传遍了整个大陆，东域、西漠、南疆、北原，无人不知无人不晓。
✅ 林天的名字，现在在茶馆里经常能听到。
```

### 模式3：以 -ing 结尾的肤浅分析

**需要注意的词汇：** 突出/强调/彰显……、确保……、反映/象征……、为……做出贡献、培养/促进……、涵盖……、展示……

**问题：** AI 聊天机器人在句子末尾添加现在分词（"-ing"）短语来增加虚假深度。

**改写前：**
> 寺庙的蓝色、绿色和金色色调与该地区的自然美景产生共鸣，象征着德克萨斯州的蓝帽花、墨西哥湾和多样化的德克萨斯州景观，反映了社区与土地的深厚联系。

**改写后：**
> 寺庙使用蓝色、绿色和金色。建筑师表示这些颜色是为了呼应当地的蓝帽花和墨西哥湾海岸。

**网文示例：**
```markdown
❌ 这一剑蕴含着他对剑道的全部理解，展示了他多年苦修的成果，彰显了宗师的威严。
✅ 这一剑，是他十年来每天挥剑三千次的结果。
```

### 模式4：宣传和广告式语言

**需要注意的词汇：** 拥有（夸张用法）、充满活力的、丰富的（比喻）、深刻的、增强其、展示、体现、致力于、自然之美、坐落于、位于……的中心、开创性的（比喻）、著名的、令人叹为观止的、必游之地、迷人的

**问题：** LLM 在保持中立语气方面存在严重问题，尤其是对于"文化遗产"话题。倾向使用夸张的宣传性语言。

**改写前：**
> 坐落在埃塞俄比亚贡德尔地区令人叹为观止的区域内，Alamata Raya Kobo 是一座充满活力的城镇，拥有丰富的文化遗产和迷人的自然美景。

**改写后：**
> Alamata Raya Kobo 是埃塞俄比亚贡德尔地区的一座城镇，以其每周集市和 18 世纪教堂而闻名。

**网文示例：**
```markdown
❌ 坐落在云雾缭绕的天山之巅，这座宏伟的宫殿是修仙界的圣地，拥有悠久的历史和令人叹为观止的建筑风格。
✅ 天山顶上有一座宫。
   修仙者叫它天宫。
   没几个人上去过。
```

### 模式5：模糊归因和含糊措辞

**需要注意的词汇：** 行业报告显示、观察者指出、专家认为、一些批评者认为、多个来源/出版物（实际引用却很少）

**问题：** AI 聊天机器人将观点归因于模糊的权威而不提供具体来源。

**改写前：**
> 由于其独特的特征，浩来河引起了研究人员和保护主义者的兴趣。专家认为它在区域生态系统中发挥着至关重要的作用。

**改写后：**
> 根据中国科学院 2019 年的调查，浩来河支持多种特有鱼类。

**网文示例：**
```markdown
❌ 许多人认为这门功法太过危险，修炼者往往会走火入魔。
✅ 这功法，五十年里练死过三个人。
```

---

### 模式6：提纲式的"挑战与未来展望"部分

**需要注意的词汇：** 尽管其……面临若干挑战……、尽管存在这些挑战、挑战与遗产、未来展望

**问题：** 许多 LLM 生成的文章包含公式化的"挑战"部分。

**改写前：**
> 尽管工业繁荣，Korattur 面临着城市地区典型的挑战，包括交通拥堵和水资源短缺。尽管存在这些挑战，凭借其战略位置和正在进行的举措，Korattur 继续蓬勃发展，成为钦奈增长不可或缺的一部分。

**改写后：**
> 2015 年三个新 IT 园区开业后，交通拥堵加剧。市政公司于 2022 年启动了雨水排水项目，以解决反复发生的洪水。

**网文示例：**
```markdown
❌ 尽管林天已经突破到金丹期，但他仍然面临着诸多挑战，包括资源的匮乏和敌人的觊觎。尽管如此，他坚信自己能够在修仙之路上走得更远。
✅ 金丹期了。
   但林天手里只剩三块灵石。
   还有三个人在外面等着杀他。
```

## 三、语言和语法模式（6种）

### 模式7：过度使用的"AI 词汇"

**高频 AI 词汇：** 此外、与……保持一致、至关重要、深入探讨、强调、持久的、增强、培养、获得、突出（动词）、相互作用、复杂/复杂性、关键（形容词）、格局（抽象名词）、关键性的、展示、织锦（抽象名词）、证明、强调（动词）、宝贵的、充满活力的

**问题：** 这些词在 2023 年后的文本中出现频率要高得多。它们经常共同出现。

**改写前：**
> 此外，索马里菜肴的一个显著特征是加入骆驼肉。意大利殖民影响的持久证明是当地烹饪格局中广泛采用意大利面，展示了这些菜肴如何融入传统饮食。

**改写后：**
> 索马里菜肴还包括骆驼肉，被认为是一种美味。在意大利殖民期间引入的意大利面菜肴仍然很常见，尤其是在南部。

**网文扩展禁用词清单：**

| 类型 | AI高频词 | 替代方案 |
|------|---------|---------|
| 程度副词 | 淡淡的、微微、不禁、竟、顿时 | 删除或用动作替代 |
| 情感副词 | 欣喜地、悲伤地、愤怒地 | 用行为/生理反应替代 |
| 语气副词 | 显然、无疑、确实、似乎 | 删除或用事实说明 |
| 时间副词 | 突然、猛然、霎时 | 用事件本身制造冲击 |
| AI学术词 | 此外、至关重要、深入探讨、强调 | 删除或换口语表达 |
| AI抽象词 | 格局、织锦、复杂性、关键性 | 删除或写具体内容 |

### 模式8：避免使用"是"（系动词回避）

**需要注意的词汇：** 作为/代表/标志着/充当 [一个]、拥有/设有/提供 [一个]

**问题：** LLM 用复杂的结构替代简单的系动词。

**改写前：**
> Gallery 825 作为 LAAA 的当代艺术展览空间。画廊设有四个独立空间，拥有超过 3000 平方英尺。

**改写后：**
> Gallery 825 是 LAAA 的当代艺术展览空间。画廊有四个房间，总面积 3000 平方英尺。

**网文示例：**
```markdown
❌ 林天作为林家唯一的继承人，拥有着无与伦比的修炼天赋。
✅ 林天是林家的独子。
   天赋很好。
```

### 模式9：否定式排比

**问题：** "不仅……而且……"或"这不仅仅是关于……，而是……"等结构被过度使用。

**改写前：**
> 这不仅仅是节拍在人声下流动；它是攻击性和氛围的一部分。这不仅仅是一首歌，而是一种声明。

**改写后：**
> 沉重的节拍增加了攻击性的基调。

**网文示例：**
```markdown
❌ 这不仅仅是一把剑，而是他身份的象征，更是他责任的体现。
✅ 这把剑跟着他十年了。
```

---

### 模式10：三段式法则过度使用

**问题：** LLM 强行将想法分成三组以显得全面。

**改写前：**
> 活动包括主题演讲、小组讨论和社交机会。与会者可以期待创新、灵感和行业洞察。

**改写后：**
> 活动包括演讲和小组讨论。会议之间还有非正式社交的时间。

**网文示例：**
```markdown
❌ 他有三个选择：第一，继续修炼；第二，寻找外援；第三，放弃复仇。
✅ 他可以继续练，或者找帮手。
   放弃？没想过。
```

---

### 模式11：刻意换词（同义词循环）

**问题：** AI 有重复惩罚代码，导致过度使用同义词替换。

**改写前：**
> 主人公面临许多挑战。主要角色必须克服障碍。中心人物最终获得胜利。英雄回到家中。

**改写后：**
> 主人公面临许多挑战，但最终获得胜利并回到家中。

**网文示例：**
```markdown
❌ 林天站起身来，青年走出房门，这位修士望向远方。
✅ 林天站起来，走出门，看了一眼天边。
```

---

### 模式12：虚假范围

**问题：** LLM 使用"从 X 到 Y"的结构，但 X 和 Y 并不在有意义的尺度上。

**改写前：**
> 我们穿越宇宙的旅程将我们从大爆炸的奇点带到宏伟的宇宙网，从恒星的诞生和死亡到暗物质的神秘舞蹈。

**改写后：**
> 这本书涵盖了大爆炸、恒星形成和当前关于暗物质的理论。

**网文示例：**
```markdown
❌ 从筑基期的渺小到金丹期的强大，从炼气期的懵懂到元婴期的通透，他走过了一条漫长的道路。
✅ 他从炼气期练到现在。
   十五年了。
```

## 四、风格模式（6种）

### 模式13：破折号过度使用

**问题：** LLM 使用破折号（—）比人类更频繁，模仿"有力"的销售文案。

**改写前：**
> 这个术语主要由荷兰机构推广——而不是由人民自己。你不会说"荷兰，欧洲"作为地址——但这种错误标记仍在继续——即使在官方文件中。

**改写后：**
> 这个术语主要由荷兰机构推广，而不是由人民自己。你不会说"荷兰，欧洲"作为地址，但这种错误标记在官方文件中仍在继续。

**网文示例：**
```markdown
❌ 林天握剑——那把跟了他十年的剑——眼中闪过一丝决然——今天，不是你死就是我活。
✅ 林天握紧剑。
   十年了。
   今天，不是你死就是我活。
```

### 模式14：粗体过度使用

**问题：** AI 聊天机器人机械地用粗体强调短语。

**改写前：**
> 它融合了 **OKR（目标和关键结果）**、**KPI（关键绩效指标）** 和视觉战略工具，如 **商业模式画布（BMC）** 和 **平衡计分卡（BSC）**。

**改写后：**
> 它融合了 OKR、KPI 和视觉战略工具，如商业模式画布和平衡计分卡。

---

### 模式15：内联标题垂直列表

**问题：** AI 输出列表，其中项目以粗体标题开头，后跟冒号。

**改写前：**
> - **用户体验：** 用户体验通过新界面得到显著改善。
> - **性能：** 性能通过优化算法得到增强。
> - **安全性：** 安全性通过端到端加密得到加强。

**改写后：**
> 更新改进了界面，通过优化算法加快了加载时间，并添加了端到端加密。

---

### 模式16：标题中的标题大写

**问题：** AI 聊天机器人将标题中的所有主要单词大写。

**改写前：**
> ## 战略谈判与全球伙伴关系

**改写后：**
> ## 战略谈判与全球伙伴关系

**注：** 中文标题通常不涉及大小写问题，此模式在中文中不太适用。

---

### 模式17：表情符号

**问题：** AI 聊天机器人经常用表情符号装饰标题或项目符号。

**改写前：**
> 🚀 **启动阶段：** 产品在第三季度发布
> 💡 **关键洞察：** 用户更喜欢简单
> ✅ **下一步：** 安排后续会议

**改写后：**
> 产品在第三季度发布。用户研究显示更喜欢简单。下一步：安排后续会议。

---

### 模式18：弯引号

**问题：** ChatGPT 使用弯引号（""）而不是直引号（""）。

**改写前：**
> 他说"项目进展顺利"，但其他人不同意。

**改写后：**
> 他说"项目进展顺利"，但其他人不同意。

**注：** 中文通常使用中文引号（「」或""），此模式在中文中表现为英文引号的使用。

## 五、交流模式（6种）

### 模式19：协作交流痕迹

**需要注意的词汇：** 希望这对您有帮助、当然！、一定！、您说得完全正确！、您想要……、请告诉我、这是一个……

**问题：** 作为聊天机器人对话的文本被粘贴为内容。

**改写前：**
> 这是法国大革命的概述。希望这对您有帮助！如果您想让我扩展任何部分，请告诉我。

**改写后：**
> 法国大革命始于 1789 年，当时财政危机和粮食短缺导致了广泛的动荡。

---

### 模式20：知识截止日期免责声明

**需要注意的词汇：** 截至 [日期]、根据我最后的训练更新、虽然具体细节有限/稀缺……、基于可用信息……

**问题：** 关于信息不完整的 AI 免责声明留在文本中。

**改写前：**
> 虽然关于公司成立的具体细节在现成资料中没有广泛记录，但它似乎是在 20 世纪 90 年代的某个时候成立的。

**改写后：**
> 根据注册文件，该公司成立于 1994 年。

### 模式21：谄媚/卑躬屈膝的语气

**问题：** 过于积极、讨好的语言。

**改写前：**
> 好问题！您说得完全正确，这是一个复杂的话题。关于经济因素，这是一个很好的观点。

**改写后：**

## 六、网文特有的AI味道

### 问题1：修仙/玄幻场景过度解释

```markdown
❌ 这是一种极为罕见的天地灵火，威力巨大，能够焚烧万物，炼丹师梦寐以求。
✅ 这是天地灵火。
   炼丹师见到它，眼睛会发绿。
```

### 问题2：系统文过度说明

```markdown
❌ 系统检测到宿主当前实力不足，建议优先提升境界，以应对即将到来的危机。
✅ 【系统提示】
   当前实力：不足
   建议：提升境界
   倒计时：3天
```

### 问题3：都市文过度书面

```markdown
❌ 经过深思熟虑，他决定先调查清楚对方的背景，再做进一步的打算。
✅ 他想了想。
   先查底，再动手。
```

### 问题4：形容词/副词滥用（网文版）

**AI 高频词清单（禁用或减用）**：

| 类型 | 高频词 | 替代方案 |
|------|--------|---------|
| 程度副词 | 淡淡的、微微、不禁、竟、顿时 | 删除或用动作替代 |
| 情感副词 | 欣喜地、悲伤地、愤怒地 | 用行为/生理反应替代 |
| 语气副词 | 显然、无疑、确实、似乎 | 删除或用事实说明 |
| 时间副词 | 突然、猛然、霎时 | 用事件本身制造冲击 |

**对比示例**：
```markdown
❌ 他不禁微微皱眉，眼中闪过一丝淡淡的忧伤。
✅ 他皱眉。手指无意识地敲着桌面。
```

### 问题5：说明代替展示（Tell vs Show）

**核心原则**：不要说情绪，要写行为、生理反应、环境变化。

| 情绪 | ❌ 说明写法 | ✅ 展示写法 |
|------|-----------|-----------|
| 生气 | 他很生气 | 他把茶杯重重放下，茶水溅了一桌 |
| 紧张 | 她很紧张 | 她攥紧衣角，指节发白 |
| 喜欢 | 他对她有好感 | 他总是"恰好"路过她常去的地方 |
| 恐惧 | 他感到恐惧 | 他后背渗出冷汗，腿有些发软 |
| 得意 | 他很得意 | 他嘴角压不住的笑意 |

### 问题6：句式单一

**AI 常见句式模板**：
```markdown
❌ 主语 + 形容词/副词 + 动词 + 补语
   "他微微一笑，淡淡地说道。"
   "她不禁红了脸，低下了头。"
```

**打破方法**：
- 短句穿插（2-5字）
- 长句拆分
- 倒装句
- 省略主语
- 对话与动作穿插

**示例**：
```markdown
❌ 他看着远处的山峦，心中不禁涌起一股复杂的情绪，既有对未来的期待，也有对过往的怀念。
✅ 远处的山峦在夕阳下泛着金光。他站了很久。往事涌上来，又被他压下去。
```

### 问题7：情感直白无层次

**AI 典型问题**：情绪直接写出来，没有铺垫、没有矛盾、没有留白。

**对比**：
```markdown
❌ 她很感动，眼泪止不住地流下来。
✅ 她张了张嘴，想说什么，最后只"嗯"了一声。转身时，她用袖子快速擦了一下眼睛。
```

## 七、改写技巧清单

### 技巧1：删副词，用动作

```markdown
❌ 他淡淡地笑了。
✅ 他嘴角扬了扬。/ 他笑了。/ 他"呵"了一声。

❌ 她不禁叹了口气。
✅ 她叹气。/ 她长出一口气。

❌ 他猛然站起身来。
✅ 他站起来。椅子向后倒去。
```

### 技巧2：删形容词，用细节

```markdown
❌ 这是一个阴森恐怖的山洞。
✅ 山洞里滴水声回响。空气湿冷，带着腐叶的气味。

❌ 她很美，气质高雅。
✅ 她进来时，原本喧闹的大厅静了一瞬。

❌ 这是一把锋利的剑。
✅ 剑出鞘的瞬间，空气里多了一道寒光。
```

### 技巧3：拆长句，加断点

```markdown
❌ 林天看着眼前这个曾经背叛过自己的人，心中涌起复杂的情绪，既有愤怒也有释然，最终还是叹了口气转身离去。
✅ 林天看着眼前这个人。
   曾经的兄弟。曾经的叛徒。
   他张口想说什么，最后只是摇摇头，转身走了。
```

### 技巧4：情绪外化

| 内心情绪 | 外化方式 | 示例 |
|---------|---------|------|
| 焦虑 | 小动作 | 指尖敲桌面、来回踱步、捏东西 |
| 愤怒 | 身体反应 | 太阳穴跳动、拳头紧握、呼吸加重 |
| 喜欢 | 行为变化 | 总是看对方、话变多/变少、靠近 |
| 厌恶 | 身体回避 | 后退半步、皱眉、移开视线 |
| 紧张 | 生理反应 | 手心出汗、声音干涩、僵硬 |

### 技巧5：对话去书面化

```markdown
❌ "我认为这件事情有欠妥当，需要重新考虑。"
✅ "这事不对，再想想。"

❌ "我很高兴能够再次见到你。"
✅ "你怎么来了？" / "你来了。"

❌ "请你不要担心，我会保护你的安全。"
✅ "别怕，有我在。"
```

### 技巧6：留白与克制

```markdown
❌ 他爱她入骨，愿意为她付出一切。
✅ 他看着她离开的背影，很久没有动。

❌ 她心如刀绞，痛苦万分。
✅ 她没说话。筷子夹起一粒米，又放下。
```

## 八、快速检查清单

- ✓ **连续三个句子长度相同？** 打断其中一个

## 八、平台 AI 检测规避策略

> ⚠️ 重要提示：2024-2025年，番茄、七猫、起点、晋江等平台开始部署 AI 检测系统。本章节提供规避策略。

### 8.1 平台 AI 检测原理

```yaml
检测维度:
  语言模式:
    - AI 高频词检测（淡淡的、微微、不禁、顿时）
    - 句式模式检测（不仅……而且……）
    - 三段式法则检测

  结构模式:
    - 段落长度均匀度
    - 句子长度变化
    - 信息密度分布

  情感模式:
    - 情感表达直白度
    - 情感词汇密度
    - 情感转折规律性

  创意模式:
    - 与训练数据相似度
    - 情节套路重复度
    - 人物设定独特性
```

### 8.2 高危特征清单

**最容易被检测的特征：**

| 特征 | 检测风险 | 规避方法 |
|------|---------|---------|
| "淡淡的/微微/不禁" | ⚠️⚠️⚠️ 高 | 删除或替换 |
| "顿时/猛然/竟然" | ⚠️⚠️⚠️ 高 | 删除或替换 |
| 三段式列举 | ⚠️⚠️⚠️ 高 | 改为两项或四项 |
| "不仅……而且……" | ⚠️⚠️ 高 | 删除一半 |
| 段落长度均匀 | ⚠️⚠️ 高 | 故意打破规律 |
| 情感直白描写 | ⚠️⚠️ 高 | 用行为暗示 |
| "作为……的证明" | ⚠️⚠️ 高 | 删除 |

### 8.3 规避技巧

**技巧1：句式随机化**

```yaml
问题模式:
  - 每句都是"主语+谓语+宾语"
  - 每段都是3-5句
  - 每句都是15-20字

解决方法:
  - 故意写超短句："他停下了。"
  - 故意写长句，中间加转折
  - 偶尔用倒装句
  - 偶尔用省略句
```

**技巧2：口语化处理**

```yaml
书面语 → 口语化:
  "他感到十分愤怒" → "他气得不行"
  "她陷入了深深的思考" → "她在那儿愣了半天"
  "这个决定非常重要" → "这事儿挺关键的"

口语化特征:
  - 用"那"、"这"代替具体名词
  - 用"了"、"吧"、"呢"等语气词
  - 适当用方言特色词
```

**技巧3：故意引入"瑕疵"**

```yaml
完美文本特征（容易被判定为AI）:
  - 无错别字
  - 无语病
  - 逻辑完美
  - 结构工整

人为特征（更像人写的）:
  - 偶尔的口语化"错误"
  - 信息的不完整传递
  - 话题的轻微跑偏
  - 情绪的不稳定表达
```

### 8.4 平台检测差异

| 平台 | 检测严格度 | 重点检测 | 应对策略 |
|------|-----------|---------|---------|
| 番茄 | ⚠️⚠️⚠️ 高 | 语言模式 | 重点去AI高频词，口语化 |
| 七猫 | ⚠️⚠️ 中高 | 结构模式 | 句式随机化，打破规律 |
| 起点 | ⚠️⚠️ 中高 | 创意模式 | 确保设定新颖，非套路 |
| 晋江 | ⚠️⚠️⚠️ 高 | 情感模式 | 情感细腻真实，非直白 |

### 8.5 检测后修正流程

```yaml
第一步：AI高频词清理
  - 搜索"淡淡的"、"微微"、"不禁"、"顿时"、"猛然"、"竟然"
  - 全部替换或删除

第二步：句式模式打破
  - 检查是否有"不仅……而且……"
  - 检查是否有三段式列举
  - 检查段落长度是否均匀

第三步：口语化改造
  - 长句拆短句
  - 书面语改口语
  - 加入语气词

第四步：情感真实化
  - 直白情感改为行为暗示
  - 加入矛盾情绪
  - 加入身体反应

第五步：最终检测
  - 通读是否自然
  - 是否有"机器感"
  - 是否有个人风格
```

### 8.6 2026年朱雀检测应对策略

> 2026年更新：朱雀检测系统已成为主流平台的核心检测工具，其算法基于句子长度变化、困惑度、突发性三大维度。

#### 8.6.1 朱雀检测核心指标

```yaml
判定标准:
  句子长度标准差:
    - 人类文本: 标准差 > 1.5（句子长短变化大）
    - AI文本: 标准差 < 1.5（句子均匀、可预测）
    - 应对: 刻意制造句子长度差异

  困惑度(Perplexity):
    - 人类文本: 困惑度较高（语言不可预测）
    - AI文本: 困惑度过低（语言过于平滑、可预测）
    - 应对: 使用意外词汇、打破常见搭配

  突发性(Burstiness):
    - 人类文本: 突发性高（句子风格跳跃）
    - AI文本: 突发性低（风格统一、节奏平稳）
    - 应对: 混合不同风格、句式、长度
```

#### 8.6.2 句子长度变化率要求

```yaml
黄金法则:
  - 每段至少包含: 1个超短句（< 5字）、1个长句（> 30字）
  - 连续3句长度相近: 必须打断（差值 < 3字视为相近）
  - 段落结尾: 避免3个以上段落以相同长度结尾

长度变化示例:
  ❌ AI模式:
    "他站起身来，走到窗边，看着外面的景色。（15字）"
    "她低下头去，沉默了片刻，不知在想什么。（15字）"
    "林天握紧了剑，目光变得锐利起来。（14字）"

  ✅ 人类模式:
    "他站起身。走到窗边。"
    "外面，天快黑了。他看着看着，忽然想起很多年前的一件事，那时候他还年轻，什么都不懂，总觉得世界很简单。"
    "林天握紧了剑。"
```

#### 8.6.3 突发性(Burstiness)提升方法

```yaml
方法1: 风格跳跃
  - 在正式叙述中插入口语化短句
  - 在长句中突然切换到极短句
  - 在描写中插入内心独白

方法2: 节奏打断
  - 用破折号、省略号制造停顿（但不要过度）
  - 用对话打断叙述流
  - 用动作打断心理描写

方法3: 信息密度变化
  - 一句信息密集（多细节）
  - 一句信息稀疏（留白）
  - 形成呼吸感

示例:
  ❌ AI平滑风格:
    "林天走进房间，环顾四周，发现这里布置得很简单，一张床，一张桌子，一把椅子。"

  ✅ 人类突发风格:
    "林天推开门。"
    "房间——怎么说呢——空得有点过分了。一张床，一张桌，一把椅子。就这些。"
    "他愣了一秒。"
    "像是有人提前知道他要来，把不需要的东西都搬走了。"
```

#### 8.6.4 困惑度(Perplexity)优化技巧

```yaml
技巧1: 打破常见搭配
  ❌ AI预测: "他微微一笑，眼中闪过一丝复杂的情绪"
  ✅ 意外表达: "他嘴角动了动，像是想笑，但最后只发出一个单音节的'呵'"

技巧2: 使用具体替代抽象
  ❌ AI预测: "她感到很开心"
  ✅ 具体表达: "她哼起了那首老歌，调子跑得有点远"

技巧3: 引入随机性
  - 加入看似无关但有趣的细节
  - 加入方言或个人习惯用语
  - 加入时代特色或个人记忆

技巧4: 避免"最可能的下一个词"
  - AI倾向于选择统计上最常见的词
  - 人会有意选择不那么常见的词
  - 例: AI说"他的心情很复杂" → 人说"他心里五味杂陈，像打翻了调味盒"
```

#### 8.6.5 高频词替换清单（朱雀检测重点词）

```markdown
【高危词 - 必须处理】
淡淡的 → 删除或具体化
  例: "她淡淡地笑了" → "她笑了笑" 或 "她嘴角扬了一下"

微微 → 删除或具体描写
  例: "他微微皱眉" → "他皱眉" 或 "他眉头皱了一下"

不禁 → 删除，直接描写动作
  例: "他不禁叹了口气" → "他叹了口气"

顿时 → 删除，直接描写结果
  例: "他顿时愣住了" → "他愣住了" 或 "他整个人僵在那儿"

【中危词 - 建议处理】
猛然 → 用动作本身制造冲击
竟然 → 删除，或用事实说明意外
霎时 → 删除，用短句替代
忽然 → 偶尔可用，但不要连续使用

【低危词 - 适量使用】
似乎 → 口语化可保留
仿佛 → 文学性强，可保留
好像 → 口语化，可保留
```markdown

#### 8.6.6 朱雀检测自检清单

```yaml
提交前必查:
  □ 句子长度标准差是否 > 1.5？
    - 计算方法: 统计10个连续句子的字数，计算标准差
    - 快速判断: 最长句与最短句差距是否 > 15字？

  □ 是否有连续3个相似长度的句子？
    - 相似定义: 字数差 < 3
    - 如有，必须打断其中一句

  □ 是否有"淡淡/微微/不禁/顿时"高危词？
    - 使用搜索功能逐一检查
    - 至少替换或删除 90%

  □ 困惑度是否足够？
    - 是否有意外的表达方式？
    - 是否有打破常见搭配的地方？

  □ 突发性是否足够？
    - 是否有风格跳跃？
    - 是否有节奏变化？
    - 是否有信息密度变化？
```
