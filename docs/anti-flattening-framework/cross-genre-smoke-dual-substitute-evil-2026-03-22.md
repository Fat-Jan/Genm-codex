## 与宫斗样本的差异

宫斗样本验证的是：

- 家门秩序
- 族面 / 切割 / 钥匙线
- 主角如何不吞掉家族群像

本样本验证的是：

- 双核心女性关系
- 替身 / 白月光 / 女配标签的反向夺权
- 机构角色与外部执行人的去功能化

这说明反脸谱化体系不是某个流派的局部补丁，而是真能跨题材迁移。

## 结论

本轮交叉验证足以确认：

1. 反脸谱化体系在 `双女主替身白月光` 路线上同样可用
2. 它能识别并约束与宫斗完全不同的风险类型
3. 同一套框架可以得到不同的工作流结果：
   - 宫斗样本：`review -> fix -> 再 review`
   - 双女主替身样本：`write -> review 直接通过`

这正是“交叉验证”真正有价值的地方：

- 不是要求所有流派都产生同一种问题
- 而是要求同一套框架能在不同流派里给出有区分度、但仍稳定的判断
