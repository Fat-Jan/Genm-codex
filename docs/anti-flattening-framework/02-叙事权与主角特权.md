# 叙事权与主角特权

## 目的

识别“只有主角像人”的结构性来源，并为大纲与评审提供纠偏动作。

## 五类主角特权

1. **解释权**
   - 只有主角理解世界，其他人只是在被解释。
2. **命名权**
   - “炮灰、恶毒女配、工具人、反派、舔狗”都由主角命名并冻结。
3. **行动发起权**
   - 重大事件必须等主角触发才开始。
4. **道德裁判权**
   - 谁对谁错、谁值得原谅，默认由主角视角盖章。
5. **后果豁免权**
   - 主角改了别人的命运，但世界不追债、不反咬、不留残账。

## 可接受的主角中心

主角中心不是问题，问题是“主角垄断世界”。

可接受状态：

- 主角是最高曝光角色
- 主角拥有更强的信息密度
- 主角主导部分关键节点

不可接受状态：

- 其他角色没有局部解释权
- 其他阵营没有自发行动链
- 所有人只围绕主角情绪转

## 失衡信号

- 配角只有在主角提问时才有想法
- 反对主角的人总是又蠢又急
- 主角的越界干预很少付账
- 别人的人生转折只是主角的成就素材

## 纠偏动作

### 大纲侧

- 给关键配角一段主角不在场的行动链
- 给关键阵营保留局部解释权
- 给至少一个角色一次“主角不愿承认但他说得对”的场面
- 让主角的一次成功显式换来后续成本

### 评审侧

- 检查章内是否只有主角在做选择
- 检查对立角色是否被削弱成 strawman
- 检查主角的成功是否总被系统、设定或作者托底

## 对 `novel-outline`

至少输出：

- 主角不能直接拿走什么
- 哪些角色拥有局部叙事主权
- 哪些事件不是主角触发也会发生

## 对 `novel-review`

至少检查：

- 主角是否吃掉了他人的解释权
- 主要对立面是否被脸谱化成“主角正确性的证明器”
- 本章是否存在“主角特权导致的零代价推进”
