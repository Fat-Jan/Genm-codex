# Genm-codex v0.1.0

首个可用版本，完成 `Genm` 到 Codex 原生 Skills 的第一阶段迁移，并拿到最小闭环 E2E 通过证据。

## 包含内容

- 6 个核心 Skill：
  - `genm-novel-init`
  - `genm-novel-outline`
  - `genm-novel-write`
  - `genm-novel-review`
  - `genm-novel-rewrite`
  - `genm-novel-export`
- 完整共享资产层：
  - `shared/profiles/`
  - `shared/references/`
  - `shared/templates/`
- 迁移基础设施：
  - `scripts/install-skills.sh`
  - `scripts/sync-shared-from-genm.sh`
  - `scripts/validate-migration.sh`

## 验证结果

最小闭环已真实通过：

`init → outline → write → review → rewrite → export`

验证样本：

- `e2e-novel/.mighty/state.json`
- `e2e-novel/大纲/总纲.md`
- `e2e-novel/大纲/章纲/第001章.md` ~ `第010章.md`
- `e2e-novel/chapters/第001章.md`
- `e2e-novel/exports/第001章.txt`

## 当前定位

`Genm-codex` 当前是 Codex 原生技能工作区，作为 `Genm` 的并行演进仓库存在：

- `Genm` 保留为源资产仓库
- `Genm-codex` 负责 Codex-native Skills 层
