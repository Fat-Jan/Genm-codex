# Genm-codex v0.12.0

第八阶段版本，完成实验性 `novel-scan` report-only 落地，以及 shared 同步可观测增强。

## 新增与增强

### `novel-scan`

- 新增 report-only 实验版
- 支持产出 `.mighty/market-data.json`
- 保留：
  - source plan
  - confidence
  - gaps
  - apply recommendations

### shared 同步增强

- `sync-shared-from-genm.sh --report-json`
- `sync-shared-from-genm.sh --domain <...>`
- shared 依赖映射文档补齐

## 说明

- 这仍然不是 `novel-scan` 的最终生产版
- 这也不是 shared selective-sync 的真正切换版
- `v0.12.0` 的意义在于：
  - 把实验实现和治理增强都推进到一个可追踪的版本检查点
