# Genm-codex v0.15.0

## Summary

`v0.15.0` marks the Phase 11 checkpoint:

- `v1.0.0` boundary definition
- default workflow convergence

## Highlights

- `docs/00-当前有效/start-here.md`
- `docs/00-当前有效/skill-usage.md`

This release is less about adding more capability and more about product clarity.

By this point, `Genm-codex` is no longer just a migration target or a pile of skills. It now has:

- a default workflow
- an experimental boundary
- a clearer path toward `v1.0.0`
