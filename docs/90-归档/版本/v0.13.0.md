# Genm-codex v0.13.0

## Summary

`v0.13.0` marks the completion of Phase 9:

- packaging generation layer
- first practical quality-loop integration round

## Highlights

### New skill

- `novel-package`

It provides first-version support for:

- title candidates
- synopsis variants
- naming proposals
- opening-hook packaging
- full packaging bundles

### Quality-loop integration

The following skills now have clearer integration behavior:

- `novel-write`
- `novel-review`
- `novel-polish`
- `novel-precheck`
- `novel-learn`

Key improvements:

- `write` can now read `learned_patterns` and `market_adjustments` conservatively
- `review` now returns a clearer repair route recommendation
- `polish` explicitly treats anti-AI and avoid-pattern guidance as first-class inputs
- `precheck` now includes learned style and project-local market hints in final readiness judgment
- `learn` now produces more downstream-usable signals for `write / polish / precheck`
