# Genm-codex v0.12.2

第八阶段 project-annotate 实验增强版发布。

## 本版本重点

在 `v0.12.1` 的基础上，进一步验证了 `novel-scan` 的项目级注释路径：

- 允许更新 `.mighty/market-data.json`
- 允许在 `.mighty/state.json -> market_adjustments` 中写入项目级建议
- 仍然不默认改动 `shared/profiles/`

## 结果

这意味着 `novel-scan` 现在已经具备两个实验层级：

- `report-only`
- `project-annotate`

## 说明

- 仍未进入 `apply-suggestion`
- 仍未进入默认生产模式
- 这是实验增强版，而不是最终稳定版
