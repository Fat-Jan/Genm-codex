## 创作指南

### 新增书籍时

1. **时间定位**: 确定新书在时间线的位置
2. **角色关联**: 确认与现有角色的关系
3. **设定继承**: 列出需要遵循的共享设定
4. **Canon审核**: 完成初稿后进行一致性检查

### 撰写番外时

1. **类型选择**: 明确是正史补充还是平行宇宙
2. **Parent Canon**: 导入原作核心设定
3. **时间线校验**: 确保不破坏正史逻辑
4. **跨书标注**: 在涉及跨书内容时添加说明

### 修订设定时

1. **影响评估**: 评估对现有书籍的影响
2. **向后兼容**: 尽量保持向后兼容
3. **版本记录**: 在设定版本控制中记录变更
4. **通知更新**: 标记需要更新的书籍

### Canon级别定义

| 级别 | 说明 | 示例 |
|------|------|------|
| 核心Canon | 不可变更的基础设定 | 力量体系基础规则 |
| 次级Canon | 可扩展但不可矛盾 | 具体势力设定 |
| 软Canon | 有弹性空间的设定 | 角色次要习惯 |
| 非Canon | 番外/IF线专属 | 平行宇宙设定 |
