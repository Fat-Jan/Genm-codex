# 番茄付费信号优先矩阵

## 目的

这份文档回答的不是：

- 哪个分类“看起来很火”

而是：

- **在番茄优先路线里，哪些长篇/短篇方向更值得先做成 MVP**
- **哪些方向更像热度桶，哪些更像收益桶**
- **当前系统应该先押哪几条线，才能更快接近真实稿费结果**

---

## 执行理念

当前这条线的默认执行原则定为：

- **明日资源，今日实现，效率至上**

含义不是盲目提速，而是：

1. 先抓最有收益相关性的公开信号
2. 先做最有商业回报的 MVP 桶
3. 先把工具做成能稳定出稿、能稳定验证
4. 不为“大而全”拖慢第一批真实收益验证

---

## 信号分层

当前对番茄分类的判断，至少要分 4 类信号：

### 1. 热度信号

来源：

- 官网排行榜 / 分类榜
- 阅读榜 / 新书榜

这层回答：

- 当前哪些内容桶有明显分发与曝光

但它**不直接等于收益**。

### 2. 收益/付费倾向信号

来源：

- 作者专区的数据口径
- 作者专区的保障计划 / 征文 / 激励方向
- 付费侧互补观察窗（如红烛小说等）

这层回答：

- 哪些方向更像平台愿意长期扶持
- 哪些方向更可能更接近付费与收益

### 3. 平台资源信号

来源：

- 官方保障金计划
- 征文分类
- 揽星计划、激励计划

这层回答：

- 平台在哪些桶上明确给过扶持

### 4. 系统适配信号

来源：

- 当前 `Genm-codex` 已有 profile / bucket / package / write / review / precheck 链

这层回答：

- 哪些方向现在能最快做成可运行的内容生产链

---

## 信号使用原则

当前阶段不要只看一个维度。

最合理的判断应该是：

- **热度**
- **收益倾向**
- **平台扶持**
- **系统适配**

四个维度一起看。

因此：

- 榜单热，不一定最赚钱
- 付费倾向高，不一定当前最容易实现
- 平台扶持强，也不一定适合先做成系统

真正要优先做的是：

- **综合回报最高的桶**

## 第一批长篇优先矩阵

### A. 稳收优先桶

#### 1. 传统玄幻

判断：

- 热度：中高
- 收益倾向：高
- 平台扶持：高
- 系统适配：高

理由：

- 在番茄公开分类里长期存在
- 历史上多次出现在保障 / 激励口径里
- 和当前项目已有玄幻 profile、番茄 profile、样本链最匹配

当前建议：

- **列为第一批必做长篇桶**

#### 2. 奇幻仙侠 / 东方仙侠

判断：

- 热度：中
- 收益倾向：高
- 平台扶持：高
- 系统适配：中高

理由：

- 更像长线收益型方向
- 平台历史扶持中反复出现
- 与传统玄幻相近，但可以承接更强修行逻辑和代价感

当前建议：

- **列为第一批长篇稳收桶**

### B. 起量优先桶

#### 3. 玄幻脑洞

判断：

- 热度：高
- 收益倾向：中高
- 平台扶持：高
- 系统适配：高

理由：

- 高概念、强钩子、机制显性
- 更适合新书期点击与首屏抓取
- 与现有玄幻链改造成本低

当前建议：

- **列为第一批长篇起量实验桶**

#### 4. 都市高武 / 都市脑洞

判断：

- 热度：高
- 收益倾向：中高
- 平台扶持：中高
- 系统适配：中

理由：

- 起量潜力强
- 适合平台流量测试
- 但当前系统在都市线上的成熟度不如玄幻线

当前建议：

- **第二批长篇实验桶**

### C. 待拆细再做桶

#### 5. 科幻末世

判断：

- 热度：中高
- 收益倾向：未知偏中
- 平台扶持：中
- 系统适配：中

问题：

- 这个分类过大
- 里面可能混着：
  - 囤货末世
  - 异能末世
  - 灾变生存
  - 规则怪谈末世

当前建议：

- **不直接作为第一批长篇桶**
- 先拆成子桶再建模

## 第一批女频付费线 MVP

这是本轮最重要的修正：

- **女频付费线应直接进入第一梯队 MVP**

原因：

- 你现在的目标不是“理论覆盖面”
- 而是“更快让系统产出更可能有收益的内容”

### 长篇女频 MVP

#### 1. 宫斗宅斗

判断：

- 热度：高
- 收益倾向：高
- 平台扶持：中高
- 系统适配：中

理由：

- 强关系冲突
- 强反转
- 章末承接天然强
- 明显适合付费追读逻辑

当前建议：

- **列入第一梯队长篇女频 MVP**

#### 2. 现实情感 / 职场婚恋（长线版）

判断：

- 热度：中高
- 收益倾向：高
- 平台扶持：中
- 系统适配：中

理由：

- 情绪拉扯强
- 付费意愿通常不低
- 适合女性读者长期追更

当前建议：

- **列入第一梯队女频长线备选**

### 短篇 / 中短女频 MVP

#### 1. 现实情感

判断：

- 热度：中高
- 收益倾向：高
- 平台扶持：高
- 系统适配：中高

理由：

- 官方短故事口径里很明确
- 天然适合情绪集中、读完率导向和渠道转化

当前建议：

- **短篇女频第一优先桶**

#### 2. 现言甜宠

判断：

- 热度：高
- 收益倾向：中高
- 平台扶持：高
- 系统适配：高

理由：

- 标题和简介公式清晰
- 起量快
- 最适合先做成短篇 MVP 模型

当前建议：

- **短篇女频第一优先桶**

#### 3. 悬疑脑洞

判断：

- 热度：高
- 收益倾向：高
- 平台扶持：中高
- 系统适配：中高

理由：

- 开头和读完率都容易做强
- 不只适合男频，也很适合作为短篇 / 中短高转化试验桶

当前建议：

- **短篇高转化试验桶**

- `都市脑洞`

- `职场婚恋`

## 对当前系统的直接要求

不是再继续发散加命令，而是：

### 1. 先把第一批 MVP 桶做成模板

每个桶至少要有：

- 标题模板
- 简介模板
- 开头模板
- 节奏模板
- precheck 模板

### 2. 先做“主桶 + 主 profile + 标签包”

不要先做复杂多 profile 混合。

当前最稳公式：

- `主 profile`
- `主 content_bucket`
- `2~4 个强标签`

### 3. 女频付费线不要延后

这一点现在已经明确：

- **女频付费线应当和男频长篇线并行推进 MVP**

不是等男频全部做完再说。

---

## 当前一句话决策

如果现在要以“更快接近真实收益”为目标推进 `v1.1`：

- **长篇先做：传统玄幻 / 奇幻仙侠 / 玄幻脑洞**
- **女频付费 MVP 同时做：宫斗宅斗（长篇）+ 现实情感 / 现言甜宠（短篇）**
- **科幻末世先拆子桶，不直接进第一批**
