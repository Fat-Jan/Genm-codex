# `v1.5` 第一批 Profile 扩面完成说明

## 目的

这份文档把 `P2-A` 当前已经落地的第一批 profile / positioning / bucket overlay 资产收成一份明确说明。

它不是 profile contract 本身，而是“哪一批已经可以被当作第一批扩面结果使用”的确认页。

---

## 第一批已落地对象

当前可作为第一批扩面结果使用的 profile 包括：

- `palace-intrigue`
- `historical-brainhole`
- `urban-daily`
- `ceo-romance`
- `xiuxian`
- `sweet-youth`
- `urban-brainhole`
- `workplace-romance`

这些对象已经具备：

- `profile.yaml`
- `platform_positioning.primary_bucket`
- 第一层可消费的 `bucket-*.yaml`

其中 `xiuxian` 还具备 `profile-tomato.yaml`，属于平台差异已经明确拆出的对象。

---

## 当前补面结论

第一批的完成口径不是“54 个 profile 全部补齐平台层”，而是：

- 把最优先的一组 bucket / profile 样本线先补成可消费资产
- 让 registry / consumer 已有稳定第一批读面
- 把后续 profile 扩面从“零散补文件”转成“按 contract 补已排序候选”

---

## 后续仍待扩的对象

当前仍不应宣称已全量完成的部分包括：

- 更大范围的 `platform_positioning` 普及
- 所有 profile 的平台 overlay 分离
- 组合题材和派生题材的系统化补面

这些仍应按后续优先级继续推进，而不是在 `v1.5` 中误报为“全量收口”。
