# 对白、动作、情绪与段落节奏

## 目标

把“人在场上”写实，而不是让段落只剩总结、转述和悬浮对话。

## 四条规则

### 1. 对白必须带来变化

好对白至少做到一件事：

- 改变信息差
- 改变关系温度
- 改变站位
- 改变后续动作

如果对白只是解释设定、复述上一章、代替作者总结，就是低效对白。

### 2. 动作要防止人变成悬浮头像

- 关键对白前后要有动作锚点
- 动作不只是“走、看、笑”，还要体现：
  - 压力
  - 犹豫
  - 试探
  - 控制

### 3. 情绪要通过后果和反应落地

- 不要只写“她很愤怒 / 他很紧张”
- 更有效的是写：
  - 声调变化
  - 动作变形
  - 决策转向
  - 关系后果

### 4. 段落节奏要利于追读

高风险失格：

- 长段解释压住动作
- 对白堆叠但没有动作切面
- 一段里同时总结多件事
- 章末靠概述交代“接下来更危险”

## 对下游技能的可执行影响

### `novel-write`

- 防止 floating dialogue
- 防止用总结代替演出

### `novel-review`

- 检查对白是否带变化
- 检查段落是否进入提纲化压缩

### `novel-precheck`

- 检查开篇和黄金三章是否存在“快但空”的段落组织
