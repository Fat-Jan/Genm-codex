# Genm-codex v0.10.2

第七阶段治理增强补丁发布。

## 本版本重点

在 `v0.10.1` 的基础上，补齐了 shared 治理层的第一批可观测工件：

- `docs/shared-asset-dependency-map.md`
- `scripts/sync-shared-from-genm.sh --report`
- `scripts/sync-shared-from-genm.sh --report --domain <...>`

## 结果

这意味着第七阶段不只是“有方案”，还已经有了可执行的治理起步能力：

- 能看到 shared 各域的文件量与目录量
- 能按域检查 profiles / references / templates
- 能把 skill 对 shared 的依赖关系文档化

## 说明

- 仍未直接切换 selective-sync
- 仍未正式实现 `novel-scan`
- 这是一个治理增强版，而不是功能扩展版
