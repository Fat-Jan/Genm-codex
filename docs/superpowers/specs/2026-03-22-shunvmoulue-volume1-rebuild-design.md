# 《庶女谋略》第一卷重修设计

**目标**

重修 `projects/庶女谋略` 第一卷 `001-020` 章章纲，使其达到可直接落正文的细纲级；同步补齐服务第一卷写作的基础设定集，并修正第001章“仅凭照镜子确认穿越身份”的失真口径。

**问题定义**

当前项目存在四个核心问题：

1. `第001章` 正文中，主角仅凭镜中容貌就确认自己是小说人物，缺少可信的身份闭合链。
2. `第003章` 之后的章纲整体过简，缺少可直接写作的场景链、因果链与代价链。
3. `设定集/` 偏抽象，缺少第一卷正文高频需要的年代、家规、宅院动线、婚配礼制、人际账本。
4. “苏照棠变了”目前更像作者告知，缺少原书版本与现版本都能成立的现实触发点。

**设计原则**

1. 第一卷仍以番茄宅斗前二十章节奏为主，不让高维暗线抢主场。
2. 主角变化先体现在反应方式、判断方式、借势方式，而不是开挂式压制。
3. 每次小胜必须伴随更大的被盯上风险，不能只写收益不写余波。
4. “她变了”必须来自外部人物可观察的连续证据，而不是旁白判断。
5. 原书版苏照棠也应有“变了”的契机，只是变化方向不同。

**第一卷结构重组**

- `001-005`：病后醒转，第一次试探，第一次得到回应变化带来的好处与反噬。
- `006-010`：低位同盟建立，父权视线落下，席面与体面成为主要战场，形成第一次立威。
- `011-015`：父亲重新看见她，但“被看见”转化成新的监视、试探和敌意。
- `016-020`：婚配线起，主角第一次意识到高嫁未必是脱困，而可能是更昂贵的再角色化。

**“她变了”的统一口径**

本次重修采用统一触发点：

- 开篇前，苏照棠已在苏府经历一次具体的“病中受辱/规矩压身”事件。
- 此事导致她发热昏沉，醒来后，府中人对她的第一观感就是“病过一场，人像是变了”。
- 原书版的变化方向：更会忍、更会缩、更懂得把自己收成一个顺规矩的庶女。
- 穿越版的变化方向：更会判断、更会借规矩回击、更不愿白白吞下去。

这样处理后：

- 第001章的性情变化有现实理由。
- 原书与现版共用一个事件锚点，不会像两套人物。
- 后续人物评价“二姑娘变了”有连续可观察的依据。

**第001章重修要求**

第001章必须把“确认身份”拆成多重线索闭合，而不是照镜子即得结论。最低应包含：

1. 屋内可验证姓名身份的物件或文书。
2. 丫鬟称呼与前情信息。
3. 当前情节节点与原著记忆的交叉验证。
4. 镜子只负责证明“身体不是原来的自己”，不负责证明“她就是苏照棠”。

**章纲升级标准**

每章章纲至少包含：

- 章节定位与卷内功能
- 本章核心问题
- 本章收益与代价
- 4-6 个场景节点
- 每个节点的主动方/防守方/误判点
- 外部人物如何感知“她变了”
- 本章留下的账本变化
- 章末钩子与下一章承接

**设定集补齐范围**

第一卷优先补齐以下基础写作资产：

- `设定集/世界观/时代与年号口径.md`
- `设定集/家族/苏府家规与日常规矩.md`
- `设定集/地点/苏府宅院结构与动线.md`
- `设定集/家族/婚配礼制与议亲流程.md`
- `设定集/势力/苏府人际关系账.md`

这些文件要服务具体写作问题：

- 请安、用度、座次、衣饰、见客、传话、罚跪、抄经、禁足如何落地
- 月例、针线、节礼、丫鬟调配如何体现身份差
- 婚配消息如何流出，谁能先知道，谁能借题发挥
- 苏府内谁是风向节点，谁会传话，谁会观望，谁只保自己

**不做内容**

本轮不做以下事项：

1. 不扩写第二卷及以后章纲。
2. 不补高门后期详细设定，只保留第一卷必要前置接口。
3. 不在本轮直接重写全卷正文。
4. 不把高维设定前置成早期明牌解释。

**验证标准**

完成后应满足：

1. `001-020` 章纲连续可读，任取一章都能直接按场景节点起正文。
2. 第001章身份确认逻辑可信，不再依赖“看脸认人”。
3. “她变了”的评价具备触发事件和外部观感链。
4. 设定集足以支持第一卷正文，不再频繁临场现编。
