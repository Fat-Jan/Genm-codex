  - 压制关系
  - 身份反咬
  - 阵营与规矩
  - 章末挂更大局
- 这 6 章基本都在服务这套逻辑
- `恶女` 叠加后的效果也成立：
  - 不是纯狠
  - 不是发疯
  - 而是“守规矩表皮下的稳、狠、会算”

### 连续性

- 婚书 -> 东宫 -> 庄子旧人 -> 外院账房 -> 伯爷书房
  这条升级链路是通的
- 主角手里的证据链也在持续累积：

  - 玉扣
