# 恶女组合线对照结论

## 目的

这份文档回答一个直接问题：

- `恶女 x 宫斗宅斗`
- `恶女 x 现实情感`
- `恶女 x 现言甜宠`

这三条线里，哪条更适合继续做成正式生产模板线。

## 1. 恶女 x 宫斗宅斗

### 当前判断

- **最值得继续深做**

### 原因

1. 它和主桶天然同构  
   `宫斗宅斗` 本来就强调：
   - 压制关系
   - 反咬翻盘
   - 阵营斗法
   - 章末挂更大局  
   `恶女` 叠加进去以后，不是换题材，而是把这些点进一步推强。

2. 恶女感不容易写悬浮  
   在这条线上，恶女不是“发疯”，而是：
   - 表面守规矩
   - 内里极能忍也极能算
   - 拿后手反咬  
   这非常符合番茄女频长篇的连续追读感。

3. 包装和正文都容易统一  
   当前样本已经能稳定呈现：
   - 一夜翻盘
   - 婚书被换
   - 后宅压制
   - 东宫抬局  
   包装、正文、后续大纲都很好衔接。

### 当前建议

- **列为第一条正式深做的恶女生产线**

---

## 2. 恶女 x 现实情感

### 当前判断

- **很强，适合作为第二条线**

### 原因

1. 现实切割感非常强  
   这条线的优势在于：
   - 不再忍
   - 主动留证
   - 一笔笔算账
   - 让对方慌  
   它很适合番茄的现实情感高痛点、高反击、高转化逻辑。

2. 更靠现实执行力，而不是纯爽感  
   这条线最强的地方不是“黑”，而是：
   - 利己
   - 有边界
   - 现实手段成立  
   所以付费线潜力高。

3. 风险在于容易写重  
   如果控制不好，容易变成：
   - 太压抑
   - 太沉重
   - 只有算账没有连续爽点  
   这就是它暂时排在第二的原因。

### 当前建议

- **列为第二条正式深做的恶女生产线**

---

## 3. 恶女 x 现言甜宠

### 当前判断

- **可以保留，但更适合作为第三条实验线**

### 原因

1. 这条线最容易出反差感  
   核心点是：
   - 她先下手
   - 误认身份
   - 关系绑死
   - 甜中带掌控欲

2. 但最容易写崩  
   因为它需要同时保持：
   - 甜宠可读性
   - 女主危险感
   - 男主不降智
   - 关系不失衡  
   稍微控制不好，就会变成：
   - 作精
   - 无脑甜
   - 狗血误会

3. 更适合在前两条跑稳后再深做  
   现在可以保留样本、保留玩法，但不建议先把资源砸在这条线上。

### 当前建议

- **列为第三条恶女实验线**

1. `恶女 x 宫斗宅斗`
2. `恶女 x 现实情感`

3. `恶女 x 现言甜宠` 保留为实验样本，等前两条稳定后再加强

## 一句话结论

如果现在必须只选一条继续深做：

- **选 `恶女 x 宫斗宅斗`。**

因为它是三条里面：

- 最不容易写崩
- 最容易保持高戏剧张力
- 最容易和番茄女频长篇追读逻辑对齐

如果要选第二条：

- **选 `恶女 x 现实情感`。**
