# 豪门总裁 Overlay

## 开篇公式

推荐公式：

- `身份差/资源差亮相 -> 强关系错位前置 -> 合约/婚约/继承局落地 -> 第一轮交换带来更大舆论或家族残账`

第一屏优先出现：

- 身份差与资源差
- 合约、婚约、继承、舆论、家族博弈
- 强关系钩子前置

## 黄金三章最低要求

- 第 1 章：
  - 落身份差
  - 落核心关系错位
  - 让读者知道这不是普通恋爱局
- 第 2 章：
  - 至少有一次利益交换或公开冲突
  - 不要只停在“他很危险/她很特别”
- 第 3 章：
  - 给一次强记忆点
  - 不要只停在设定介绍或外形描写

## 剧情层次重点

- 主线：关系博弈
- 次级线：家族权力、商业资源、舆论风险

## 高频故障清单

- 只剩人设标签和金句
- 霸总权力万能，冲突不成立
- 包装很猛，正文全靠嘴说
- 女主处境不具体，只剩“被看上”
- 家族或商业资源不进场，导致豪门只剩壳子

## precheck 专项检查点

- 关系钩子是否具体
- 资源差是否真的参与换账
- 开篇 promise 是否超出正文可承载范围
- 第三章前是否至少出现一次公开代价或舆论风险
- 家族/商业线是否只是背景板
