# 都市脑洞 Overlay

## 开篇公式

推荐公式：

- `现代困局落地 -> 脑洞机制快速点亮 -> 第一次现实反馈可见 -> 小打脸或收益兑现后挂更大钩子`

第一屏优先出现：

- 现代生活场景 + 异常机制
- 系统、神豪、直播、文娱、签到等脑洞元素快速入场
- 即时可见的反差爽点

## 黄金三章最低要求

- 第 1 章：
  - 点亮脑洞机制
  - 让读者知道它将如何改变现实
- 第 2 章：
  - 机制第一次改变现实处境
  - 最好伴随一次试错或边界确认
- 第 3 章：
  - 形成可复制的爽点循环雏形
  - 但不要直接无敌通关

## 剧情层次重点

- 主线：脑洞机制如何改变主角现实位置
- 次级线：社交反馈、职业变化、资源增长

## 高频故障清单

- 只有设定新鲜，没有剧情交换
- 现实层面没有真实反馈
- 爽点密度不够，章节平
- 机制像万能许愿机，没有边界和成本
- 只有网上震惊，没有现实后果

## precheck 专项检查点

- 脑洞机制是否马上见效
- 现实世界反馈是否可见
- 章末是否有下一轮更大钩子
- 第三章前是否形成一次完整的小爽点循环
- 现实困局是否真的被改变，而不是只在设定层说改变了
