# 青春甜宠 Overlay

## 开篇公式

推荐公式：

- `关系错位或误判 -> 情绪入口成立 -> 校园/青春场景承载互动 -> 小心动兑现但不直接吃满`

第一屏优先出现：

- 校园关系错位
- 明显的情绪入口
- 青春成长压力和心动场景并置

## 黄金三章最低要求

- 第 1 章：
  - 看见关系张力
  - 看见校园/青春环境真的参与互动
- 第 2 章：
  - 给一次互动升级、误判或吃醋节点
  - 不能只写“觉得他好特别”
- 第 3 章：
  - 要有小兑现
  - 不能一直停在暧昧空转

## 剧情层次重点

- 主线：心动与成长并行
- 次级线：同学圈、家庭压力、成绩/前途

## 高频故障清单

- 只有甜，没有关系推进
- 只有心动感受，没有具体事件
- 校园背景像贴纸，不参与剧情
- 角色台词太像成年人职场恋爱
- 甜点全靠男主发糖，女主没有主动反应或成长线

## precheck 专项检查点

- 第一屏情绪入口是否成立
- 甜点是否建立在事件上
- 校园场景是否真的影响关系变化
- 前三章是否有一次明确互动升级
- 成长压力是否只是装饰，而没有真的影响选择
