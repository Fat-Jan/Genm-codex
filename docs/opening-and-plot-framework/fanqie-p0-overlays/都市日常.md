# 都市日常 Overlay

## 开篇公式

推荐公式：

- `具体生活切口 -> 小人物困局落地 -> 一次现实问题开始变化 -> 小改善伴随新残账`

第一屏优先出现：

- 生活压力的具体切口
- 行业/职业真实细节
- 小人物困局和现实改善欲

## 黄金三章最低要求

- 第 1 章：
  - 给明确生活处境
  - 给一个让读者代入的现实问题
- 第 2 章：
  - 让具体问题开始变化
  - 最好有试错、卡点或条件交换
- 第 3 章：
  - 给一次现实层面的可感改善或反噬
  - 不要一直停在氛围描写

## 剧情层次重点

- 主线：生活改善 / 事业推进
- 次级线：家庭、同事、人情、两性关系

## 高频故障清单

- 流水账
- 只写氛围，不写问题
- 过度悬浮，失去都市真实感
- 每章都有事，但没有任何账本变化
- 只有治愈，没有现实摩擦

## precheck 专项检查点

- 现实问题是否具体
- 推进是不是只靠总结句
- 日常是否真的承载冲突而不是纯散文
- 第三章前是否已经出现一次可感的现实换账
- 行业/职业细节是否真实，而不是万能背景词
