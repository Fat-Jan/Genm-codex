# 历史脑洞 Overlay

## 开篇公式

推荐公式：

- `历史处境或朝局落地 -> 脑洞切口亮相 -> 第一次用新机制碰撞旧秩序 -> 局部破局后留下更大时代残账`

第一屏优先出现：

- 历史环境中的高概念脑洞
- 科举、争霸、经营、征战等目标尽快明确
- 金手指和历史局势同场

## 黄金三章最低要求

- 第 1 章：
  - 明确历史处境
  - 明确脑洞切口
  - 让读者知道这是“历史局 + 新机制”的组合
- 第 2 章：
  - 脑洞机制第一次作用于现实局面
  - 不要只停在展示聪明或设定说明
- 第 3 章：
  - 给一次阶段性破局
  - 但保留更大的历史残账、政局压力或时代门槛

## 剧情层次重点

- 主线：破局 / 上位 / 改局
- 次级线：家族、政局、地域、资源、时代规则

## 高频故障清单

- 历史背景只当贴图
- 金手指与历史局势脱节
- 一路平推，没有阻力与代价
- 历史氛围全靠称谓堆砌
- 脑洞有创意，但没有真正改变局势账本

## precheck 专项检查点

- 历史环境是否真正参与冲突
- 脑洞机制是否改变了局势，不只是炫技
- 章末是否留下更大时代压力
- 第三章前是否完成一次“新机制碰撞旧秩序”的可感闭环
- 语言是否既有历史感，又不至于把阅读流畅性打断
