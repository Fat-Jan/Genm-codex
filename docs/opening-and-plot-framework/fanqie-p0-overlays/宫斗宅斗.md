# 宫斗宅斗 Overlay

## 开篇公式

推荐公式：

- `名分或婚配错位 -> 具体压迫落地 -> 主角识别局面 -> 小换账或反卡位`

第一屏优先出现：

- 婚配错位
- 家族位阶压迫
- 嫡庶/名分冲突
- 权力交易单元

这一类开篇最怕“规矩讲了一堆，但没人真的出手”。

## 黄金三章最低要求

- 第 1 章：
  - 见到具体压迫对象
  - 见到具体卡位点
  - 见到主角不是被动挨打，而是在判断局
- 第 2 章：
  - 至少出现一次压迫 -> 回应 / 换账
  - 最好有礼法、人证、家主态度或内宅门槛来卡主角
- 第 3 章：
  - 给半胜、险胜或局部翻盘
  - 但必须留下更大的规矩风险、树敌后果或证据缺口

## 剧情层次重点

- 主线：上位 / 翻盘 / 自保
- 次级线：家族关系线、后宅势力线、婚配线
- 规则线：法统、位阶、称谓、家权

## 高频故障清单

- 只写压迫，不写换账
- 全靠权力捷径解决
- 关系词好听，但谱系不闭合
- 第三章赢得太满，后面只剩升级位份
- 反派像纸片人，只会恶意发言不会布局

## precheck 专项检查点

- 首屏冲突是否足够具体，不是纯背景介绍
- 权力账本有没有变化：
  - 信息
  - 礼法
  - 家权
  - 关系
- 称谓与真值是否一致
- 前三章是否至少完成一次压迫 -> 回应 -> 局部收益
- 章末是否留了下一轮更危险的残账
