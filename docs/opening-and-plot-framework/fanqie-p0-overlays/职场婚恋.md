# 职场婚恋 Overlay

## 开篇公式

推荐公式：

- `职业压力落地 -> 关系对象进入同一局 -> 合作或对抗形成利害 -> 交换后留下双线残账`

第一屏优先出现：

- 工作危机和情感危机同场
- 项目、岗位、升职、裁员、协作失败等现实压力
- 男女主第一次同场就带利害关系

## 黄金三章最低要求

- 第 1 章：
  - 明确职业处境
  - 明确关系阻力
  - 明确主角不是恋爱脑先行，而是有现实压力
- 第 2 章：
  - 合作或对抗必须产生具体后果
  - 不能只靠一句“你很特别”推进关系
- 第 3 章：
  - 给一次关系或职业层面的阶段换账
  - 但不要一章把信任、心动、事业收益全拿完

## 剧情层次重点

- 主线：事业推进与关系推进的互相牵制
- 次级线：家庭期待、同事关系、资源竞争

## 高频故障清单

- 只有恋爱氛围，没有职业真实压力
- 一次配合后立即同频
- 职场名词堆砌但没有真实利害
- 事业线和感情线完全割裂
- 男女主第一次同场没有任务、项目或利益绑定

## precheck 专项检查点

- 职场冲突是不是可验证
- 关系推进是否过快
- 甜和压有没有同场存在
- 项目、岗位、汇报、晋升链是否真实可感
- 第三章是否已经把双线收益吃满，导致后续无张力
