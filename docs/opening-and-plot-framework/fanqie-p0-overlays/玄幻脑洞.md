# 玄幻脑洞 Overlay

## 开篇公式

推荐公式：

- `高概念或金手指出现 -> 当前困局压实 -> 能力第一次见效 -> 第一轮硬兑现带来更大风险`

第一屏优先出现：

- 高概念外挂
- 开局反差
- 规则异常或爽点机制快速亮相

## 黄金三章最低要求

- 第 1 章：
  - 亮金手指或高概念
  - 不要世界观说明压过故事
- 第 2 章：
  - 让能力和阻力同时上场
  - 不能只有介绍能力，没有试错或代价
- 第 3 章：
  - 给一次能证明路线成立的硬兑现
  - 但不要直接把主线冲突打穿

## 剧情层次重点

- 主线：升级 / 生存 / 破局
- 次级线：世界规则、宗门/势力、资源争夺

## 高频故障清单

- 设定讲解过多
- 金手指亮了但没有第一轮兑现
- 升级太顺，没代价
- 战斗有声量，没有真实收益变化
- 世界规则一股脑塞入第一章

## precheck 专项检查点

- 高概念是否首屏可见
- 爽点是否兑现而非只预告
- 规则解释是否压过故事推进
- 能力、阻力、收益是否在前三章完成一次闭环
- 第三章是否只剩震惊，而没有更大的后续风险
