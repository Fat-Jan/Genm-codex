# 开篇构件与组合公式

## 目标

把“怎么抓人”拆成可组合的构件，而不是只说“开头要有吸引力”。

## 六个高频构件

### 1. 冲突

让故事不是静态介绍，而是有人被推着动。

常见形态：

- 外部压迫
- 关系失衡
- 身份错位
- 规则卡死
- 资源争夺

### 2. 悬念

让读者往下翻不是因为被讲解，而是因为还不知道答案。

### 3. 情绪代入

让读者知道自己该带着什么情绪读：

- 怒
- 急
- 惜
- 爽前压抑
- 危机式好奇

### 4. 信息差

信息差不是单纯藏设定，而是让读者或角色知道得不一样，从而推动误判、等待或恐惧。

### 5. 主承诺

明确这本书的主要交易是什么：

- 复仇
- 逆袭
- 翻案
- 升级
- 情感反转
- 权力换账

### 6. 章末留钩

一章结束时，最好让账没结干净，而不是把水烧开又立刻晾凉。

## 通用组合顺序

### 组合 A：冲突 -> 情绪 -> 未解问题

适合：

- 爽文
- 宫斗宅斗
- 现实情感压迫局

### 组合 B：异常 -> 信息差 -> 冲突落地

适合：

- 悬疑
- 玄幻设定异常
- 系统/穿书/规则流

### 组合 C：关系失衡 -> 主承诺 -> 章末换账

适合：

- 感情线驱动
- 双女主 / 多人关系博弈
- 家族权力题材

## 黄金三章最低要求

- 第 1 章：给到主承诺和第一轮冲突
- 第 2 章：出现卡点、误判、条件交换或局势升级
- 第 3 章：给一次半胜、惨胜或带残留风险的兑现

## 使用提醒

- 构件可以叠加，但不能全靠口头说明
- 至少要有一两个构件落到具体事件、具体人、具体物
- 越靠前，越要避免“讲规则代替演出”
