## 对 `novel-precheck`

- 优先拦前三章只有压迫没有换账
- 优先拦称谓、法统、婚配词与真值不一致

## 核心检查问题

- 是否存在具体争夺物、具体阻力、具体账本变化
- 是否至少完成一次压迫 -> 回应 / 换账闭环
- 对手是否有独立算盘，而不是只会针对主角
- 称谓、位阶、家法、婚配词是否能回到真值表

## 红灯判定

- 本章没有有效交易单元
- 主要收益全靠身份、父权或家法捷径自动到账
- 第三章前没有真正完成第一次换账
- 人物称谓、法统、齿序明显打架
