# 玄幻脑洞专项检查卡

## 自动触发范围

- `meta.platform = 番茄` 且当前 `genre_profile.bucket = 玄幻脑洞`
- 显式传入 `content_bucket = 玄幻脑洞`
- 题材明显走番茄快节奏玄幻脑洞路线

## 对 `novel-review`

- 先看高概念或金手指是否首屏可见
- 先看前三章有没有能力 -> 阻力 -> 收益闭环
- 如果世界观说明压过故事，优先报开篇失真

## 对 `novel-precheck`

- 优先拦首章仍在讲设定，没有主冲突
- 优先拦金手指出现了但三章内没有硬兑现
- 优先拦升级过顺、无阻力、无代价

## 核心检查问题

- 金手指 / 高概念是否快速入场
- 能力第一次见效是否改变了局势
- 阻力和代价是否和爽点一起出现
- 世界规则解释是否被控制在故事之下

## 红灯判定

- 大段讲解设定，故事迟迟不进
- 三章内没有一次能证明路线成立的硬兑现
- 升级纯白拿，没有阻力和代价
- 战斗只有震惊，没有真实收益变化
