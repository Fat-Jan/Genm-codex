# 番茄 P0 专项检查卡

这组文档是 `opening-and-plot-framework` 在 `review / precheck` 侧的 bucket 级专项卡。

它们回答的不是“这类题材怎么写”，而是：

- 这一桶最常见的塌章 / 塌开篇 / 塌承诺问题是什么
- `novel-review` 该先看哪几件事
- `novel-precheck` 在投稿前该优先拦哪几件事

## 使用原则

- 这些卡只在 `platform=番茄` 且当前 bucket 命中对应分类时按需读取
- 它们是专项门，不是新的通用规则源
- 它们优先抓结构性误判、过顺推进、promise 跑飞和桶味失真

## 默认结构

每张卡都包含：

1. `自动触发范围`
2. `对 novel-review`
3. `对 novel-precheck`
4. `核心检查问题`
5. `红灯判定`
