# 青春甜宠专项检查卡

## 自动触发范围

- `meta.platform = 番茄` 且当前 `genre_profile.bucket = 青春甜宠`
- 显式传入 `content_bucket = 青春甜宠`

## 对 `novel-review`

- 先看情绪入口是否成立
- 先看互动是不是建立在事件上
- 如果只有“觉得他好特别”而无实际互动升级，优先报空转

## 对 `novel-precheck`

- 优先拦前三章只有暧昧感，没有小兑现
- 优先拦校园/青春环境不参与推进
- 优先拦角色说话像成年职场恋爱，不像青春线

## 核心检查问题

- 第一屏是否给到明确情绪入口
- 互动升级、误判、吃醋或小心动是否有具体事件承托
- 校园/青春环境是否真的改变关系选择
- 成长线是否在推进，而不是只有发糖

## 红灯判定

- 只有甜味，没有关系推进
- 三章内没有一次互动升级
- 场景脱离校园/青春语境
- 甜点全靠单方投喂，另一方没有主动反应或变化
