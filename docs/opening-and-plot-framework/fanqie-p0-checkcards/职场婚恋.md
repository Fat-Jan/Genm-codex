# 职场婚恋专项检查卡

## 自动触发范围

- `meta.platform = 番茄` 且当前 `genre_profile.bucket = 职场婚恋`
- 显式传入 `content_bucket = 职场婚恋`

## 对 `novel-review`

- 先看事业线和关系线是不是同场推进
- 先看职业冲突有没有真实利害
- 如果整章只剩暧昧气氛，没有工作后果，优先报结构偏空

## 对 `novel-precheck`

- 优先拦“恋爱在走，职场没动”
- 优先拦“合作一次就同频”
- 优先拦包装里写职场博弈，正文却没有真实任务链

## 核心检查问题

- 项目、岗位、汇报、升职、裁员、资源竞争是否真实可感
- 男女主的第一次或前几次同场是否带利害关系
- 关系推进是否建立在任务、合作、对抗或误判上
- 事业线和感情线是否互相改变彼此账本

## 红灯判定

- 职场只是背景贴纸
- 一次配合后直接信任、心动、同频全到位
- 角色职位与能力明显不匹配
- 标题/简介承诺“职场婚恋”，正文却只剩恋爱糖
