# 历史脑洞专项检查卡

## 自动触发范围

- `meta.platform = 番茄` 且当前 `genre_profile.bucket = 历史脑洞`
- 显式传入 `content_bucket = 历史脑洞`

## 对 `novel-review`

- 先看历史处境和脑洞切口是否同场
- 先看新机制有没有真正碰撞旧秩序
- 如果历史只当贴图，优先报题材失真

## 对 `novel-precheck`

- 优先拦前三章只讲创意，不改局势
- 优先拦语言全靠古风词堆砌，阅读不顺
- 优先拦脑洞有趣，但历史环境没有真实压力

## 核心检查问题

- 历史环境是否真正参与冲突和选择
- 脑洞机制是否改变了现实局面
- 政局、家族、地域、制度是否形成阻力
- 语言是否有历史感但不阻碍阅读

## 红灯判定

- 历史背景只剩装饰
- 脑洞机制与时代压力脱节
- 第三章前没有一次“新机制碰撞旧秩序”的闭环
- 一路平推，没有更大的时代残账
