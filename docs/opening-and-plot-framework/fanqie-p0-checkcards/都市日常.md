# 都市日常专项检查卡

## 自动触发范围

- `meta.platform = 番茄` 且当前 `genre_profile.bucket = 都市日常`
- 显式传入 `content_bucket = 都市日常`

## 对 `novel-review`

- 先看现实问题是否具体
- 先看日常是否真的承载冲突和换账
- 如果是流水账、散文感、生活蒙太奇，优先报推进失真

## 对 `novel-precheck`

- 优先拦前三章全是氛围，没有问题变化
- 优先拦行业/职业细节悬浮
- 优先拦“治愈感”建立在空镜和总结句上

## 核心检查问题

- 小人物困局是否具体到人、钱、房、工作、人情或家庭
- 第 2-3 章是否已经让至少一个问题开始变化
- 日常场景有没有改变关系、资源或生活处境
- 真实感是否建立在可验证细节上

## 红灯判定

- 只有氛围没有问题
- 只有总结句推进，没有事件推进
- 第三章前没有可感的现实换账
- 都市生活失去真实质地，悬浮感过强
