# ensemble-return

## best_for

- 都市日常
- 群像
- 家庭/社区
- 多角色关系织网

## not_for

- 主要卖点是单线反打或纯系统升级

## reader_promise

- 看主角进入一个关系网络，并不断留下回流残账

## chapter_1_3_minimums

- 第 1 章建立主要关系入口
- 第 2 章出现第一次关系交换
- 第 3 章留下可回流的人情账或误会账

## scene_engine

- `Goal -> Conflict -> Outcome/Disaster -> Reaction -> Dilemma -> Decision`

## review_watchpoints

- 群像是否只剩功能位
- 日常是否只有流水账
- 回流残账是否成立

## precheck_failures

- 关系很多，但没有可追的主入口
- 日常只温不钩

## package_guardrails

- 不把长期群像回报提前包装成已兑现高潮
