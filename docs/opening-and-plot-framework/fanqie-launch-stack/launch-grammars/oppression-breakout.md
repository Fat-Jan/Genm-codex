# oppression-breakout

## best_for

- 强压迫开场
- 恶线反打
- 身份低位或规则不公的反击型起盘

## not_for

- 主要卖点是温吞关系养成
- 前三章没有明确压制来源

## reader_promise

- 看主角先被压，再找到第一轮破局口

## chapter_1_3_minimums

- 第 1 章出现压制或不公
- 第 2 章出现反击准备、交换或误判
- 第 3 章出现一次小破局或半胜

## scene_engine

- `Goal -> Conflict -> Outcome/Disaster -> Reaction -> Dilemma -> Decision`

## review_watchpoints

- 压制是否太假
- 反击是否太晚
- 第一次赢是否白拿

## precheck_failures

- 前三章只有被压没有反击
- 反击太顺没有代价

## package_guardrails

- 不要把“彻底翻盘”包装成已兑现卖点
