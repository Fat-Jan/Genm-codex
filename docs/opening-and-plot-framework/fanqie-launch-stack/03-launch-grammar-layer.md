# Launch Grammar Layer

## 目标

给前 1-3 章选择最适合的起盘推进语法。

## 这层不是什么

- 不是整本书的总结构
- 不是题材 bucket
- 不是平台协议

1. `oppression-breakout`

卡片入口：

## 推荐输出

- `launch_grammar.primary`
- `launch_grammar.candidates`
- `launch_grammar.confidence`

- 优先看前三章真实要交什么
- 再看 bucket 与 premise 的相容性
