# Pivot Layer

## 目标

先判断这本书主要靠什么“故事支点”起盘，再决定后面的推进语法。

## 第一版 `pivot_family`

1. `character-conflict`
2. `rule-world`
3. `gift-system`
4. `mystery-gap`
5. `transgressive-action`
6. `inciting-event`

## 用法

- `primary_pivot`
  当前最主要的起盘支点
- `secondary_pivot`
  辅助支点，用来避免单线程过薄
- `pivot_confidence`
  `low / medium / high`

## 判断要点

### `character-conflict`

- 读者主要想看关系裂口、站位变化、人物互卡

### `rule-world`

- 读者主要想看异常规则或世界机制怎么咬人

### `gift-system`

- 读者主要想看金手指/系统/外挂怎样改变局面

### `mystery-gap`

- 读者主要想看真相、内幕、身份、旧事的揭露

### `transgressive-action`

- 读者主要被“有人做了危险/越线/过激的事”吸住

### `inciting-event`

- 读者主要被一件强推动事件拉入主线

## 原则

- 支点优先级高于抽象结构名
- 第一版允许 `1 主 + 1 辅`
- 不做每章漂移
